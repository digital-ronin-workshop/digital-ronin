# Первый пост за 30 минут

> Пошаговая инструкция от установки до публикации

---

## Обзор

```
УСТАНОВКА (5 мин) → МИНИ-ИНТЕРВЬЮ (10 мин) → ГЕНЕРАЦИЯ (15 мин) → ПУБЛИКАЦИЯ
```

К концу этого гайда у тебя будет готовый пост для Threads.

---

## Шаг 1: Запусти Claude Code (5 минут)

Убедись, что ты в папке скелета:

```bash
cd /путь/к/content-factory-template
claude
```

---

## Шаг 2: Мини-интервью (10 минут)

Напиши Claude:

```
интервью мини
```

Claude задаст 3 вопроса:

### Вопрос 1: Кто ты и чем занимаешься?

Ответь в 2-3 предложениях:
- Твоя профессия
- Чем занимаешься
- Опыт

**Пример:** "Я врач-психотерапевт, 9 лет практики. Помогаю людям справляться с тревогой и паническими атаками."

### Вопрос 2: Кому помогаешь?

Опиши типичного клиента:
- Кто это (пол, возраст)
- С какой проблемой приходят

**Пример:** "Женщины 30-40 лет после родов или сильного стресса. Приходят с паническими атаками."

### Вопрос 3: Что продаёшь?

- Что предлагаешь
- Какой результат получает клиент

**Пример:** "Онлайн-консультации 60 минут. Человек получает понимание что с ним происходит и план лечения."

---

## Шаг 3: Генерация постов (15 минут)

После интервью напиши:

```
threads 3
```

Claude сгенерирует 3 поста для Threads.

### Что происходит внутри:

```
IDEATOR → WRITER → EDITOR → AUDIENCE SIMULATOR
 (темы)   (тексты)  (стиль)     (оценка)
```

Claude:
1. Придумает темы на основе твоих ответов
2. Напишет посты
3. Отредактирует под твой голос
4. Оценит потенциал каждого поста

---

## Шаг 4: Выбери и опубликуй

Из 3 постов выбери лучший.

**На что смотреть:**
- Хук (первая строка) — цепляет?
- Текст — похоже на тебя?
- CTA — понятно что делать?

Скопируй → Вставь в Threads → Опубликуй.

---

## Готово!

Твой первый пост создан за 30 минут.

## Что дальше?

### Хочешь глубже настроить?

Пройди полное интервью:

```
интервью
```

Это займёт 20-30 минут и добавит:
- Голос и стиль
- Детальные аватары аудитории
- Форматы постов
- Learning loop

### Хочешь больше постов?

```
threads 10
```

### Хочешь YouTube скрипт?

```
youtube script [тема]
```

## Частые вопросы

**Q: Посты слишком сухие/не похожи на меня**
A: Пройди полное `интервью` — Claude лучше поймёт твой голос.

**Q: Claude не понимает мою нишу**
A: Добавь больше деталей в ответы. Конкретные кейсы работают лучше абстракций.

**Q: Как изменить стиль постов?**
A: Отредактируй `brand/voice-style.md` в папке твоего проекта.
