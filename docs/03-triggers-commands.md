# Триггеры и команды

> Быстрые команды для работы с Claude — триггеры активируют нужных агентов автоматически

---

## Сессия

| Команда | Что делает |
|---------|------------|
| `начать сессию` | Показывает статус, активные задачи |
| `закрыть сессию` | Фиксирует прогресс |
| `итоги` | Checkpoint перед переключением |

**Workflow:**
```
начать сессию → работа → закрыть сессию
```

---

## Фиксация прогресса

| Команда | Что делает |
|---------|------------|
| `фиксируй: [что сделал]` | Быстрая запись |
| `прервано: [контекст]` | Сохранить при отвлечении |
| `ошибка: [описание]` | Записать для анализа |

**Примеры:**
```
фиксируй: написал 3 поста для Threads
прервано: разбирал структуру, дошёл до карусели
ошибка: промпт генерит слишком длинные посты
```

---

## Контент (автоактивация skill)

| Триггер | Активирует |
|---------|------------|
| `threads post` | Skill threads → генерация поста |
| `youtube script` | Skill youtube → сценарий |
| `carousel` | Skill carousel → 10 слайдов |
| `reels`, `shorts` | Skill reels → вертикальный контент |
| `thumbnail`, `превью` | Skill nano-banana → обложка |

**Примеры:**
```
threads post про AI-агентов
youtube script: как выбрать нишу
carousel: 5 ошибок новичков
```

---

## Аналитика и инсайты

| Триггер | Что делает | Уточняет |
|---------|------------|----------|
| `инсайты` | Анализ контента и выводы | Проект? Платформа? Период? |
| `что сработало` | Разбор успешного контента | Какой период? |
| `почему провал` | Анализ неудачного поста | Какой пост? |
| `разбери метрики` | Общий анализ | Платформа? 7d/30d/90d? |
| `обнови learning` | Записать выводы | Какой проект? |

**Примеры:**
```
инсайты
→ Какой проект анализируем?
→ YouTube или Threads?
→ За какой период?
```

---

## Продающие смыслы

| Триггер | Что делает | Уточняет |
|---------|------------|----------|
| `продающие смыслы` | Полный цикл работы со смыслами | Для чего? Какой проект? |
| `смысл` | Извлечь или создать продающий смысл | Для чего? Какая ЦА? |
| `почему купят` | Найти главную причину покупки | Какой продукт? |
| `боль аудитории` | Выявить боли ЦА | Какой продукт? |
| `усиль` | Применить триггеры к тексту | Какой текст? |

**Примеры:**
```
продающие смыслы для консультаций
смысл: курс по AI-агентам
почему купят мой продукт?
усиль этот текст
```

---

## Обучение системы

| Команда | Что делает | Куда пишет |
|---------|------------|------------|
| `это сработало` | Записать паттерн | `learning/patterns.md` |
| `это не сработало` | Записать антипаттерн | `learning/anti-patterns.md` |
| `запомни пример` | Сохранить контент | `learning/examples/` |
| `запомни стиль` | Обновить голос | `brand/voice-style.md` |

**После генерации Claude предложит:**
```
Контент готов. Сохранить как пример?
```

**После ваших правок:**
```
Вижу правки. Записать как паттерн?
```

## Полный список триггеров

### Сессия
- `начать сессию` — статус
- `закрыть сессию` — фиксация
- `итоги` — checkpoint

### Фиксация
- `фиксируй: [что]` — запись
- `прервано: [что]` — контекст
- `ошибка: [что]` — ошибка

### Контент
- `threads post` — пост Threads
- `youtube script` — сценарий
- `carousel` — карусель
- `reels` / `shorts` — вертикальное видео
- `thumbnail` / `превью` — обложка

### Аналитика
- `инсайты` — анализ и выводы
- `что сработало` — разбор успеха
- `почему провал` — разбор неудачи
- `разбери метрики` — общий анализ
- `обнови learning` — записать выводы

### Смыслы
- `продающие смыслы` — полный цикл
- `смысл` — извлечь/создать
- `почему купят` — причина покупки
- `боль аудитории` — выявить боли
- `усиль` — применить триггеры

### Обучение
- `это сработало` — паттерн
- `это не сработало` — антипаттерн
- `запомни пример` — сохранить
- `запомни стиль` — обновить голос

### Утилиты
- `inbox` — разобрать заметки
- `research [тема]` — исследование

### Ежедневный
```
начать сессию

```

```

инсайты
```
