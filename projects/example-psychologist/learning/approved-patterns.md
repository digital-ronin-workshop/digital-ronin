## Threads — 2026-01-15

**What worked:** Личная история + симптом

**Context:**
- Format: Threads post
- Topic: Паническая атака на работе

**Pattern:**
- **Element:** Hook с личной историей
- **Why it worked:** Узнаваемость, "это про меня"
- **Example:**
  ```
  Первая паническая атака случилась со мной на планёрке.

  Сидела, слушала отчёт коллеги, и вдруг —
  сердце забилось так, будто сейчас выпрыгнет.
  Руки похолодели. В глазах потемнело.

  Я была уверена: это инфаркт.

  Спойлер: это была не смерть. Это была паника.
  И это лечится.
  ```

**Replication steps:**
1. Начать с конкретной ситуации (место, время)
2. Описать физические симптомы
3. Показать ложную интерпретацию
4. Дать надежду в конце

**Tags:** #threads #hook #personal-story

## Threads — 2026-01-20

**What worked:** Развенчание мифа

**Context:**
- Format: Threads post
- Topic: "Панические атаки — это не про слабых"

**Pattern:**
- **Element:** Контринтуитивный хук
- **Why it worked:** Снятие стыда, поддержка
- **Example:**
  ```
  "Возьми себя в руки" — худший совет при панике.

  Знаете почему?

  Потому что паническая атака — это не слабость.
  Это сбой в системе "бей или беги".

  Ваш мозг защищает вас от несуществующей угрозы.
  Он не виноват. И вы — тоже.
  ```

**Replication steps:**
1. Начать с плохого совета (который все слышали)
2. Объяснить почему он не работает
3. Дать научное объяснение
4. Снять вину с читателя

**Tags:** #threads #myth-busting #empathy

## Threads — 2026-01-22

**What worked:** Список симптомов

**Context:**
- Format: Threads list
- Topic: 7 симптомов тревожного расстройства

**Pattern:**
- **Element:** Чеклист для самодиагностики
- **Why it worked:** Интерактивность, "сколько у меня?"
- **Example:**
  ```
  7 признаков тревожного расстройства:

  1. Постоянное беспокойство без причины
  2. Проблемы со сном
  3. Мышечное напряжение
  4. Раздражительность
  5. Трудности с концентрацией
  6. Усталость
  7. Избегание ситуаций

  Если узнали 4+ — это повод обратиться к специалисту.

  Не для паники. Для понимания.
  ```

**Replication steps:**
1. Заголовок с числом
2. Простой список без лишних слов
3. Порог ("если X из Y")
4. Мягкий CTA (без давления)

**Tags:** #threads #list #checklist
