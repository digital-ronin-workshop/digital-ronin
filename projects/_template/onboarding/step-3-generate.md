# Шаг 3: Генерация первых постов

> После интервью и выбора форматов — генерируем первый batch постов.

---

## Что должно быть готово

Перед генерацией проверь, что заполнены:

- [ ] `brand/profile.md` — кто эксперт
- [ ] `brand/audience.md` — аватары и боли
- [ ] `brand/voice-style.md` — как писать
- [ ] `content/threads-playbook.md` — форматы и шапки

---

## Команда запуска

```
threads 5
```

Генерирует 5 постов через AI-пайплайн.

---

## Как работает пайплайн

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│   IDEATOR    │ ──→ │   WRITER     │ ──→ │   EDITOR     │ ──→ │  AUDIENCE    │
│  (темы)      │     │  (тексты)    │     │  (стиль)     │     │  SIMULATOR   │
└──────────────┘     └──────────────┘     └──────────────┘     └──────┬───────┘
                                                                      │
                                                                      ↓
                                                               ┌────────────┐
                                                               │  Score ≥70%│
                                                               └──────┬─────┘
                                                                 YES  │  NO
                                                                 ↓    ↓
                                                            [ГОТОВО] [ПЕРЕПИСАТЬ]
```

### Агент 1: IDEATOR
Выбирает темы на основе:
- Болей аудитории (`brand/audience.md`)
- Форматов (`content/threads-playbook.md`)

### Агент 2: WRITER
Пишет посты по:
- Темам от IDEATOR
- Голосу эксперта (`brand/voice-style.md`)
- Шаблонам форматов

### Агент 3: EDITOR
Проверяет:
- Запрещённую лексику
- Тон (соответствует ли голосу)
- Длину (100-300 слов для Threads)

### Агент 4: AUDIENCE SIMULATOR
Имитирует реакцию аватаров:
- Каждый аватар оценивает пост
- Если средний score ≥70% → готов к публикации
- Если <70% → отправляет на переписывание

---

## Критерии оценки

Каждый аватар оценивает по 4 параметрам:

| Критерий | Вес | Вопрос |
|----------|-----|--------|
| Узнаваемость | 30% | «Это про меня?» |
| Эмоциональный отклик | 25% | Вызывает чувства? |
| Полезность | 25% | Даёт понимание? |
| Желание поделиться | 20% | «Перешлю подруге?» |

---

## Результат

После генерации получаешь:

```markdown
## Пост 1: [Тема] (Формат: [Тип])

**Score:** 78% ✅

```
[Текст поста]
```

**Оценки аватаров:**
- Марина (34): 85% — «Узнала себя в описании»
- Ольга (42): 70% — «Полезно, но не про меня»
```

---

## Что делать с результатом

### Если score ≥70%
Пост готов к публикации. Можно:
1. Скопировать текст
2. Опубликовать в Threads
3. Через 1 час — добавить ссылку на Telegram в комментарии

### Если score <70%
Claude автоматически переписывает с учётом фидбека аватаров.
Максимум 2 попытки, потом — отклонить и взять другую тему.

---

## Рекомендации для первого batch

1. **Начни с 5 постов** — не больше
2. **Посмотри оценки аватаров** — они показывают, что резонирует
3. **Запомни, что сработало** — эти темы развивать дальше
4. **Не публикуй всё сразу** — 1-2 поста в день
