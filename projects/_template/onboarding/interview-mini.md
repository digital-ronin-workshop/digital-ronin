# Мини-интервью (быстрый старт)

> **Цель:** Первый пост за 30 минут
> **Время интервью:** 10 минут
> **Команда:** `начать` / `/setup` или `интервью мини` / `/interview mini`

---

## ⚠️ ВАЖНО: Где создавать файлы

**Claude, ОБЯЗАТЕЛЬНО выполни перед интервью:**

### 1. Спроси имя проекта

```
Как назовём твой проект?
(латиницей, одним словом — например: daria, psycholog, coach)
```

### 2. Создай папку проекта

```
projects/{имя-проекта}/
```

### 3. Создай структуру внутри

```
projects/{имя-проекта}/
├── brand/
│   ├── profile.md
│   ├── audience.md
│   └── voice-style.md
├── learning/
│   ├── approved-patterns.md
│   └── anti-patterns.md
└── content/
```

### 4. ❌ НЕ создавай файлы в корне!

**Правильно:**
```
projects/daria/brand/profile.md
projects/daria/brand/audience.md
```

**Неправильно:**
```
brand/profile.md          ← В КОРНЕ — НЕЛЬЗЯ!
brand/audience.md         ← В КОРНЕ — НЕЛЬЗЯ!
```

## Инструкция для Claude

При команде `начать`, `/setup`, `интервью мини` или `/interview mini`:

1. **Приветствие:**
   ```
   Привет! Я помогу тебе настроить Content Factory за 10 минут.

   Отвечай как удобно — текстом или голосом.
   Я задам 3 вопроса, потом сразу сгенерирую первые посты.

   Готов? Начинаем!
   ```

2. **Задавай вопросы по одному** — дождись ответа перед следующим

3. **После каждого ответа:**
   - Подтверди что понял: "Понял! Ты [краткое резюме]"
   - Если ответ слишком короткий — уточни: "Расскажи чуть подробнее про..."

4. **После всех вопросов:**
   - Заполни файлы brand/
   - Сгенерируй 3 поста
   - Предложи полное интервью

## 3 вопроса

### Вопрос 1: Кто ты?

```
Расскажи про себя — кто ты и чем занимаешься?
(2-3 предложения достаточно)
```

**Уточнения если нужно:**
- Какая профессия/ниша?
- Сколько лет опыта?
- Что делает тебя экспертом?

> *Пример ответа: "Я врач-психотерапевт, 9 лет практики. Помогаю людям справляться с тревогой и паническими атаками."*

→ **Записываем в:** `projects/{имя-проекта}/brand/profile.md`

---

### Вопрос 2: Кому помогаешь?

```
Кто твои клиенты? С какой проблемой приходят?
```

**Уточнения если нужно:**
- Опиши типичного клиента (пол, возраст, ситуация)
- С чем именно приходят?
- Какими словами описывают проблему?

> *Пример ответа: "Женщины 30-40 лет, после родов или сильного стресса. Приходят с паническими атаками — думают что умирают, но врачи говорят 'вы здоровы'."*

→ **Записываем в:** `projects/{имя-проекта}/brand/audience.md`

---

### Вопрос 3: Как общаешься?

```
Как ты обычно общаешься — на "ты" или на "Вы"?
Какой стиль ближе — тёплый и неформальный или профессиональный и сдержанный?
```

**Уточнения если нужно:**
- Есть фразы которые часто используешь?
- Есть слова которые НЕ используешь?
- Юмор уместен или нет?

> *Пример ответа: "На Вы, но тепло. Без официоза. Юмор мягкий, без сарказма."*

→ **Записываем в:** `projects/{имя-проекта}/brand/voice-style.md`

## После интервью

### Шаг 1: Заполнить файлы

Claude автоматически создаёт:

**brand/profile.md:**
```markdown
## Кто ты
[Имя], [профессия]

## Опыт
[Количество лет], [достижения]

## Экспертиза
[В чём специализация]
```

**brand/audience.md:**
```markdown
## Твой клиент
[Описание типичного клиента]

## С какой проблемой приходят
[Боли и запросы]

## Какими словами описывают
[Цитаты клиентов]
```

**brand/voice-style.md:**
```markdown
## Тон
- На [ты/Вы]
- [Тёплый/Профессиональный/Дружеский]

## Характер
[2-3 характеристики]

## Табу
[Что НЕ используем]
```

### Шаг 2: Первые посты

```
Отлично! Твой профиль готов.

Сейчас сгенерирую 3 поста для Threads.
Выбери тот, который больше похож на тебя — это поможет мне лучше понять твой стиль.
```

→ Генерируем 3 разных поста на основе profile + audience + voice

### Шаг 3: Предложить полное интервью

```
Базовая настройка готова!

Хочешь глубже настроить голос?
Скажи "интервью" — это займёт ещё 20 минут,
но посты будут намного точнее попадать в твой стиль.

Или можем сразу генерировать контент — скажи "threads 5"
```
