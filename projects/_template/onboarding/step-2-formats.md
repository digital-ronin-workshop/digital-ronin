# Шаг 2: Выбор форматов постов

> После интервью у тебя есть профиль, аудитория, голос.
> Теперь выбираем **форматы** — типы постов, которые будем генерировать.

---

## Зачем это нужно?

Форматы — это **шаблоны**, по которым AI будет писать посты.
Чем точнее описаны — тем лучше результат.

---

## Универсальные форматы

Работают для любой ниши. Выбери 3-5 подходящих.

### 1. ФАКТ
**Что это:** Короткий экспертный факт по теме.

**Структура:**
```
[Шапка эксперта]

[Один факт в 2-3 предложениях]
```

**Пример (врач):**
```
Я врач-психотерапевт. Лечу тревогу.

Факт 157:
Вы проверяете по десять раз — выключили ли плиту.
Это не плохая память. Это тревога не доверяет вашим действиям.
```

**Пример (маркетолог):**
```
Я 10 лет в маркетинге.

Факт:
Контент без стратегии — это не маркетинг. Это хобби.
```

---

### 2. ИСТОРИЯ
**Что это:** Реальный кейс из практики.

**Структура:**
```
[Клиент пришёл с проблемой]
[Что пробовал/говорили другие]
[Что сделали]
[Результат]
[Вопрос на вовлечение]
```

**Пример:**
```
Пришла клиентка. Три года лечила симптом. Врачи ничего не находили.

Мы разобрали ситуацию. Оказалось — [инсайт].

Результат: [что изменилось].

А у вас так было?
```

---

### 3. СИМПТОМ/ПРОБЛЕМА + ОБЪЯСНЕНИЕ
**Что это:** Берём боль аудитории → объясняем причину.

**Структура:**
```
[Узнаваемая проблема]? [Вопрос-крючок]

Вот что происходит:
[Простое объяснение в 2-3 предложениях]

Это не [страшное]. Это [реальная причина].

[Что делать]
```

---

### 4. МИФ
**Что это:** Разрушаем популярное заблуждение.

**Структура:**
```
«[Распространённый миф]»

Так говорят многие. На самом деле:

[Правда в 2-3 предложениях]

[Что это означает для читателя]
```

---

### 5. ПРОВОКАЦИЯ / НЕПОПУЛЯРНОЕ МНЕНИЕ
**Что это:** Смелый тезис, который цепляет.

**Структура:**
```
[Провокационный заголовок]

[Объяснение — почему это правда]

[Вывод]
```

**Пример:**
```
Психосоматика — это не «в голове».

Часто слышу: «Мне говорят, это психосоматика. Типа, я придумываю?»

Нет. Боль реальна. Тело реально страдает.
```

---

### 6. ВОПРОС НА ВОВЛЕЧЕНИЕ
**Что это:** Короткий пост, цель — комментарии.

**Структура:**
```
[Контекст в 1-2 предложениях]

[Открытый вопрос]
```

**Пример:**
```
Тревога у всех проявляется по-разному.

А какой симптом мучает вас больше всего?
```

---

### 7. ЛИЧНОЕ
**Что это:** История от первого лица. Показывает человека за экспертом.

**Структура:**
```
[Личная ситуация]
[Чему научила]
[Связь с темой эксперта]
```

---

## Форматы шапок

Шапка — первые строки поста. Показывает экспертность.

**Выбери 2-3 шапки для ротации:**

```
1. Я [профессия]. [Опыт]. Вот что [говорю клиентам / понял за N лет]:
2. Я [профессия]. Вот правда о [теме]:
3. То, что вам не говорят о [теме]:
4. Я N лет [что делаю]. Вот что поняла:
5. [Профессия] не должен это говорить, но:
```

---

## Микс форматов

Рекомендуемое соотношение на 10 постов:

| Формат | Количество | Зачем |
|--------|------------|-------|
| Факты | 3-4 | Основа — показывает экспертность |
| Истории | 2 | Вовлечение — люди любят кейсы |
| Симптом + объяснение | 2 | Узнаваемость — «это про меня» |
| Миф / Провокация | 1-2 | Виральность — цепляет |
| Вопрос / Личное | 1 | Вовлечение — комментарии |

---

## CTA (призывы к действию)

### Для вовлечения (в конце поста)
- «Узнали себя?»
- «А у вас так?»
- «Расскажите в комментариях»
- «Какой [симптом/проблема] беспокоит вас?»

### На Telegram (в комментариях, не в посте!)
```
Если узнали себя — у меня в телеграме больше таких разборов: t.me/[канал]
```

**Правило:** Ссылки только в комментариях, и только после 1000+ просмотров.
