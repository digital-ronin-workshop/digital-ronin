# Skill Learning

> Здесь накапливается твой опыт работы со skills.

## Как это работает

Каждый skill может учиться на твоём опыте:
- Что сработало → `patterns.md`
- Что не сработало → `anti-patterns.md`
- Удачные примеры → `examples/`

## Команды обучения

| Команда | Куда записывает |
|---------|-----------------|
| "это сработало" | skill-learning/{skill}/patterns.md |
| "это не сработало" | skill-learning/{skill}/anti-patterns.md |
| "запомни пример" | skill-learning/{skill}/examples/ |

## Структура

```
skill-learning/
├── threads/          ← твои паттерны для threads
├── carousel/         ← твои паттерны для каруселей
├── youtube/          ← твои паттерны для YouTube
├── selling-meanings/ ← твои паттерны для смыслов
├── reels/           ← твои паттерны для reels
└── ...
```

## Важно

- Эта папка **НЕ обновляется** при апдейтах конструктора
- Все твои наработки остаются
- Skills читают и global patterns (от Max), и твои local patterns
