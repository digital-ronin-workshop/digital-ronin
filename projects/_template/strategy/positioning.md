## Формула позиционирования

```
[Имя] — [профессия], который помогает [кому]
[понять/решить/получить что],
через [метод/подход] — [в чём отличие от других].
```

**Пример (врач):**
```
[Имя] — врач-психотерапевт, который помогает людям с тревогой
понять, что с ними происходит, и вернуться к нормальной жизни —
через понимание психики, а не только таблетки.
```

## Конкурентная матрица

Где ты находишься относительно конкурентов?

```
                    [Качество 1]
                         ↑
                         |
    [Конкурент А]        |     ТЫ ★
                         |
    ←────────────────────┼────────────────────→
    [Качество 2]         |          [Качество 3]
                         |
    [Конкурент Б]        |     [Конкурент В]
                         |
                         ↓
                    [Качество 4]
```

**Типичные оси:**
- Тёплый ↔ Холодный
- Попсовый ↔ Экспертный
- Дорогой ↔ Дешёвый
- Массовый ↔ Персональный
