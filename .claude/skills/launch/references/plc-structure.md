# Структура PLC (Prelaunch Content)

## Что такое PLC

**PLC** — прогревающий контент перед запуском. 3 единицы контента (видео, посты, письма), которые обучают, вызывают доверие и создают предвкушение.

---

## Формула 3 PLC

```
PLC 1: ВОЗМОЖНОСТЬ     →     PLC 2: ТРАНСФОРМАЦИЯ     →     PLC 3: ВЛАДЕНИЕ
   "Почему сейчас"              "Как это работает"           "Что вы получите"
```

---

## PLC 1: Возможность (Opportunity)

### Цель
Показать, что существует возможность, которую аудитория упускает.

### Структура

| Блок | Содержание | Длительность |
|------|------------|--------------|
| **Хук** | Интригующее начало | 30 сек |
| **Обещание** | Что они узнают | 1 мин |
| **История** | Ваш путь к открытию | 3-5 мин |
| **Возможность** | Что изменилось в мире | 5-7 мин |
| **Доказательство** | Кейсы, цифры | 3-5 мин |
| **Тизер PLC 2** | Что будет дальше | 1 мин |
| **CTA** | Поделиться, комментировать | 30 сек |

### Ментальные триггеры
- **Авторитет** — покажите экспертизу
- **Социальное доказательство** — результаты других
- **Предвкушение** — тизер следующего

### Шаблон текста

```
[ХУК]
Что если я скажу, что [общепринятое убеждение] — неправда?

[ИСТОРИЯ]
3 года назад я [ситуация до].
Потом я обнаружил [инсайт].
И всё изменилось: [результат].

[ВОЗМОЖНОСТЬ]
Сейчас [что изменилось в мире].
Это значит, что [новая возможность].
И те, кто поймёт это первыми — [выгода].

[ДОКАЗАТЕЛЬСТВА]
Вот как это работает:
- [Кейс 1]
- [Кейс 2]
- [Статистика]

[ТИЗЕР]
В следующем видео я покажу точный процесс, как [достичь результата].

[CTA]
Напишите в комментариях [вопрос].
```

## PLC 2: Трансформация (Transformation)

### Цель
Показать КАК это работает. Дать реальную ценность.

### Структура

| Блок | Содержание | Длительность |
|------|------------|--------------|
| **Напоминание** | Что было в PLC 1 | 1 мин |
| **Обещание** | Что узнают сегодня | 1 мин |
| **Обучение** | Конкретный метод/процесс | 10-15 мин |
| **Кейс** | Как это сработало | 3-5 мин |
| **Возражения** | Ответы на сомнения | 3-5 мин |
| **Тизер PLC 3** | Что будет дальше | 1 мин |
| **CTA** | Вопросы, шеринг | 30 сек |

### Ментальные триггеры
- **Взаимность** — даёте ценность бесплатно
- **Доверие** — показываете внутреннюю кухню
- **Сообщество** — создаёте диалог

### Шаблон текста

```
[НАПОМИНАНИЕ]
В прошлый раз мы говорили о [возможность].
Вы узнали, что [главный инсайт].

[ОБЕЩАНИЕ]
Сегодня я покажу точный процесс, как [результат].
К концу этого видео вы будете знать [конкретное знание].

[ОБУЧЕНИЕ — Метод из 3-5 шагов]
Шаг 1: [Название]
[Объяснение]
[Пример]

Шаг 2: [Название]
[Объяснение]
[Пример]

Шаг 3: [Название]
[Объяснение]
[Пример]

[КЕЙС]
Вот как [имя] применил этот метод:
- Было: [до]
- Сделал: [действия]
- Стало: [после]

[ВОЗРАЖЕНИЯ]
Возможно вы думаете:
"[Возражение 1]" — на самом деле [ответ]
"[Возражение 2]" — вот почему это не так [ответ]

[ТИЗЕР]
В следующем видео я расскажу, как получить [полное решение].
И у меня есть кое-что особенное для вас...

[CTA]
Какой шаг для вас самый сложный? Напишите в комментариях.
```

## PLC 3: Владение (Ownership)

### Цель
Передать "владение" результатом. Показать полную картину.

### Структура

| Блок | Содержание | Длительность |
|------|------------|--------------|
| **Итог** | Что узнали в PLC 1-2 | 2 мин |
| **Большая картина** | Как всё связано | 3-5 мин |
| **Препятствия** | Что мешает людям | 5-7 мин |
| **Решение** | Как преодолеть | 5-7 мин |
| **Кейс** | Полная история успеха | 3-5 мин |
| **Анонс** | Что вы предлагаете | 3-5 мин |
| **CTA** | Присоединиться | 1 мин |

### Ментальные триггеры
- **Предвкушение** — завтра открываем
- **Сообщество** — станьте частью
- **Редкость** — ограниченные места

### Шаблон текста

```
[ИТОГ СЕРИИ]
За последние дни мы прошли путь:
- PLC 1: Вы узнали о [возможность]
- PLC 2: Вы освоили [метод]
Сегодня — финальный шаг.

[БОЛЬШАЯ КАРТИНА]
Теперь вы видите полную картину:
[Визуализация системы/процесса]
Когда [A] + [B] + [C] = [Результат]

[ПРЕПЯТСТВИЯ]
Но я знаю, что мешает людям начать:
1. [Препятствие 1] — вот почему это не проблема
2. [Препятствие 2] — вот как это решить
3. [Препятствие 3] — вот что делать

[ИСТОРИЯ УСПЕХА]
Хочу рассказать историю [имя].
Когда он/она пришёл ко мне, [ситуация].
Сейчас [результат].
И вот что он/она говорит: "[цитата]"

[АНОНС]
Завтра я открываю [название продукта].
Это [краткое описание].

Внутри:
- [Компонент 1]
- [Компонент 2]
- [Компонент 3]

[РЕДКОСТЬ]
Мест ограничено: [число].
Двери открываются завтра в [время].

[CTA]
Следите за почтой. Завтра в [время] я пришлю ссылку.
```

## Форматы PLC

PLC может быть в разных форматах:

| Формат | Плюсы | Минусы |
|--------|-------|--------|
| **Видео** | Высокое доверие, эмоции | Сложнее производство |
| **Живой стрим** | Интерактив, FOMO | Нужна аудитория |
| **PDF/Guide** | Легко создать | Меньше вовлечения |
| **Серия постов** | Быстро, доступно | Меньше глубины |
| **Email-серия** | Персонально | Ограничен охват |

### Рекомендация по форматам

```
Первый запуск (Seed Launch):
  → Email + посты

Внутренний запуск (свой список):
  → Видео + email

JV/Партнёрский запуск:
  → Видео (для партнёров важно качество)

Live Launch:
  → Стримы + email + посты
```

## Тайминг между PLC

```
День 1-3:    PLC 1
День 4-7:    PLC 2
День 8-11:   PLC 3
День 12:     Открытие корзины
```

### Поддерживающий контент между PLC

| Между | Контент |
|-------|---------|
| PLC 1 → PLC 2 | Напоминание, 1 инсайт, тизер |
| PLC 2 → PLC 3 | Кейс, Q&A, тизер анонса |
| PLC 3 → Open | "Завтра открываем", countdown |

## Checklist создания PLC

### PLC 1
- [ ] Хук написан и проверен
- [ ] История аутентичная, не выдуманная
- [ ] Возможность чётко артикулирована
- [ ] Есть 2-3 доказательства/кейса
- [ ] Тизер PLC 2 интригует
- [ ] CTA понятный

### PLC 2
- [ ] Метод разбит на 3-5 шагов
- [ ] Каждый шаг имеет пример
- [ ] Есть кейс применения
- [ ] Возражения закрыты
- [ ] Тизер PLC 3 создаёт предвкушение

### PLC 3
- [ ] Большая картина ясна
- [ ] Препятствия названы и закрыты
- [ ] История успеха эмоциональная
- [ ] Оффер анонсирован (но не продан)
- [ ] Редкость/срочность обозначены
- [ ] CTA: следить за открытием
