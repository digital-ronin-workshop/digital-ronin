# Экспресс-запуск (Сердитов)

> 5 единиц контента = запуск за 1-5 дней. Самая короткая связка для монетизации блога.

---

## Главная формула

### [FORMULA] 5 вопросов любого прогрева

Любой прогрев — от stories до вебинара — отвечает на 5 вопросов. Раскрой каждый в одной единице контента:

```
ДЕНЬ 1: Реальна ли эта проблема?
ДЕНЬ 2: Можно ли её решить в принципе?
ДЕНЬ 3: Можно ли решить её конкретно МНЕ?
ДЕНЬ 4: Можно ли решить её конкретно ВАШИМ способом?
ДЕНЬ 5: Почему нужно покупать именно у ВАС?
         → ОФФЕР
```

### [PATTERN] Это универсальная схема
Та же структура работает для:
- Экспресс-запуска за 1-5 дней
- Закупки рекламы у блогеров
- Продающего поста
- Продающего вебинара
- Рекламного креатива

Смыслы вытекают друг из друга → проводят клиента по лестнице Ханта от холодного к горячему.

---

## Разбор каждого вопроса

### Вопрос 1: Реальна ли проблема?

**Цель:** Прощупать — откликается ли тема аудитории.

**Как сделать:**
- Story-tell про себя: "Встала на весы — 83 кг при норме 55. Офигела. Решила действовать."
- Голосовалка: "Актуально для вас? Да/Нет"
- Формат: пост / stories / Reels — что сильнее у вас

**Триггеры проблемы:** Проблемы, которые касаются каждого — деньги, здоровье, отношения, вес.

### [ANTI-PATTERN] Мало отклика = эта тема аудитории не нужна
Если голосуют мало — НЕ продолжайте. Тема не резонирует. Ищите другую.

---

### Вопрос 2: Можно ли решить в принципе?

**Цель:** Показать что проблема решаема. НЕ вашим продуктом — просто в принципе.

**Как сделать:**
- Личный кейс
- Кейс ученика
- Просто показать что решение существует
- "С этой проблемой жить не обязательно вечно"

### [ANTI-PATTERN] Кринжовый прогрев
```
❌ "Хочешь похудеть? Как раз со мной можно!"
```
Люди ещё НЕ готовы знакомиться с вашей методикой. Сначала — что проблема решаема в ПРИНЦИПЕ.

---

### Вопрос 3: Можно ли решить конкретно МНЕ?

**Цель:** Закрыть возражения: "У них-то понятно, а у меня не прокатит."

**Как сделать:**
1. Выпиши ВСЕ углы возражений (mind-карта или список)
2. Возьми самые популярные
3. Создай единицу контента, закрывающую каждое

**Типичные возражения (пример: похудение):**
- Напряжённый график
- Сидячая работа
- Есть дети
- Пробовала 350 раз — не помогло

### [ANTI-PATTERN] Слишком много мотивации
- Прогрев ТОЛЬКО на мотивации = вода водянистая
- "Встань и иди" без конкретики → аудитория задолбалась
- Мотивация = максимум 1/5 от запуска

---

### Вопрос 4: Можно ли решить ВАШИМ способом?

**Цель:** Презентация продукта — но НЕ вываливать всё.

**Как сделать:**
- Несколько убойных killer-feature
- Почему ИМЕННО ваша методика работает
- Конкурентная отстройка (не "все остальные плохие", а "мой метод уникален потому что...")
- Пруфы: "По этой методике похудели 1500 из 1600 человек"

### [ANTI-PATTERN] Что НЕ делать
```
❌ Не вываливать ВСЁ содержание продукта
❌ Не продавать длительность ("курс 3 месяца")
❌ Не показывать 16 stories про чат и мини-группы
```
Продавать ПОЧЕМУ методика решит проблему, а НЕ из чего состоит.

---

### Вопрос 5: Почему покупать у ВАС?

**Цель:** Подтверждение экспертности.

**Как сделать:**
- Сильные кейсы
- Личная история
- Показать что вы — точка Б для аудитории

**Преимущество блога:** Даже 10 человек охвата → 7 незнакомых смотрят вас, потому что вы для них авторитет/точка Б. Они УЖЕ готовы покупать.

---

## После 5 единиц контента

### [TECHNIQUE] Выкатка оффера

**Формат не важен:**
- Лендинг
- Просто предложение в Stories
- "Пиши в директ: ХОЧУ НА КУРС"
- Ссылка на оплату

Как только дашь возможность купить — купят.

---

## Вариант за 1 день

### [TECHNIQUE] Экспресс-запуск за 1 день

Все 5 смыслов раскрываются в одной линейке Stories / одном длинном посте / одном эфире.

```
1/5 линейки: Проблема реальна
2/5: Решаема
3/5: Решаема для тебя
4/5: Мой метод
5/5: Почему я + ОФФЕР
```

---

## Вариант для закупки рекламы

### [TECHNIQUE] Та же схема для рекламных креативов

При закупке рекламы у блогеров выстраивай подачу по этим же 5 смыслам:
1. Проблема реальна (хук)
2. Решаема (надежда)
3. Для тебя тоже (преодоление возражений)
4. Вот способ (killer-feature)
5. Вот эксперт (доверие) → CTA

Если подача выстроена по другой схеме → приход слабый.
Если по этой → качественная, целевая, готовая покупать аудитория.

## Чек-лист

### [CHECKLIST] Экспресс-запуск за 5 дней

- [ ] День 1: Единица контента "проблема реальна" + голосовалка
- [ ] Проверить отклик: если мало — сменить тему
- [ ] День 2: Единица контента "проблема решаема" (кейс/пример)
- [ ] День 3: Единица контента "решаема для ТЕБЯ" (закрытие возражений)
- [ ] День 4: Презентация метода (2-3 killer-feature, НЕ полная программа)
- [ ] День 5: Экспертность + кейсы + ОФФЕР
- [ ] Оффер: ссылка/директ/лендинг — любой формат
- [ ] Дедлайн: ограничение по времени или количеству мест

## Edge Cases и Q&A (из эфира)

### [ANTI-PATTERN] Ошибки в создании ажиотажа

Если мало реакций на первых этапах — проблема НЕ в технике ажиотажа, а в одном из трёх:

| Причина | Пример |
|---------|--------|
| **Не болит** | Предложили то, что аудитории не нужно |
| **Триггер нереалистичный** | 2000 подписчиков, 200 охват — и "1000 мест". Люди не верят |
| **Неадекватные ограничения** | Слишком мало времени (оффер в 16:00, дедлайн в 24:00 = ощущение манипуляции). Слишком много времени ("до 1 ноября" → придут 30-31 октября) |

**Оптимальный дедлайн:** 1-3 дня. Достаточно для принятия решения, недостаточно чтобы забыть.

### [PATTERN] Работа с кейсами при суточном прогреве

Когда мало кейсов или есть и большие, и маленькие:

```
❌ Только крупные кейсы (10 млн, 4.5 млн) → пугают нулевичков
❌ Один и тот же кейс каждые 2 дня → "3 года прошло, а он всё в тех же плавках"
✅ Чередовать крупные и мелкие → чтобы менчилось с картиной мира аудитории
✅ Если кейс один — закрывать через storytelling, боли, потребности
✅ Свой личный кейс — первый, базовый. Начинай с себя
```

### [ANTI-PATTERN] Опросники не показывают готовность покупать

```
Реагировать эмоциями ≠ реагировать деньгами

- Реакции в stories показывают ИНТЕРЕС, но НЕ платёжеспособность
- Люди врут на CustDev: "Планирую 3 млн до конца года" при доходе 50к
- По реакциям можно понять только: есть СПРОС или нет (30%+ от охвата = можно пробовать)
- Но 30% реакций ≠ 30% купят
```

### [TECHNIQUE] Скипнуть анкету — сразу на созвон

Для маленьких аккаунтов (нет потока на 10 000 заполненных анкет):

```
Вместо: Анкета Google Forms → анализ → созвон
Делай:  Лид-форма на диагностику → созвон

= Низкострессовый путь. Люди просто оставляют заявку →
вы/ассистент назначаете время → каздевите + продаёте на созвоне.
```

### [ANTI-PATTERN] Высокострессовые анкеты

Ошибки при составлении анкеты для CustDev:

```
❌ Первые 5 вопросов — контакты (инста, телеграм, почта, телефон)
   → Собираете то, что не нужно. Если воронка в инсте — нужна только ссылка на инсту

❌ Открытые вопросы, требующие длинных ответов
   → Люди спотыкаются и закрывают анкету навсегда

✅ Предлагайте варианты ответов (люди выбирают, а не пишут)
✅ Фильтр для каждого вопроса: "Низкострессово ли это?"
✅ Адаптируйте под нишу (кофемейкеру нет смысла спрашивать "какие проблемы закрывает банка кофе")
```

### [PATTERN] Экспресс-запуск НЕ подходит для клубов

Закрытый клуб/канал → продвигать через вечнозелёные воронки, НЕ через пятидневный запуск.

### [PATTERN] Ценообразование (быстрая формула)

```
Цена продукта = max(Рынок, Ценность)

1. Рынок: что дают конкуренты и по какой цене
2. Ценность: стоимость вашего часа × количество часов участия
3. Продакшн НЕ влияет на цену (влияет только на ваши расходы)
```

### [PATTERN] После запуска: доводите до конца

Внутри курса всегда продавайте людям идею пройти его до конца:
- Задача = получить результат и замерить NPS
- Если не доведёте — потеряете кейсы и получите возвраты
- Короткий путь от идеи до действия = главный принцип (пример: Аяз внедрял прямо во время мастер-класса Гусейна)

### [ANTI-PATTERN] Аудитория выгорает от запусков

- Аудиторию нужно ОБНОВЛЯТЬ (платный + бесплатный трафик)
- Признак выгорания: падение охвата + вовлечённости + спроса
- Не пережигайте аудиторию: экспресс-запуск → пауза → обновление аудитории → следующий запуск
- Если после запуска хотите ещё один — меняйте тему/оффер
