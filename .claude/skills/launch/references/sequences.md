## 1. Пре-предзапуск (Shot Across the Bow)

### Цели
- Вызвать интерес
- Собрать обратную связь
- Понять возражения
- Начать формировать список запуска

### Формат сообщения

```
Привет!

Работаю над [новым продуктом/курсом/программой], который поможет [результат].

Прежде чем завершить, хочу убедиться, что он решит ваши задачи.

Подскажите: что для вас самое сложное в [теме]?

Буду признателен за ответ.
```

### Триггеры этапа
- Любопытство
- Взаимность (просите помощи)
- Вовлечение

---

## 2. Предзапуск (Prelaunch Content)

### Структура 3 PLC

| PLC | Тема | Цель | Триггеры |
|-----|------|------|----------|
| **PLC 1** | Возможность | "Почему это важно сейчас" | Авторитет, Социальное доказательство |
| **PLC 2** | Трансформация | "Как это работает" | Взаимность, Доверие |
| **PLC 3** | Владение | "Что вы получите" | Предвкушение, Сообщество |

### Расписание писем предзапуска

```
День 1:  "PLC 1 уже доступен" (ссылка на видео/контент)
День 2:  Напоминание + 1 инсайт из PLC 1
День 4:  "PLC 2 уже здесь" + тизер
День 5:  Кейс/история успеха
День 7:  "PLC 3 — финальное видео"
День 8:  "Завтра открываем!"
```

## 3. Открытая корзина (Open Cart)

### 7-дневная последовательность

| День | Тема письма | Фокус |
|------|-------------|-------|
| **День 1** | "Мы открылись!" | Объявление + полный оффер |
| **День 2** | Кейс #1 | Социальное доказательство |
| **День 3** | FAQ | Работа с возражениями |
| **День 4** | Бонус | Добавление ценности |
| **День 5** | Кейс #2 | Ещё одна история успеха |
| **День 6** | "Осталось 24 часа" | Срочность |
| **День 7** | "Последний шанс" | Дедлайн (2-3 письма) |

### Структура письма "Мы открылись"

```
1. Короткое напоминание о PLC
2. Объявление: "Двери открыты"
3. Ссылка на страницу продаж
4. Краткий список что включено
5. Гарантия
6. Дедлайн
7. P.S. с главной выгодой
```

### Последний день — 3 письма

```
Утро:    "Сегодня последний день"
День:    "Осталось 6 часов" + ответы на частые вопросы
Вечер:   "Закрываем через 2 часа" (за 2-4 часа до конца)
```

## 4. Постзапуск

### Для купивших

| День | Действие |
|------|----------|
| День 0 | Приветственное письмо + доступ |
| День 1 | "Как начать" — первый шаг |
| День 3 | Проверка: "Как дела? Есть вопросы?" |
| День 7 | Бонус-сюрприз |

### Для НЕ купивших

| День | Действие |
|------|----------|
| День 1 | "Двери закрылись" (подтверждение) |
| День 3 | Опрос: "Почему не подошло?" |
| День 7 | Альтернатива или wait-list |

## Seed Launch (первый запуск)

Для тех, у кого нет списка и продукта.

### Структура

```
1. Идея продукта (гипотеза)
       ↓
2. Пред-предзапуск (опрос, обратная связь)
       ↓
3. Предзапуск (3 единицы контента)
       ↓
4. Продажа в "живом" формате (вебинар/созвон)
       ↓
5. Создание продукта ВМЕСТЕ с первыми клиентами
       ↓
6. Кейсы для следующего запуска
```

### Особенности Seed Launch
- Продукт ещё не создан
- Цена ниже финальной
- Обещание: "Создаём вместе"
- Ограничение мест (10-30)
- Результат: продукт + кейсы + деньги

## JV/Affiliate Launch

### Структура работы с партнёрами

```
За 4-6 недель: Приглашение партнёров
За 2-3 недели: Материалы для промо (swipe copy)
За 1 неделю:   Напоминание + бонусы для топ-партнёров
Во время:      Ежедневный leaderboard
После:         Выплаты + благодарности
```

### Что дать партнёрам
- Email swipe copy (5-7 писем)
- Banners (3-4 размера)
- Social media posts
- Affiliate link + tracking
- Комиссия (30-50%)

## Live Launch (со стримами)

### Структура

```
Стрим 1 (PLC 1): Обзор + Возможность
Стрим 2 (PLC 2): Глубокое обучение
Стрим 3 (PLC 3): Q&A + Анонс оффера
Стрим 4:         Продажа в прямом эфире
```

### Преимущества Live Launch
- Выше вовлечение
- Instant feedback
- FOMO (пропустишь — не узнаешь)
- Возможность продавать в моменте
