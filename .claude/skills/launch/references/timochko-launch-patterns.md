# Единая система запусков: PLF × Тимочко

> Синтез западной Product Launch Formula (Jeff Walker) и российской школы запусков (Тимочко).
> Не сумма двух систем — а единая методология, где каждый элемент усиливает другой.

---

## Философия: почему нужны ОБЕ системы

PLF даёт **архитектуру** — структурированные последовательности, PLC-видео, email-цепочки. Это скелет.
Тимочко даёт **плоть** — lifestyle-контент, сюжетные линии, психологию Instagram-аудитории. Это энергия.

PLF без Тимочко = сухой запуск, который не цепляет эмоционально.
Тимочко без PLF = хаотичный прогрев без чёткой конвертации.

**Правило:** Используй PLF для структуры запуска. Используй Тимочко для наполнения контентом и прогрева.

## ЧАСТЬ 1: ДИАГНОСТИКА ПЕРЕД ЗАПУСКОМ

### 1.1 Шесть паззлов (обязательная проверка)

Перед любым запуском проверь ВСЕ 6. Отсутствие одного — провал.

| # | Паззл | Проверочный вопрос | Красные флаги |
|---|-------|--------------------|---------------|
| 1 | **Денежная тема** | Ниша решает потребность первого порядка? | Хобби, «духовность», узкие софт-скиллы |
| 2 | **Сильный спикер** | Есть результат + 2+ года + довольные клиенты? | Нет кейсов, не может говорить на камеру, нет финансовой мотивации |
| 3 | **Аудитория** | Охват сториз 400+, подписчики органические? | Гивы, накрутки, охват <150 |
| 4 | **Прогрев + продажи** | Есть системный план воздействия? | «Просто расскажу и купят» |
| 5 | **Качественный продукт** | Результат повторяем для других? | Нет методологии, только «мой опыт» |
| 6 | **Постоянство** | Готов делать запуски регулярно? | Одноразовый проект |

### 1.2 Набор счастливчика (продукт)

7 критериев продукта, который БУДЕТ продаваться:

| Критерий | Что значит | Как проверить |
|----------|-----------|---------------|
| **Денежная ниша** | Заработок > здоровье > отношения | Есть ли конкуренты с выручкой >10 млн? |
| **Первая потребность** | Не «хочу», а «надо» | Будут ли платить в кризис? |
| **Конкретный метод** | Не «развитие», а «10к подписчиков за 30 дней» | Можно ли измерить результат цифрой? |
| **Волшебная таблетка** | Легко + быстро + секретно + мощный результат | Есть ли «вау-реакция» при описании? |
| **Чек от 50к₽** | Маржинальность для школы | Если продаётся за 30к — можно и за 50к+ |
| **Рассрочки** | Обязательно | Без рассрочки оборот в 5-7 раз ниже |
| **Соответствие ЦА** | Аудитория может и хочет это купить | Кастдевы подтвердили? |

**Формула:** если продукт набирает <5 из 7 — переделай перед запуском.

### 1.3 Оценка эксперта (для продюсера)

11 вопросов. Ответ «нет» на 3+ из первых 7 — не берись.

| # | Вопрос | Минимум |
|---|--------|---------|
| 1 | Востребованная ниша? | Есть конкуренты с выручкой |
| 2 | Охваты в сториз? | 300+ (лучше 400-500) |
| 3 | Откуда подписчики? | Органика, НЕ гивы |
| 4 | Как давно ведёт блог? | 6+ месяцев активно |
| 5 | Сформирован контекст? | Люди понимают чем занимается |
| 6 | Есть крутые клиенты? | Хотя бы 3-5 кейсов |
| 7 | Есть крутые ученики? | Хотя бы 1-2 с результатом |
| 8 | Были ли запуски? | Любой опыт — плюс |
| 9 | Насколько успешны? | Не критично, но показательно |
| 10 | Есть запрос от аудитории? | Спрашивают «а есть курс?» |
| 11 | Реакция на экспертный контент? | Сохранения, репосты, вопросы в директ |

**Обязательные черты:** вовлечённость, финансовая мотивация, самоорганизация, вера в продукт, способность говорить на камеру.

**Особый случай — «блогер»:** аудитория 5000+ в сториз компенсирует слабую экспертизу. Можно строить запуск вокруг аудитории, докручивая продукт.

### 1.4 Концепт продукта (8 вопросов)

Ответь на все 8 ДО начала прогрева:

1. **Для кого этот курс?** — конкретный аватар, не «все»
2. **Главный результат курса?** — одна фраза, измеримая
3. **Путь человека на курсе?** — точка А → точка Б
4. **Программа курса?** — модули и темы
5. **Формат обратной связи?** — чат, созвоны, проверки
6. **Что ещё входит?** — бонусы, доступы, сообщество
7. **Цена и тарифы?** — минимум 2 тарифа + рассрочка
8. **Гарантии?** — возврат, доступ, результат

## ЧАСТЬ 2: ЕДИНАЯ КАРТА ЗАПУСКА

### 2.1 Объединённый таймлайн

```
ФАЗА 0: ПОДГОТОВКА                         │ 1-3 мес до запуска
  Тимочко: создание контекста в блоге       │ 30-40 контентных единиц
  PLF: сбор/прогрев списка                  │ лид-магниты, подписчики
                                             │
ФАЗА 1: ПРЕ-ПРОГРЕВ                        │ 30-10 дней до запуска
  Тимочко: мягкий прогрев (lifestyle)       │ тема всплывает «фоново»
  PLF: пре-предзапуск (shot across bow)     │ разведка + обратная связь
                                             │
ФАЗА 2: АКТИВНЫЙ ПРОГРЕВ                   │ 10-3 дня до бесплатника
  Тимочко: анонс идеи + активный прогрев    │ сториз, инфоповоды, кейсы
  PLF: PLC 1 → PLC 2                       │ видео/посты с ценностью
                                             │
ФАЗА 3: КОНВЕРСИЯ                          │ 3-1 день до продаж
  Тимочко: бесплатник + кульминация         │ вебинар/гайд + пик напряжения
  PLF: PLC 3 + анонс оффера                │ «завтра открываем»
                                             │
ФАЗА 4: ПРОДАЖИ                            │ 5-7 дней
  Тимочко: ажиотаж + кейсы + дедлайн       │ stories каждый день
  PLF: Open Cart sequence                   │ email-цепочка 7 дней
                                             │
ФАЗА 5: ПОСТЗАПУСК                         │ 1-2 недели
  Тимочко: упаковка кейсов для след. запуска│ контент о результатах
  PLF: onboarding + follow-up              │ welcome-серия + опрос
```

### 2.2 Где PLF усиливает Тимочко

| Слабость Тимочко | Решение из PLF |
|------------------|----------------|
| Хаотичный прогрев без структуры | PLC 1→2→3: чёткая последовательность «Возможность→Трансформация→Владение» |
| Нет системы email-маркетинга | Последовательности писем для каждого этапа |
| Слабый постзапуск | Welcome-серия + опрос не купивших + альтернатива |
| Нет JV/партнёрской модели | JV Launch: swipe copy, leaderboard, комиссии |
| Зависимость от одной платформы | Мультиканальность: email + видео + соцсети |

### 2.3 Где Тимочко усиливает PLF

| Слабость PLF | Решение из Тимочко |
|--------------|--------------------|
| Сухие PLC-видео без эмоций | Lifestyle-контент, сюжетные линии, «душа наружу» |
| Нет работы с аудиторией ДО запуска | Фаза 0: месяцы создания контекста и доверия |
| Абстрактные триггеры | 18 конкретных триггеров + шкала убедительности |
| Нет диагностики продукта | «Набор счастливчика» + 6 паззлов |
| Не учитывает Instagram-механики | 8 типов контента, правила сториз, охваты |
| Нет модели продюсирования | Система работы «продюсер + эксперт» |

## ЧАСТЬ 3: ПРОГРЕВ — ПОЛНАЯ СИСТЕМА

### 3.1 Четыре чекбокса в голове клиента

Прогрев — это последовательная установка 4 галочек в голове покупателя:

```
☐ НИША    → «Точно ли эта тема решит мою потребность?»
☐ ЭКСПЕРТ → «Подходит ли МНЕ именно этот эксперт?»
☐ ВЕРА    → «Получится ли У МЕНЯ?»
☐ МОМЕНТ  → «Хочу ли я покупать ИМЕННО СЕЙЧАС?»
```

**Правило последовательности:** чекбоксы ставятся ТОЛЬКО в этом порядке. Нельзя продавать «момент» (скидка, дедлайн), пока не закрыты «ниша» и «эксперт».

| Чекбокс | Контент для установки | Триггеры | Фаза запуска |
|---------|----------------------|----------|--------------|
| **НИША** | Тренды рынка, статистика, истории успеха в нише, «почему сейчас» | Авторитет, конкретность | Фаза 1 (мягкий прогрев) |
| **ЭКСПЕРТ** | Результаты, кейсы, lifestyle, ценности, история, «душа наружу» | Соц.доказательство, симпатия, доверие | Фаза 1-2 |
| **ВЕРА** | Кейсы учеников ПОХОЖИХ на ЦА, методология, «легко+быстро», демонстрация | Соц.доказательство, контраст, история | Фаза 2-3 |
| **МОМЕНТ** | Дедлайн, ограничение мест, бонусы, ажиотаж, «цена вырастет» | Дефицит, срочность, FOMO, потери | Фаза 4 |

**Связь с PLF:**
- PLC 1 (Возможность) → устанавливает чекбокс НИША
- PLC 2 (Трансформация) → устанавливает чекбоксы ЭКСПЕРТ + ВЕРА
- PLC 3 (Владение) → устанавливает чекбокс МОМЕНТ

### 3.2 Семь этапов прогрева (детально)

#### Этап 1: Регулярный контент (постоянно)

**Цель:** сформировать контекст — аудитория понимает, кто ты и чем занимаешься.

**Критерий готовности:** люди сами пишут в директ с вопросами по теме.

Пропорция: **30% экспертность / 70% личность**.

Экспертный контент:
- Разборы проектов
- Ошибки новичков
- Инсайты и лайфхаки
- Тренды и прогнозы
- Ответы на вопросы подписчиков

Личный контент:
- Как проходит день
- Цели и мечты
- Сложности и провалы
- Отношения и ценности
- Покупки на заработанные деньги
- Путешествия и приключения

**Минимум:** 30-40 контентных единиц за 1-2 месяца.

**Связь с PLF:** это аналог «наращивания списка» — но через контент, а не через лид-магниты. Идеально: делать ОБА.

#### Этап 2: Мягкий прогрев (10-30 дней до запуска)

**Цель:** тема продукта начинает «всплывать» органично в контенте.

**Как:** не говоришь «я делаю курс», а показываешь контекст:
- Lifestyle от профессии (покупки, путешествия, свобода)
- Тренды и возможности в нише
- «Случайные» истории успеха знакомых/клиентов
- Увлечённость своей темой
- Рабочие процессы

**Интенсивность:** низкая. Тема — одна из многих в блоге.

**Связь с PLF:** это растянутый пре-предзапуск. PLF рекомендует «shot across the bow» — одно письмо. Тимочко превращает это в 10-30 дней нативного погружения. Комбинируй: делай мягкий прогрев в сториз + отправляй shot across the bow на email-список.

#### Этап 3: Анонс идеи (2-3 дня)

**Цель:** показать, что продукт создаётся по запросу аудитории, а не «впаривается».

**Техника демонстрации спроса:**
- «1000 заявок за сутки»
- Скриншоты сообщений в директ
- Опрос: «Хотели бы?» → показать результат
- «Вы столько раз просили — я решил сделать»

**Триггеры:** социальное доказательство, группа, любопытство.

**Связь с PLF:** нет прямого аналога. Уникальная техника Тимочко — создание ажиотажа ДО объявления продукта.

#### Этап 4: Активный прогрев (5-10 дней)

**Цель:** максимально «разогреть» аудиторию через все каналы воздействия.

**Конструктор контента для активного прогрева:**

| Блок | Что публиковать | Какой чекбокс закрывает |
|------|-----------------|------------------------|
| **Кейсы клиентов** | До/после, процесс, отзыв | ВЕРА |
| **Кейсы учеников** | Результаты, истории трансформации | ВЕРА |
| **Экспертиза** | Разборы, методология, «кухня» | ЭКСПЕРТ |
| **Инфоповоды** | Яркое событие + экспертный смысл | НИША + ЭКСПЕРТ |
| **Ажиотаж** | Количество заявок, очереди, вопросы | МОМЕНТ |
| **Lifestyle** | Результат профессии в жизни | НИША + ЭКСПЕРТ |
| **Возражения** | Ответы на «дорого», «не получится», «не мой уровень» | ВЕРА |

**Связь с PLF:** это PLC 1 + PLC 2, но в формате ежедневных сториз вместо двух больших видео. Комбинируй: PLC-видео для email-подписчиков + ежедневный прогрев в сториз.

#### Этап 5: Бесплатник (2-3 дня)

**Цель:** дать ценность + собрать горячих лидов + финальный прогрев перед продажами.

**Форматы:** вебинар, мастер-класс, гайд, марафон, серия уроков.

**Структура бесплатника (синтез PLF + Тимочко):**
1. Хук — зачем смотреть (30 сек)
2. История эксперта — путь + уязвимость (5 мин)
3. Возможность — почему сейчас (5 мин)
4. Обучение — конкретный метод, 3-5 шагов (15-20 мин)
5. Кейсы — 2-3 истории трансформации (5-10 мин)
6. Переход к продукту — «но чтобы получить полный результат...» (5 мин)
7. Оффер — описание продукта (5-7 мин)
8. Закрытие возражений — FAQ (5 мин)
9. CTA — как записаться (2 мин)

**Связь с PLF:** это PLC 3 (Владение) в живом формате. PLF рекомендует записанное видео, Тимочко — живой вебинар. Живой формат даёт FOMO + интерактив + продажи в моменте.

#### Этап 6: Кульминация (1 день)

**Цель:** максимальное эмоциональное напряжение перед открытием продаж.

**Техники:**
- Countdown (обратный отсчёт)
- «Завтра в [время] я открою запись»
- Последние отзывы и кейсы
- Личная история «почему я это делаю»
- Предупреждение об ограничении мест

**Связь с PLF:** это последний день перед Open Cart. PLF использует email-countdown. Тимочко добавляет stories-драматургию.

#### Этап 7: Продажи (5-7 дней)

**Цель:** конвертировать максимум прогретой аудитории в покупателей.

**Комбинированная последовательность:**

| День | Email (PLF) | Stories (Тимочко) |
|------|------------|-------------------|
| **1** | «Мы открылись!» + полный оффер | Анонс + эмоции + первые продажи |
| **2** | Кейс #1 | Отзывы в реальном времени |
| **3** | FAQ + возражения | Ответы на вопросы из директа |
| **4** | Бонус (добавление ценности) | «Осталось X мест» + новый кейс |
| **5** | Кейс #2 | Сомневающимся: разбор страхов |
| **6** | «Осталось 24 часа» | Countdown + последние истории |
| **7** | 3 письма: утро/день/вечер | «Закрываем через 2 часа» |

### 3.3 Конструктор прогрева (11 модулей)

Готовые модули, из которых собирается прогрев. Выбери 20-40 идей, расставь по датам.

**Модуль 1: Фоновый контент** — контекст и доверие
- Как проходит рабочий день
- За кулисами проектов
- Личная жизнь и ценности
- Философия и принципы работы
- Цели на период

**Модуль 2: Стиль жизни** — демонстрация результата профессии
- Покупки на заработанные деньги
- Путешествия и локации
- Свобода выбора и графика
- Красивые рабочие пространства
- «Обычный день» который для многих — мечта

**Модуль 3: Продажа темы** — формирование потребности в нише
- Тренды рынка + цифры
- Истории успеха из индустрии
- «Почему именно сейчас»
- Что изменилось за последний год
- Упущенные возможности (чекбокс НИША)

**Модуль 4: Экспертиза** — демонстрация компетенции
- Разборы реальных проектов
- Топ ошибок новичков
- Неочевидные инсайты
- Методология (фрагмент)
- Прогнозы и аналитика

**Модуль 5: Инфоповоды** — привлечение внимания через события
- Яркое событие + экспертный смысл
- Комментарий к новостям отрасли
- Реакция на тренды
- Позиционирование через дискуссию
- «Инфоповод — король привлечения внимания»

**Модуль 6: Клиенты и проекты** — социальное доказательство
- Работы до/после
- Процесс работы с клиентом
- Отзывы и благодарности
- Интересные кейсы
- Неожиданные результаты

**Модуль 7: Ученики и комьюнити** — демонстрация обучающих способностей
- Результаты учеников (чекбокс ВЕРА)
- Истории трансформации
- Активность сообщества
- Взаимопомощь участников
- «Похожие на тебя уже сделали»

**Модуль 8: Продукт** — подготовка к продажам
- Работа над созданием продукта
- Тестирование подхода
- Улучшения по обратной связи
- Уникальность метода
- «Для кого это» и «для кого НЕ»

**Модуль 9: Ажиотаж** — создание FOMO
- Количество заявок / предзаписей
- Скриншоты директа с вопросами
- Очередь на запись
- Восторженные отзывы участников прошлых потоков
- «Мест осталось Х из Y»

**Модуль 10: Бесплатник** — финальный прогрев
- Анонс бесплатного мероприятия
- Ценность: что получат бесплатно
- Ограничение мест/времени
- Напоминания и countdown
- Результаты участников бесплатника

**Модуль 11: Продажи** — конвертация
- Анонс открытия
- Полное описание оффера
- Ответы на возражения
- Истории последних сомневающихся
- Дедлайн и финальное закрытие

### 3.4 Алгоритм сборки прогрева

```
1. Анализ ЦА → кастдевы → выявление болей и возражений
2. Заполнить 4 чекбокса: какой контент закроет каждый
3. Выбрать модули из конструктора (п.3.3)
4. Отобрать 20-40 конкретных идей
5. Расставить по таймлайну (п.2.1):
   - Фаза 1: модули 1-3 (контекст + lifestyle + тема)
   - Фаза 2: модули 4-7 (экспертиза + кейсы + ажиотаж)
   - Фаза 3: модули 8-10 (продукт + бесплатник)
   - Фаза 4: модуль 11 (продажи)
6. Добавить триггеры (п.4)
7. Проверить: все 4 чекбокса закрыты?
```

## ЧАСТЬ 5: КОНТЕНТ-АРХИТЕКТУРА

### 5.1 Восемь типов контента

| # | Тип | Функция | Частота | Пример |
|---|-----|---------|---------|--------|
| 1 | **Лайф** | Связующая нитка между всем контентом | Ежедневно | Утренний кофе, дорога, быт |
| 2 | **Сюжетные линии** | Удержание на много дней (сериал) | 1-2/мес | «Как мы делаем запуск на 10 млн» |
| 3 | **Линии одного дня** | Событийность | По событиям | «День на конференции» |
| 4 | **Умная мысль** | Экспертное позиционирование | 2-3/нед | Инсайт + контекст |
| 5 | **Сторителлинг** | Эмоции + смыслы | 1-2/нед | История из прошлого с уроком |
| 6 | **Инфоповоды** | Привлечение нового внимания | По новостям | Реакция на тренд + экспертный вывод |
| 7 | **Душа наружу** | Максимальное доверие | По настроению | Откровенный рассказ о сложностях |
| 8 | **Общение** | Вовлечение и feedback | Постоянно | Вопросы, опросы, реакции |

### 5.2 Сюжетные линии — уникальная техника Тимочко

**Что это:** история с развитием, которая тянется дни или недели, как сериал.

**Структура:**
```
ГЕРОЙ + ЦЕЛЬ + ПРЕПЯТСТВИЯ + ВЫБОР + ЭМОЦИИ + ИНТРИГА (= продолжение завтра)
```

**Примеры для запуска:**
- «Как мы готовим запуск курса» — показывать процесс изнутри
- «Первый ученик на бета-тесте» — его путь и результаты
- «Моя история: от нуля до [результат]» — растянуть на неделю

**Правила:**
- Каждый эпизод должен быть понятен с нуля (не все смотрят с начала)
- Объяснять ситуацию, мотивацию и действия героя
- Заканчивать интригой: «а что произошло дальше — расскажу завтра»
- Не начинать историю с середины

**Связь с PLF:** PLF не использует сериальный формат. Это уникальное преимущество Instagram-запусков. Сюжетная линия может БЫТЬ вашим PLC — растянутым на дни вместо одного длинного видео.

### 5.3 Правило «блог как сериал»

Люди приходят в Instagram за 4 вещами:
1. **Эмоции** — кайфовать, отдыхать
2. **Решение задач** — найти ответ на вопрос
3. **Подглядывание** — как живут другие
4. **Ролевая модель** — найти пример для подражания

**Критически важно:** люди заходят НЕ учиться. Они заходят кайфовать. Поэтому:
- Экспертность упаковывай в эмоции и истории
- Не делай «лекции в сториз»
- Простота > глубина
- Визуал > текст
- Показывай > рассказывай

### 5.4 Антипаттерны контента (что убивает охваты и запуски)

**Смертельно:**
- Портянки текста в сториз
- Только говорящая голова
- Узкоэкспертный контент без эмоций
- Абстрактные мотивационные фразы
- Рекламные призывы каждый день
- Слишком сложный контент

**Что поднимает охваты:**
- Сюжетные линии (удержание)
- Личные темы (интерес)
- Юмор и лёгкость (отдых мозгу)
- Визуальный контент (Instagram = визуальная платформа)
- Взаимодействие с людьми (алгоритм)
- Милота (кошки, дети, собаки)
- Игры и развлечения

### 5.5 Пять правил сториз

1. **Наглядность** — всё показывать: скрины, фото, видео. Слова — самый слабый формат.
2. **Смена ракурсов** — каждые 2-3 кадра менять визуал. Монотонность = пролистывание.
3. **Текст — король** — сториз смотрят «одним глазом, выходя из лифта». Крупный шрифт, контраст.
4. **Простота** — одна мысль = один кадр. Понятно с первого взгляда.
5. **Иерархия текста** — важное крупнее, детали мельче.

|------|---------------------|
| **Эксперт** | Контент, экспертиза, съёмка, вебинары |

### 7.2 Финансовая модель

| Уровень продюсера | % от прибыли | Условие |
|-------------------|--------------|---------|
| Новичок | 30-40% | Первые запуски |
| Опытный | 40-50% | Стабильные результаты |
| Частичная работа | 15-20% | Не все функции |
| С крупными экспертами | договорной | «10% от большого > 70% от малого» |

**KPI-лестница (пример):**
- Выручка 2 млн = 20%
- Выручка 2.5 млн = 30%
- Выручка 3 млн = 35%
- Выручка 4.5 млн = 50%

**Правило:** «Вы получаете не то, что заслужили, а то, на что договорились». Возвраты — ответственность эксперта, не продюсера.

### 7.3 Распаковка эксперта

**Техника:** интервью по готовым вопросам для извлечения контента.

**Что извлекать:**
- 2 раздела: экспертность + личность/жизнь
- Один ответ = несколько единиц контента
- Вехи и ключевые моменты биографии

**Результат:** 50-300 идей для контента из одной распаковки.

**Пропорция использования:** 30% экспертность / 70% лайф.

## ЧАСТЬ 10: ЦИТАТЫ-ЯКОРЯ
