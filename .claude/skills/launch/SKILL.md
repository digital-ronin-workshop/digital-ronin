## Формула запуска

### Единый таймлайн (PLF + Тимочко)

```
ФАЗА 0: ПОДГОТОВКА (1-3 мес)     ФАЗА 1: ПРЕ-ПРОГРЕВ (30-10 дн)     ФАЗА 2: АКТИВНЫЙ (10-3 дн)
  Контекст + список                 Мягкий прогрев + разведка            PLC 1-2 + сториз
         ↓                                    ↓                                  ↓
ФАЗА 3: КОНВЕРСИЯ (3-1 дн)      ФАЗА 4: ПРОДАЖИ (5-7 дн)           ФАЗА 5: ПОСТЗАПУСК (1-2 нед)
  Бесплатник + PLC 3               Open Cart + сториз                  Кейсы + onboarding
```

### 6 паззлов (проверка перед запуском)
```
ДЕНЕЖНАЯ ТЕМА + СИЛЬНЫЙ СПИКЕР + АУДИТОРИЯ + ПРОГРЕВ/ПРОДАЖИ + КАЧЕСТВЕННЫЙ ПРОДУКТ + ПОСТОЯНСТВО
```
*Отсутствие любого паззла = провал. Проверяй ВСЕ 6.*

### 4 чекбокса в голове клиента
```
☐ НИША    → «Точно ли эта тема решит мою проблему?»
☐ ЭКСПЕРТ → «Подходит ли МНЕ этот человек?»
☐ ВЕРА    → «Получится ли У МЕНЯ?»
☐ МОМЕНТ  → «Нужно ли покупать ИМЕННО СЕЙЧАС?»
```
*Чекбоксы ставятся ТОЛЬКО в этом порядке.*

## Типы запусков

### Дерево выбора
```
Есть аудитория? → НЕТ → Seed Launch (PLF + набор аудитории)
                → ДА  → Есть email-список? → НЕТ → Stories Launch (Тимочко)
                                             → ДА  → Есть партнёры? → НЕТ → Internal (PLF+Тимочко)
                                                                     → ДА  → JV Launch (PLF)
```

| Тип | Аудитория | Продукт | Подход |
|-----|-----------|---------|--------|
| **Seed Launch** | <500 | Нет | PLF: создавать с первыми клиентами |
| **Stories Launch** | 500-5000 в сториз | Есть | Тимочко: 7 этапов прогрева |
| **Internal Launch** | Email 1000+ | Есть | PLF + Тимочко: полная комбинация |
| **JV/Affiliate** | Через партнёров | Есть | PLF: swipe copy + комиссии |
| **Lifestyle Launch** | Блогер 5000+ | Любой | Тимочко: через демонстрацию жизни |
| **Live Launch** | Любая | Есть | PLF стримы + Тимочко сториз |
| **Evergreen** | Автоворонка | Проверенный | PLF: автоматизация |

## Ключевые формулы

```
КОНТЕКСТ (месяцы) + ПРОГРЕВ (недели) + СОБЫТИЕ (дни) + ДЕДЛАЙН (часы) = ДЕНЬГИ

70% ЛИЧНОСТЬ + 30% ЭКСПЕРТНОСТЬ = ДОВЕРИЕ

ЭМОЦИИ → ДЕЙСТВИЯ  |  РАЦИО → ВЫВОДЫ

СЛОВА < КРАСИВЫЕ СЛОВА < МНЕНИЯ < ЦИФРЫ < ДЕМОНСТРАЦИИ < ПРУФЫ < ОПЫТ
```
