# Метод скользкой горки — Практические формулы

По книге Халилова «Метод скользкой горки» (2022). Выжимка рабочих инструментов.

---

## Суть метода

> Задача первого предложения — заставить прочитать второе.

Каждое предложение передаёт «эстафету любопытства» следующему. Читатель катится как с горки — не может остановиться.

**Анатомия:**
```
ТОЧКА ВХОДА (заголовок / первое предложение)
  ↓ зацепил внимание
ПЕРВОЕ СЦЕПЛЕНИЕ (интрига / обещание / факт)
  ↓ поставил на горку
СКОЛЬЖЕНИЕ (семена любопытства между блоками)
  ↓ не даёт бросить
УСКОРЕНИЕ (триггеры, твисты, клиффхэнгеры)
  ↓ разгоняет
ФИНАЛ (резюме + CTA)
```

## Формулы заголовков

### 4 требования к заголовку:
| Требование | Что даёт |
|-----------|----------|
| **Полезный** | Польза сразу или обещание |
| **Интригующий** | Недосказанность → стимул читать |
| **Конкретный** | Цифры, факты, понимание темы |
| **Подталкивающий** | Ведёт к чтению текста |

> Хороший заголовок = минимум 3 из 4

### Эталон (все 4):
> «Дамир, узнайте, как пользователи заработали 5 389 022 рубля на карантине»
- ✅ Полезно (возможность заработать)
- ✅ Интрига (как именно?)
- ✅ Конкретика (точная цифра)
- ✅ Подталкивание («узнайте»)

## Склейки между блоками

### Семена любопытства (техника Шугермана)
Короткие фразы-мостики в конце абзаца → тянут к следующему:

```
«Но это ещё не всё»
«Идём дальше»
«Есть и более эффективная альтернатива»
«Поясню»
«Например»
«Однако не всё так просто»
«Более того…»
«Причин много»
«И вот ещё один удар»
«Что даст нововведение?»
```

> Ставить в блоках, где внимание может ослабевать (перечисление УТП, список функций)

### Клиффхэнгеры
Обрыв на самом напряжённом месте. «Продолжение следует» = «пользователь не уходит».

**Форматы:**
| Приём | Как работает |
|-------|-------------|
| Связующие слова | «Но самое интересное было дальше...» |
| Продолжение в карусели | Обрываем мысль → «Продолжение в галерее» |
| Вопрос-ответ | Вопрос в одной Stories → ответ в следующей |
| Обещание | Завершаем блок обещанием пользы в следующем |
| «Как бы вы поступили?» | В кульминации → вопрос к аудитории |
| Сериальная модель | Серия постов, каждый кончается на пике |

### Пример из Акунина:
Две линии повествования. Каждая глава доводится до кульминации → обрывается → начинается другая линия с неспешного повествования. Читатель торопится дочитать, чтобы вернуться к предыдущей → проваливается в новую → и так до конца книги.

## Твисты

> Резкий поворот, заставляющий пересмотреть всю историю.

### 2 принципа:
1. **Подводка** — мелкие подсказки по ходу текста. В финале: «Как я не догадался!»
2. **Мощный разворот** — от первоначальной картины не остаётся ничего

### Шаблоны твистов:
- Злодей оказался героем
- Клиент с негативным отзывом стал сверхлояльным
- Случайный попутчик — инвестор
- Устаревший формат оказался будущим
- «Официант» оказался владельцем ресторана

### Пример: фермерский рынок
```
1. СТЕРЕОТИП (детально): толчея, шум, обвес, грязь
2. НИЖНЯЯ ТОЧКА: «устаревшая форма торговли»
3. РАЗВОРОТ 180°: «Давайте вместе пройдём по рынку Х»
4. КОНТРАСТ: охранник, навигация, рыбные ряды как в Ницце, повара при вас, крафтовое пиво, концерты
```
> Чем ниже упал — тем мощнее взлёт

## Структуры текста

### «Перевёрнутая пирамида» (от общего к частному) ★
Идеально для скользкой горки. Заходим с козырей.
```
ЗАГОЛОВОК: ключевая тема
ТЕЗИС: сжато главная идея
ДЕТАЛИ: разбор по пунктам
```

### «От частного к общему»
Начинаем с истории/примера → выводим к главной мысли. Формат TED.
> ⚠️ Сложнее держать внимание. Работает на интриге «к чему это?»

### FAQ
Вопрос-ответ. Удобная навигация, но слабая горка — читают выборочно.

### AIDA (только для продающих)
```
A — Attention: привлечь внимание
I — Interest: привязать к потребностям
D — Desire: усилить интерес, привязать к продукту
A — Action: призыв к действию
```

## Психологические триггеры (краткая выжимка)

| Триггер | Как использовать в тексте |
|---------|--------------------------|
| **Последовательность** | «Вы предприниматель? Тогда вы знаете, что такое ответственность» — согласился → согласится с выводами |
| **Соц. доказательство** | Отзывы, кейсы, цифры, лайки. «Мне часто задают вопрос» |
| **Сопричастность** | «Станьте частью сообщества из 30 000 человек» |
| **Схожесть** | Разные отзывы разным сегментам ЦА. Автор «свой» |
| **Наименьшее напряжение** | «Потребуется только ноутбук и 2 часа». Упрощайте вход |
| **Разрыв шаблона** | Факт, опровергающий стереотип → мозг включается |
| **Дефицит** | «Остался 1 номер», «пост удалю через 6 часов» |
| **Взаимный обмен** | Дал пользу → читатель склонен ответить действием |
| **Ментальное соучастие** | Мини-тест, вопрос → «инвестировал усилия» → ценность текста растёт |
| **Авторитет** | Регалии, статусные клиенты, СМИ, экспертные отзывы |

## Болезни текста (анти-паттерны)

### Смертельные:
| Болезнь | Пример | Лечение |
|---------|--------|---------|
| **Канцелярит** | «В рамках реализации противоэпидемических мероприятий...» | Тёплый человечный язык |
| **Клише** | «Уютный уголок Италии», «индивидуальный подход» | Конкретика и свежие формулировки |
| **Давление** | «КУПИТЕ! АКЦИЯ! ЗВОНИТЕ!» | Язык задач клиента |
| **Текст ни о чём** | «Грузоперевозки играют значительную роль в современном мире» | Конкретные факты и польза |
| **Нет позиции** | «С одной стороны... с другой стороны...» | «Будь за белых или за красных» |

### Структурные:
| Болезнь | Пример | Лечение |
|---------|--------|---------|
| **Нет цели** | «Всем отличной пятницы!» | Ответить «зачем?» перед написанием |
| **Нет структуры** | Бессвязные мысли | Скелет поста ДО написания |
| **Нет введения** | Сразу в список без контекста | Первый абзац = зачем читать |
| **Нет завершения** | Обрыв на полуслове | Резюме / вывод / CTA |
| **Цена в начале** | Цена до описания ценности | Сначала выгоды → потом цена |
| **Кликбейт** | Заголовок ≠ содержание | Всегда выполнять обещание заголовка |

### Стилистические:
| Болезнь | Как распознать |
|---------|----------------|
| **Длинные предложения** | Если не можешь проговорить на одном дыхании — разбей |
| **Пассивный залог** | «Было проведено мероприятие» → «Мы провели мероприятие» |
| **Абстракции** | «Высококлассный специалист» → «732 процесса за 21 год» |
| **Полутона** | «Неоднозначное впечатление» → «Я в восторге!» или «Провал» |
| **Писательская неуверенность** | «Не умею писать, но...» — не пиши тогда |

## Быстрые формулы (копируй и адаптируй)

### Формула продающего поста:
```
[Проблема читателя — узнаваемая ситуация]
[Нагнетание — что будет если не решить]
[Решение — ваш продукт/услуга]  
[Факты — цифры, отзывы, кейсы]
[Выгода — что получит клиент]
[CTA — один простой шаг]
```

### Формула экспертного поста:
```
[Заголовок с количеством пунктов]
[Вводная: обещание пользы + «сохраните пост»]
[Список пунктов: первые 3 — самые сильные]
[Завершение на высокой ноте]
[CTA: вопрос к аудитории]
```

### Формула поста-истории:
```
[Зацепка — удивительный факт или эмоция]
[Контекст — как я оказался в ситуации]
[Проблема — что пошло не так]
[Кульминация — пиковый момент]
[Твист или развязка]
[Вывод → привязка к теме блога]
[CTA]
```

### Формула развенчания мифа:
```
[Стереотип — максимально узнаваемо]
[Нижняя точка — «всё плохо»]
[РАЗВОРОТ — «но на самом деле...»]
[Контраст — как есть на самом деле]
[Факты и доказательства]
[CTA]
```
