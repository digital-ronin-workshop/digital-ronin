# Паттерны удержания внимания

Механики захвата и удержания внимания. Для видео (первые 3 сек) и текста (первые 2 строки).

---

## 1. Открытые петли (Open Loops)

Главный приём удержания: начать мысль — не закончить. Мозг не может бросить незавершённое.

### Типы открытых петель

| Тип | Механика | Пример |
|-----|----------|--------|
| **Отложенный ответ** | Задать вопрос → ответ в конце | «Есть одна фраза, которая удваивает конверсию. Но сначала...» |
| **Намёк на секрет** | Упомянуть → не раскрыть | «Я нашёл способ, о котором не говорят. Расскажу через минуту» |
| **Нумерация с паузой** | Начать список → прервать | «3 ошибки: первая — очевидная, вторая — неожиданная, а третья...» |
| **Незаконченная история** | Начать сюжет → оборвать | «Когда клиент написал мне это сообщение, я понял одну вещь...» |
| **Контринтуитивное** | Сказать что-то странное → не объяснить сразу | «Лучшее, что я сделал для продаж — перестал продавать» |

### Правило открытых петель
```
Открыть петлю → Дать ценность → Закрыть петлю → Открыть новую
Никогда не закрывай все петли сразу — держи минимум одну открытой
```

## 2. Разрыв шаблона (Pattern Interrupt)

Мозг игнорирует привычное. Чтобы зацепить — сломай ожидание.

### Приёмы разрыва

| Приём | Что ломает | Пример |
|-------|-----------|--------|
| **Противоречие** | Ожидание логики | «Хотите больше клиентов? Перестаньте их искать» |
| **Абсурд** | Ожидание серьёзности | «Мой кот понимает маркетинг лучше 90% экспертов» |
| **Конфликт** | Ожидание согласия | «Все говорят "делай рилсы". Я говорю — не делай» |
| **Неожиданный факт** | Ожидание банальности | «Вы тратите 6 часов в неделю на контент, который никто не видит» |
| **Смена масштаба** | Ожидание пропорций | «Одно слово в заголовке увеличило продажи на 300%» |
| **Чужой контекст** | Ожидание темы | «Что Netflix знает о продажах, а вы — нет» |

### Правило разрыва
```
Первое предложение = разрыв (цепляет)
Второе предложение = мост (связывает с темой)
Без моста — разрыв выглядит кликбейтом
```

## 3. Конкретика vs Абстракция

Мозг цепляется за конкретное и пролистывает абстрактное.

### Таблица усиления

| ❌ Абстрактно | ✅ Конкретно | Почему работает |
|---------------|-------------|----------------|
| Много денег | 347 000₽ за месяц | Число = доверие |
| Быстро | За 14 минут | Время = измеримо |
| Недавно | Вчера в 23:17 | Детали = реальность |
| Большой опыт | 11 лет и 2 340 клиентов | Факт = экспертиза |
| Помогу с контентом | Напишу 30 постов за 3 дня | Объём = понятно |
| Хороший результат | С 200 до 15 000 подписчиков | До/после = доказательство |

### Правило конкретики
```
Каждое утверждение → можно ли добавить число?
Каждый результат → можно ли добавить срок?
Каждый факт → можно ли добавить деталь?
Если да — добавляй. Конкретика побеждает всегда.
```

## 4. Первые 3 секунды (видео)

80% зрителей уходят в первые 3 секунды. Это единственный шанс.

### Что работает

| Приём | Тайминг | Пример |
|-------|---------|--------|
| **Провокационное утверждение** | 0-1 сек | «Забудьте всё, что вы знали о рилсах» |
| **Шокирующая цифра** | 0-1 сек | «97% рилсов набирают меньше 500 просмотров» |
| **Прямое обращение** | 0-2 сек | «Если вы эксперт и у вас мало продаж — смотрите» |
| **Действие/движение** | 0-1 сек | Резкий жест, смена кадра, объект в камеру |
| **Незаконченная мысль** | 0-3 сек | «Я потерял 500К на рекламе, потому что не знал одну вещь...» |
| **Визуальный контраст** | 0-1 сек | Текст на экране крупно + яркий фон |

### Что НЕ работает

| ❌ Убийца внимания | Почему |
|-------------------|--------|
| Приветствие («Привет, ребят!») | Не несёт ценности, зритель уходит |
| Длинная подводка | Ответ кажется далёким |
| Тихое начало | Нет энергии для остановки скролла |
| Повторение заголовка | Уже видели, зачем смотреть |
| Просьба подписаться в начале | Нет доверия, рано просить |

### Формула открывашки видео
```
[СТОП-слово/действие] + [Обещание/боль за 1 фразу] + [Открытая петля]

Пример: «Стоп.» + «Я нашёл формулу рилсов, которая работает в любой нише.» + «И третий пункт удивит вас больше всего.»
```

## 5. Первые 2 строки (текст: Threads, Telegram, посты)

В ленте видны только 1-2 строки. Это ваш заголовок.

### Что работает

| Приём | Пример |
|-------|--------|
| **Числа + боль** | «5 ошибок, из-за которых ваш Telegram не растёт» |
| **Провокация** | «Контент-планы не работают. Вот что работает» |
| **Личный опыт** | «Я потратил 2 года впустую. Вот что я понял» |
| **Вопрос-триггер** | «Почему ваш контент видят только мама и бот?» |
| **Неожиданное сравнение** | «Маркетинг как свидание: если начинать с продажи — убежит» |
| **Конфликт мнений** | «Все говорят "постить каждый день". Это худший совет» |

### Что НЕ работает

| ❌ Убийца | Почему |
|----------|--------|
| «Доброе утро, друзья!» | Пустые калории |
| «Хочу поделиться мыслями» | Нет крючка |
| Длинное предложение 30+ слов | Не считывается |
| Абстрактное обещание | «Полезный пост» — ни о чём |
| Начало с хештегов | Шум, не информация |

### Формула первых 2 строк
```
Строка 1: [Хук — зацепить] — боль/число/провокация
Строка 2: [Мост — удержать] — обещание ценности или открытая петля

Пример:
Строка 1: «90% экспертов совершают эту ошибку в рилсах.»
Строка 2: «Третий пункт — причина, по которой я сам терял охваты.»
```

## 6. Триггеры внимания (универсальные)

| Триггер | Сила | Как использовать |
|---------|------|-----------------|
| **Страх потери** | ⭐⭐⭐⭐⭐ | «Вы теряете...» / «Пока вы не делаете X...» |
| **Любопытство** | ⭐⭐⭐⭐⭐ | Открытые петли, незаконченные мысли |
| **Социальное доказательство** | ⭐⭐⭐⭐ | «90% людей...» / «Все делают...» |
| **Актуальность** | ⭐⭐⭐⭐ | «В 2025...» / «После обновления...» |
| **Идентификация** | ⭐⭐⭐⭐ | «Если вы [кто]...» / «Знакомо?» |
| **Авторитет** | ⭐⭐⭐ | «Harvard доказал...» / «Эксперты говорят...» |
| **Новизна** | ⭐⭐⭐ | «Новый способ...» / «Впервые...» |
| **Противоречие** | ⭐⭐⭐⭐⭐ | «Все думают X, но на самом деле Y» |

## 7. Каскад удержания (полная структура)

```
Секунда 0-1: СТОП → Визуал/звук/текст останавливает скролл
Секунда 1-3: КРЮЧОК → Обещание + открытая петля
Секунда 3-7: МОСТ → Почему это важно для ТЕБЯ (идентификация)
Секунда 7-15: ЦЕННОСТЬ → Первый пункт/инсайт (закрыть маленькую петлю)
Секунда 15-30: УДЕРЖАНИЕ → Новая петля + нарастание
Секунда 30+: РАЗВЯЗКА → Главный инсайт + CTA

Для текста:
Строка 1: СТОП + КРЮЧОК
Строка 2: МОСТ
Строка 3-5: ЦЕННОСТЬ (первая порция)
Далее: Чередование ценности и открытых петель
Конец: РАЗВЯЗКА + CTA
```
