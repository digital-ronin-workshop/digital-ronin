# Headlines — Генератор заголовков и хуков

Скилл для создания заголовков, хуков и открывашек на основе проверенных формул.

## Core Principle

```
Заголовок = 80% успеха контента
Без сильного заголовка — контент не откроют
1 заголовок = 1 обещание (не больше)
```

## Workflow: Генерация заголовков

```
1. Определи платформу → адаптируй длину и стиль
2. Определи цель → клик / дискуссия / сохранение / шер
3. Выбери 2-3 категории формул
4. Сгенерируй 5-10 вариантов
5. Проверь через чеклист
6. Отбери ТОП-3
```

## Адаптация под платформу

| Платформа | Длина | Особенности |
|-----------|-------|-------------|
| **Threads** | 1-2 строки (до 100 символов) | Провокация, вопрос, незавершённость |
| **Telegram** | 1 строка (до 80 символов) | Конкретика, цифры, польза |
| **YouTube (title)** | 50-60 символов | Кликбейт + честность, caps на 1-2 слова |
| **YouTube (hook)** | 3 секунды | Обещание + интрига, не раскрывать |
| **Instagram Reels** | Текст на экране, 5-7 слов | Крупный шрифт, 1 мысль |
| **Email** | 30-50 символов | Персонализация, urgency |

## Чеклист качества заголовка

- [ ] Есть конкретика (число, факт, результат)?
- [ ] Обещает выгоду или вызывает эмоцию?
- [ ] Понятен за 2 секунды?
- [ ] НЕ раскрывает ответ?
- [ ] Целевая аудитория узнает себя?
- [ ] Вызывает желание кликнуть/читать дальше?
- [ ] Адаптирован под платформу?

## Усилители заголовков

| Приём | Пример |
|-------|--------|
| **Число в начале** | "7 способов..." вместо "Способы..." |
| **Скобки/уточнение** | "...за 30 дней (без бюджета)" |
| **Год/актуальность** | "...в 2025 году" |
| **Негативный угол** | "перестаньте" / "хватит" / "никогда" |
| **Неожиданность** | "...и это не то, что вы думаете" |
| **Конкретный результат** | "с 0 до 10К подписчиков" |
| **Социальное доказательство** | "почему 90% делают неправильно" |

## Critics System

После генерации заголовков — 3 критика.

### Click Expert
- ✅ Вызывает желание кликнуть
- ✅ Есть интрига / незавершённость
- ❌ Ответ уже в заголовке

### Clarity Expert
- ✅ Понятен за 2 секунды
- ✅ Одно обещание, не перегружен
- ❌ Слишком абстрактно

### Platform Expert
- ✅ Длина подходит под платформу
- ✅ Стиль соответствует аудитории
- ❌ Не адаптирован
