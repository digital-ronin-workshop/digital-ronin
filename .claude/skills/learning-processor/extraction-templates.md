# Extraction Templates

Шаблоны форматирования для записи извлечённого контента.

## Для Skills (reference-файлы)

### TECHNIQUE — Техника/Приём

```markdown
### [Название техники]

> [Краткое описание в 1 предложение]

**Когда использовать:** [контекст]

**Процесс:**
1. [Шаг 1]
2. [Шаг 2]
3. [Шаг 3]

**Пример:**
[Конкретный пример применения]

---
*Источник: [название материала]*
```

### FORMULA — Формула/Фреймворк

```markdown
### [АББРЕВИАТУРА] — [Расшифровка]

**Формула:**
```
[A] → [B] → [C] → [D]
```

| Элемент | Описание | Пример |
|---------|----------|--------|
| A | [что это] | [пример] |
| B | [что это] | [пример] |

**Применение:** [когда использовать]

---
*Источник: [название материала]*
```

## Для Projects (learning-файлы)

### PATTERN — Что работает

```markdown
## [Дата] [Источник]

### [Название паттерна]

**Контекст:** [где/когда применялось]

**Суть:** [что именно сделали]

**Результат:** [что получилось, метрики если есть]

**Почему сработало:**
- [Причина 1]
- [Причина 2]

**Как применить:**
- [ ] [Конкретное действие для репликации]
```

### WARNING — Что НЕ работает

```markdown
## [Дата] [Источник]

### [Название антипаттерна]

**Контекст:** [где/когда пробовали]

**Что делали:** [описание ошибки]

**Результат:** [что пошло не так]

**Почему не сработало:**
- [Причина 1]
- [Причина 2]

**Как избежать:**
- [ ] [Что делать вместо этого]
```

### MEANING — Продающий смысл

```markdown
## [Дата] [Источник]

### "[Формулировка смысла]"

**Триггер:** [какой психологический триггер использован]

**Целевая аудитория:** [для кого резонирует]

**Контекст использования:** [где применять]

**Сила (1-10):** [оценка]

**Варианты формулировок:**
- [Вариант 1]
- [Вариант 2]
```

### QUOTE — Ценная цитата

```markdown
## [Дата] [Источник]

> "[Цитата]"
> — [Автор, контекст]

**Применение:** [как использовать в контенте]
```

## Формат отчёта

```markdown
# Отчёт обработки: [Название материала]

**Дата:** [YYYY-MM-DD]
**Источник:** [путь к файлу]
**Тип материала:** [транскрипт курса / вебинар / интервью / встреча]
**Определённая тема:** [youtube / threads / launch / ...]

## Правила форматирования

1. **Всегда указывать источник** — откуда извлечено
2. **Дата в формате** YYYY-MM-DD
3. **Заголовки с ###** для отдельных элементов
4. **Markdown-разделители** `---` между элементами
5. **Чеклисты** `- [ ]` для action items
