# Telegram Skills

> Универсальные методологии для создания Telegram-контента

---

## Что это

Telegram-скиллы — адаптация копирайтинг-методологий под специфику Telegram:
- Формат одного поста (не ветки)
- До 4096 символов
- **Жирное** форматирование работает
- Ссылки не режутся алгоритмами
- Аудитория лояльнее (подписчики vs лента)

---

## Когда загружать

**Auto-load при:**
- `max telegram [тема]`
- `тг пост [тема]`
- `tg post [topic]`

**Порядок загрузки:**
1. `telegram-copywriting.md` — структура
2. `telegram-hooks.md` — хуки
3. `projects/{project}/telegram/style.md` — стилистика проекта
4. `projects/{project}/telegram/learning/anti-patterns.md` — чего избегать
5. `projects/{project}/brand/voice-style.md` — голос

---

## Отличия от Threads

| Аспект | Threads | Telegram |
|--------|---------|----------|
| Формат | Ветки по 500 символов | Один пост до 4096 |
| Ссылки | В закреп-коммент | Можно в посте |
| Алгоритм | Охваты зависят от engagement | Доставка всем подписчикам |
| Хуки | Агрессивные, короткие | Можно спокойнее, **жирные** |
| Контент | Короткий, вирусный | Глубже, длиннее |

## Источники

Скиллы созданы на основе:
- Анализ 600+ постов Telegram-постов (адаптация Threads-методологии)
- Threads-методология (адаптированная)
- Learning loop из rejected/approved постов
