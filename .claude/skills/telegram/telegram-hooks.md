## Telegram vs Threads Hooks

| Аспект | Threads | Telegram |
|--------|---------|----------|
| Видимость | Первая строка в ленте | Начало поста в превью |
| Длина | До 100-150 символов | Можно длиннее, жирный работает |
| Формат | Plain text | **Жирный** = scroll-stopper |
| Эмоции | Обязательны (конкуренция) | Можно спокойнее (подписчики = лояльная аудитория) |

## Core Hook Formulas (Telegram)

### 1. Жирный заголовок-утверждение

**Формат:**
```
**[Сильное утверждение с конкретикой].**
```

**Примеры из канала Max:**
- **На моей фабрике контента сотрудник №1 — ИИ-сценарист.**
- **Залетевший ролик — это не везение. Это статистика.**
- **Я не представляю как без этого.**
- **Реалистичные AI-фото — тренд 2026?**

**Когда использовать:** Инструменты, закулисье, философия

---

### 2. Хук с числами

**Формат:**
```
**[Число] + [контекст]**
или
[Число в начале предложения]
```

**Примеры:**
- **+151 подписчик в телеграм за 13 дней**
- **100 км за неделю** — и вот, что я понял
- **4 месяца назад** я сделал это решение
- **$4 за минуту** видео с аватаром

**Почему работает:** Конкретика = доверие. Числа — первое, за что цепляется глаз.

---

### 3. Хук-вопрос

**Формат:**
```
[Вопрос, на который хочется ответить]?
```

**Примеры:**
- Какие планы на завтра?
- Что думаете, готовы отдать свои видео на ведение клонам?
- Когда последний раз вы заполняли свою доску желаний?

**Правила:**
- НЕ агрессивный ("Почему у тебя до сих пор нет...")
- НЕ риторический (должен провоцировать ответ)
- Лучше личный, чем абстрактный

---

### 4. Результат + Эмоция

**Формат:**
```
**[Результат]!** [Короткий контекст]
```

**Примеры:**
- **Первая продажа с Telegram!** 3 недели работы над воронкой.
- **Обновление HeyGen — теперь $4/мин** вместо $7.
- **Ученик сделал первые 100k** — разбираю, что сработало.

---

### 5. Контрарианство

**Формат:**
```
**[Общепринятое мнение] — [почему это не так]**
```

**Примеры:**
- **"Количество убивает качество" — это миф.**
- **Не нужен большой бюджет на рекламу.** Нужна система.
- **ИИ не заменит креатив.** Он его усилит.

---

### 6. Интрига с обрывом

**Формат:**
```
[Начало истории]... [намёк на продолжение]
```

**Примеры:**
- Вчера получил сообщение от подписчика. Читаю — и понимаю...
- 3 недели назад принял решение, которое изменило всё.
- Сначала я думал, это просто инструмент. А потом...

## Hooks by Content Type

### Для инструментов

| Формула | Пример |
|---------|--------|
| **[Инструмент] — [что даёт]** | **SuperWhisper — печатаю голосом быстрее пальцев** |
| **[Проблема] → [Решение]** | **Час на пост? С этим — 10 минут** |
| **[Результат с инструментом]** | **Сценарий за 30 минут вместо 4 часов** |

### Для закулисья

| Формула | Пример |
|---------|--------|
| **На моей [X] работает [Y]** | **На моей фабрике контента сотрудник №1 — ИИ** |
| **[Процесс] — как это устроено** | **Как я собираю контент на неделю** |
| **[Время] назад я [действие]** | **Месяц назад перестроил всю систему** |

### Для личного

| Формула | Пример |
|---------|--------|
| **[Действие] + инсайт** | **Пробежал 100 км за неделю — вот что понял** |
| **[Ситуация] научила меня** | **Провал запуска научил меня главному** |
| **[Состояние]** | **Устал. Но продолжаю.** |

### Для философии

| Формула | Пример |
|---------|--------|
| **[Тезис]** | **Количество создаёт качество. Не наоборот.** |
| **[Контрарианство]** | **Все говорят про X — я вижу другое** |
| **[Вопрос + позиция]** | **Почему мы боимся выкладывать "сырое"?** |

### Для продукта

| Формула | Пример |
|---------|--------|
| **[Результат клиента]** | **Ученик сделал первые 100k — разбираю** |
| **[Анонс + ценность]** | **В субботу — эфир про воронки. Без воды.** |
| **[Новость + польза]** | **Обновил шаблон Notion — забирай** |

## Anti-patterns (Что НЕ работает)

| Плохой хук | Почему | Как исправить |
|------------|--------|---------------|
| **Кайф.** | Нет контекста, не цепляет | Добавить конкретику: "Кайф, когда план работает" |
| **Сегодня хочу поговорить о...** | Скучно, затянуто | Сразу в суть |
| **Всем привет!** | Пустой | Убрать, начать с хука |
| **Серьёзно.** | Паттерн нейросети | Переформулировать |
| **Вы знали, что...** | Избитый формат | Личный опыт вместо "вы" |
| **Инструмент X — это...** | Определение скучно | Результат использования |

## Telegram-специфичные фишки

### Жирный = scroll-stopper

В Telegram **жирный текст** выделяется сильнее, чем в других платформах. Используй для первой строки:

```
**Сильное утверждение.**

Дальше — развитие мысли.
```

### Можно длиннее

В Threads первые 2 строки критичны. В Telegram подписчики лояльнее — можно позволить себе 2-3 предложения в хуке:

```
**Залетевший ролик — это не везение.**

Это статистика. Когда выпускаешь 30 роликов — математика начинает работать.
```

### Меньше кликбейта

Telegram-аудитория = подписчики, которые выбрали тебя. Не нужно "кричать" как в Threads/Reels. Работает:
- Честность
- Конкретика
- Личный опыт

## Testing Framework

### Что отслеживать
1. Просмотры (охват)
2. Реакции (вовлечение)
3. Репосты (виральность)
4. Комментарии (глубина вовлечения)

### A/B Testing
Публикуй похожие темы с разными хуками в разные дни. Сравнивай метрики.

### База победителей
Веди список хуков, которые сработали лучше среднего:

## Quick Checklist

Перед публикацией проверь хук:

- [ ] Есть конкретика (числа, имена, детали)?
- [ ] Не начинается с "Сегодня хочу..." или "Привет"?
- [ ] Первая строка **жирная** (если подходит)?
- [ ] Можно понять тему за 2 секунды?
- [ ] Есть интрига для чтения дальше?
- [ ] Соответствует типу контента?
- [ ] Не звучит как нейросеть?

## Pro Tips

1. **50% времени на хук** — остальное напишется само
2. **Напиши 3-5 вариантов** — выбери лучший
3. **Жирный = власть** — используй для главной мысли
4. **Конкретика > абстракция** — "100 км" > "много бегал"
5. **Личный опыт** — "я сделал" > "можно сделать"
6. **Прочитай вслух** — если спотыкаешься, переписывай
