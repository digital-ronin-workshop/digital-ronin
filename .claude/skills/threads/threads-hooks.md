## Why Hooks Matter

- Первая строка видна в ленте - решает scroll-stop или scroll-past
- 1-2 секунды на захват внимания
- Определяет, будут ли читать дальше
- **Время удержания** - ключевая метрика алгоритмов (как YouTube)

## Core Triggers (Работают всегда)

| Триггер | Описание | Пример |
|---------|----------|--------|
| **Зависть** | Результат, которого хочет ЦА | "Как я набрала 11к подписчиков из Threads" |
| **Жадность** | Выгода, польза, экономия | "5 скрытых функций ChatGPT, о которых молчат 99%" |
| **Любопытство** | Интрига, незавершенность | "Все говорят про X, но я увидела другую закономерность..." |
| **Тайное знание** | Инсайдерская информация | "Почему 80% онлайн-курсов убивают мотивацию студентов" |
| **Результат** | Конкретные цифры | "327 подписчиков за 5 дней с нулевого аккаунта" |

## Hook Formulas (Проверенные)

### 1. Контрарианный хук
```
[Общепринятое мнение] - это неправда/миф/ошибка.
```
**Пример:** "Почему 80% онлайн-курсов убивают мотивацию студентов"

### 2. Исследование/Статистика
```
Исследование [авторитет] показало [шокирующий факт]...
```
**Пример:** "Исследование Гарварда уничтожило миф о деньгах и счастье"

### 3. Личная история с конфликтом
```
[Ситуация из прошлого] изменила все / научила меня...
```
**Пример:** "Клиентка отказалась работать со мной, оценив меня по кухне"

### 4. Цифры + Обещание
```
[X] способов/правил/ошибок, которые [желаемый результат]
```
**Пример:** "5 приемов, которые делают снимки в 3 раза более вовлекающими"

### 5. Провокационный вопрос
```
Почему [целевая аудитория] до сих пор [ошибка]?
```
**Пример:** "Почему 99% фотографов снимают портреты неправильно"

### 6. Эмоциональный взрыв
```
Боже/Ура/Шок! [Событие + эмоция]
```
**Пример:** "Божечки, моя клиентка забеременела после 5 лет попыток!"

### 7. Интрига с обрывом
```
[Начало истории]... и тогда все изменилось.
```
**Пример:** "Мой парень не разрешает мне брать сумочку, когда мы куда-то идем..."

### 8. Афоризм от эксперта
```
Как [профессия] официально заявляю: [смелое утверждение]
```
**Пример:** "Как психолог официально заявляю: вы не сломаны"

## Hooks by Content Type

### Для вопросов (раскачка аккаунта)
| Формула | Пример |
|---------|--------|
| [ЦА], расскажите, как вы [действие]? | "Психологи, расскажите, как вы пришли в профессию?" |
| Какие у вас [тема]? Делитесь! | "Какие у вас денежные ритуалы? Кофе с корицей?" |
| [Интересный факт]. А как у вас? | "Я узнала, что 90% людей... А вы?" |

**Важно:** Вопрос должен быть таким, чтобы человек ЗАХОТЕЛ ответить. Не агрессивный, не обвиняющий.

### Для "лайкушников" (раскачка охватов)
| Формула | Пример |
|---------|--------|
| Лайк, если хочешь [желание] | "Лайк, если хочешь много денег в этом месяце" |
| [Пожелание]. Ставь лайк! | "Желаю тебе сегодня встретить человека, который изменит твою жизнь" |
| Самый приятный звук - [X]. Лайк! | "Самый приятный звук - уведомление о поступлении денег на карту" |

### Для историй (вовлечение)
| Формула | Пример |
|---------|--------|
| [Время назад] я [ситуация]... | "Три недели назад мы приехали в Бразилию..." |
| Когда мне было [возраст], я [событие] | "Когда мне было 7, мама сказала фразу, которая..." |
| У меня была клиентка, которая [проблема] | "У меня была клиентка, которая 5 лет не могла..." |

### Для экспертного контента
| Формула | Пример |
|---------|--------|
| [X] вещей, которые [результат] | "7 техник, которые оживляют любой текст" |
| Как [действие] за [время] | "Как создать вирусный пост за 5 минут" |
| Почему [проблема] и как исправить | "Почему ваши тексты не вызывают эмоции" |

## Hooks for Different Niches

### Мягкие ниши (психологи, эзотерики, коучи)

**Астрология/Нумерология:**
- "Люди, родившиеся [числа], в [месяц] ждет..."
- "Эти знаки выйдут замуж за миллионеров"
- "Что принесет транзит Урана по Близнецам? Овен:"

**Психология:**
- "Как психолог официально заявляю..."
- "Ни лицо, ни фигура — глубина принятия себя"
- "Ты не сломанная, ты просто..."

**Метафорические карты:**
- "Выбери карту и узнай послание от Вселенной"
- "Шаманские карты показали мне..."

### Твердые ниши (маркетологи, SMM, продюсеры)

**Результаты:**
- "Боже, у меня первая продажа консультации с Threads!"
- "134 подписчика за день с одного поста"
- "Мы сделали 8 миллионов на запуске"

**Лайфхаки:**
- "5 скрытых функций, которые..."
- "Как я экономлю 20к на рекламе каждый день"

## Anti-patterns (What NOT to do)

| Плохой хук | Почему не работает | Как исправить |
|------------|-------------------|---------------|
| "Сегодня хочу поговорить о..." | Скучно, нет интриги | Начни с результата или проблемы |
| "Всем привет! Как дела?" | Пустой, не цепляет | Сразу в суть с триггером |
| Общий вопрос без контекста | Не хочется отвечать | Добавь примеры, сделай личным |
| Агрессивный вопрос | Отталкивает | "Почему у тебя нет денег?" → "Какие денежные привычки работают у вас?" |
| Ссылка в первой строке | Режется алгоритмами | Ссылка только в комментарий |
| Кликбейт без payoff | Теряется доверие | Хук должен соответствовать содержанию |

## Testing & Iteration

### A/B Testing Framework
1. Напиши 3-5 вариантов хука на одну тему
2. Опубликуй с разницей в несколько дней
3. Сравни охваты и вовлечение
4. Зафиксируй победителя

## Quick Checklist

Перед публикацией проверь хук:

- [ ] Цепляет за 1-2 секунды?
- [ ] Есть триггер (зависть/жадность/любопытство)?
- [ ] Создает интригу "что дальше"?
- [ ] Релевантен целевой аудитории?
- [ ] Нет ссылок в первой строке?
- [ ] Соответствует содержанию поста?
- [ ] Не агрессивный/обвиняющий?

## Pro Tips

1. **Представь, что рассказываешь историю другу в баре** - это создает правильный тон
2. **Листай ленту** - насмотренность формирует понимание, что залетает
3. **Сохраняй чужие хуки** - создай свою библиотеку референсов
4. **GPT плохо понимает юмор** - юмористические хуки пиши сам
5. **Повторяй работающие хуки** - но они выгорают через 1-2 недели
