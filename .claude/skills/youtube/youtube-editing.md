# YouTube Editing (Монтаж)

## Когда использовать
- Постановка ТЗ монтажёру
- Работа с командой
- Определение стиля монтажа
- Обратная связь по правкам

---

## Главный принцип

**Качественно собранный текст > визуал**

Вырезать из речи:
- Воду
- Душные моменты
- Заминки
- Неинтересные части

---

## Правила работы с монтажёром

### 1. Детальное ТЗ + референсы

**Созвон:**
- С монтажёром + постпродакшн менеджером
- Совместный просмотр референсов
- Покадровое обсуждение

**Конкретные указания:**
```
"Здесь нравятся титры — размер и динамика, НО НЕ цвет"
"Отсюда возьмём переходы, картинки — нет"
"Эта склейка-вспышка — нужна, она не клишейная"
```

### 2. Чеклист проекта

**Что включать:**
- Где искать картинки (конкретные сайты)
- Что НЕЛЬЗЯ использовать
- Личный архив клиента
- Папка с оформлением
- Специфика ниши

**Пример для ниши "Драгоценные камни":**
```
- Не брать "жопу камня"
- Никаких мыльных/заблюренных фото
- Ссылка на 2-часовой созвон: что красиво, что нет
```

### 3. Контроль → Доверие

**Этап притирки (начало):**
- Усиленный контроль
- Проверка всех пожеланий

**Этап стабилизации:**
- Отпустить контроль
- Дать свободу для креатива

**⚠️ Слишком жёсткий контроль:**
- Потеря вдохновения
- Выгорание
- "Зачем стараться, если только по ТЗ можно"

### 4. Важное vs Неважное

**Главная цель:**
- Рост канала
- Продажи
- НЕ идеальный монтаж

**Не важно:**
- Пылинка на кофте (общий план)
- Мелкие визуальные недочёты

**Важно:**
- Как человек рассказывает
- Насколько полезно и актуально
- Нет ли воды

**Анти-пример:**
```
Клиент-маркетолог:
- Снял видео в марте
- До сих пор не опубликованы
- Переснимал 10+ раз
- Результат: НОЛЬ
```

### 5. Стиль монтажа = ЦА

**Ниша: Строительство/Отопление**
```
ЦА: Простой мужик из регионов
Монтаж: ПРОСТОЙ, без лакшери
Почему: Дорогой стиль ОТПУГНЁТ
```

**Ниша: Драгоценные камни**
```
ЦА: Покупатели за 500K+ руб
Монтаж: ДОРОГОЙ, качественный
Почему: Колхоз = диссонанс с ценой
```

### 6. Система управления задачами

**❌ Не работает:**
- Устные договорённости
- Задачи в Telegram (теряются)

**✅ Работает:**
- Notion
- Всё фиксировать письменно
- Чёткие дедлайны

### 7. Детальный бриф с клиентом

**Вытаскивать:**
- Брендбук
- Несколько примеров монтажа
- Информация о продукте и ЦА
- Бюджет на монтаж

## Обучение сборке текста

### Показать монтажёру аналитику
```
"Вот график — opening 5 минут, нас смотрят так"
"Здесь отвал зрителей"
```

### Пометки на съёмках
```
"Здесь задушнил — обрезать"
"Здесь воды налил"
"Вот это вырезать"
```

---

## Что бесит дизайнера/монтажёра

| Проблема | Решение |
|----------|---------|
| Нет указания "по референсу/не по референсу" | Прописывать в каждой задаче |
| Нет обратной связи день в день | Договориться о формате |
| Неясность по количеству концептов | Держать в курсе |
| "Не нравится, переделай" | Разложить до атомов |
| Несвоевременные исходники | Отправлять сразу после съёмки |

---

## Anti-patterns монтажа

### Стоковые вставки
**❌ Плохо:**
- Человек радуется → показываем радостного
- Кринжовые клише

**✅ Хорошо:**
- С плёночным эффектом
- Со стильной рамочкой
- С красивыми шрифтами

---

## Обратная связь

### ❌ Плохая
```
"Не нравится"
"Переделай"
"Некрасиво"
```

### ✅ Хорошая
```
"Спикер лопоухий — поменять на [ссылка]"
"Убил интригу — показал все 5 пар кроссовок, убери три"
"Фото некрасивое из-за большой челюсти"
```

---

## Делиться с командой

**Обязательно:**
- Скриншоты с высоким CTR
- Скриншоты с низким CTR
- Результаты A/B-тестов

**Эффект:**
- Монтажёр "обучается как ChatGPT"
- Меньше времени на правки
- Быстрее делает то, что нужно
