# YouTube Hooks & Openers — Полный гайд

## Когда использовать
- Написание первых 30 секунд видео
- Создание opening для шортсов/рилсов
- Работа с превью и заголовками
- Проверка opener по ВИСП

**Критическая метрика:** Все видео с 100K+ просмотров имеют удержание первых 30 секунд >60%

---

## Скользкая горка (Халилов/Шугерман)

**Главный принцип:** Задача первых 5 секунд — заставить смотреть следующие 5.

```
Opening → Каждый следующий блок
          ↓
"Эстафета любопытства" — зритель не может выключить
```

### Элементы горки в видео

| Элемент | Как работает | Пример |
|---------|--------------|--------|
| **Сцепление** | Первая фраза цепляет | "732 мужчины прошли этот тест" |
| **Интрига** | Открытая петля | "Но главное я скажу в конце..." |
| **Твист** | Неожиданный поворот | "Всё оказалось не так..." |
| **Конкретика** | Цифры, факты | За что цепляется мозг |

### Пружины горки

- **Упругость** — нет воды, каждая секунда на месте
- **Визуальность** — картинка в голове зрителя
- **Эмоция** — не оставляет равнодушным
- **Разрыв шаблона** — удивить = включить внимание

**Правило "пересказа":** Если opening можно пересказать другу за 10 секунд — он работает.

---

## ВИСП Framework

**ВИСП = 4 компонента, которые зритель должен получить с первых секунд:**

| Компонент | Значение | Вопрос для проверки | Маркеры в тексте |
|-----------|----------|---------------------|------------------|
| **В**ыгода | Конкретная польза после просмотра | Что получит зритель? | числа, сроки, "навсегда", "моментально", "легко" |
| **И**нтрига | Неопределённость, желание узнать | Есть ли вопрос в голове? | "это", "эти", "секрет", "мало кто знает", открытые вопросы |
| **С**рочность | Ощущение "надо СЕЙЧАС" | Почему не отложить? | "сейчас", "срочно", "пока не поздно", временные рамки |
| **П**ричастность | Это про меня, моя история | Для кого это? | "ты", "вы", демографика, узнаваемые ситуации |

### Формула TTS (George Blackman) для образовательного контента

| Элемент | Определение | Пример |
|---------|-------------|--------|
| **TARGET** | Кто ваша аудитория. Показать понимание их ситуации. | "Мужчины старше 50, просыпающиеся 3 раза за ночь..." |
| **TRANSFORMATION** | Какую трансформацию получат. Кристально ясное обещание. | "...сегодня покажу протокол для сна без перерывов." |
| **STAKES** | Что на кону. Цена бездействия. | "Каждый месяц промедления — воспаление хроническое." |

### Универсальный шаблон
```
[СРОЧНОСТЬ] + [ВЫГОДА] + [ДЛЯ КОГО] + [ИНТРИГА]

Пример: "Секрет для тех, кому надоело давить комедоны,
который спас сотни моих клиентов"
```

## 22 типа опенингов

### 1. Вопрос (Самострел)
> Спикер задаёт себе вопрос от лица аудитории

**Правило:** Попасть в лексику аудитории — как бы они сами задали этот вопрос.

```
"Почему же парень так долго не делает предложение,
если мы встречаемся уже три года?"

"Какой самый вредный продукт для мужчин после 40?"
```

**Супер-приём:** Задавать себе вопрос каждые 3-5 минут для возврата внимания.

---

### 2. Результаты / Обещание
> Конкретное обещание, что изменится после просмотра

```
"После этого видео ваш парень сделает вам предложение в ближайший год."

"Три способа, которые ускорят процесс — опробовано на 100 зрителях."
```

**Усилитель:** Добавить социальное доказательство (цифры, кейсы).

---

### 3. Статистика
> Начать с удивляющего факта или исследования

```
"По статистике, 70% мужчин не делают предложения в первые три года
отношений — и они не сделают его никогда."

"87% людей неправильно чистят зубы — врачи скрывают это годами."
```

**Важно:** Показать источник (плашка или документ) для доверия.

---

### 4. История из жизни
> Личная история с эмоциональными прилагательными

```
"Короче, была у меня подруга в Челябинске. 7 лет встречались
с парнем — душа в душу, общие друзья, хобби. А потом я узнаю,
что они женятся. Через две недели встречаемся — и она мне говорит:
'Мы сейчас разведёмся'..."
```

**Ключ:** Прилагательные передают эмоции ("душа в душу", "классно, здорово").

---

### 5. Тест / Самопроверка
> Зритель отвечает на вопросы и узнаёт себя

```
"Ответьте на три вопроса — если три 'да', ваш парень никогда
не сделает предложение:
1. Занимаетесь сексом реже двух раз в неделю?
2. У вас больше дружеские отношения?
3. Вы живёте вместе больше года?"
```

**Результат:** Человек думает "Боже, это про меня!" и смотрит дальше.

---

### 6. Образ (Представьте)
> Погружение зрителя в визуальную картинку

```
"Представьте: вы просыпаетесь утром в загородном доме.
Открываете дверь — солнце заливает пространство.
К вам подбегает лабрадор. Дети играют во дворе.
Вы начинаете готовить завтрак для всей семьи..."
```

**Правило:** Добавлять мелкие детали ("лабрадор бежевого цвета", "мальчик постарше").

---

### 7. До-После
> Показать трансформацию визуально

```
"Вот девушка два года назад — лицо в прыщах, комедоны, угри.
А вот она сейчас — кожа как попка младенца."
```

**Применение:** Идеально для бьюти, фитнес, результаты обучения.

---

### 8. Фотографии / Артефакты / Кейсы
> Показать документальное подтверждение

```
"Смотрите, вот моя переписка с парнем до брака —
'Ок', 'Нормально', 'Поем'.
А вот после брака — сердечки, длинные сообщения."
```

---

### 9. Честность / Факапы
> Признаться в ошибке для создания доверия

```
"За мою практику есть один кейс, которым я вообще не горжусь.
Мне реально за него стыдно. Это канал Филипп Захаров Экоферма..."
```

**Эффект:** Не теряется доверие — наоборот, растёт.

---

### 10. Жиза (Образ знакомый каждому)
> Ситуация, которая была у КАЖДОГО

```
"За последний год все ваши подруги вышли замуж, а вы завели кота."

"А вас в детстве тоже родители мучили ингаляциями над картошкой?"

"А вы тоже в детский сад ходили с жёлтой коробочкой
от киндер-сюрприза с чесноком?"
```

**Тест:** Если >80% аудитории скажут "да" — это жиза.

---

### 11. Юмор
> Начать с шутки, которая ушла в народ

```
"Муж и жена — одна сатана и один муж."

"Так воспринимается брак мужчинами. Сегодня расскажу,
как донести парню, что брак — это не конец жизни."
```

**Важно:** Тестировать шутку на живой аудитории перед съёмкой.

---

### 12. Чужой залетевший Reels
> Показать успешный короткий ролик в начале видео

```
[Спикер смотрит в телефон, на экране вставляется чужой Reels]
"Недавно наткнулся на это... И задумался — почему это работает?"
```

**Логика:** Если Reels залетел — у него отличное удержание. Используем это.

---

### 13. Топ-список
> Объяснить логику построения списка

```
"Сегодня три фразы, которые ускорят предложение.
Расположил по сложности — от простой к мощной.
Самую сильную оставил на конец."
```

**Правило:** Объяснить принцип сортировки (сложность, эффективность, опасность).

---

### 14. Сравнение
> Сравнить два продукта/подхода

```
"Железо за 1000 рублей и железо за 3000.
Сегодня разберём — стоит ли переплачивать?"
```

---

### 15. Мифы и суеверия
> Начать с распространённого заблуждения

```
"Многие верят, что видеть невесту в платье до свадьбы — плохая примета.
Но гораздо хуже — 7 лет жить с парнем без предложения."
```

---

### 16. Никогда нельзя
> Категоричный запрет с экспрессией

```
"Никогда, блять, не покупай эти акции!"

"Никогда не говори по телефону шестизначный код из СМС."
```

**Важно:** Требует харизматичного спикера, нельзя говорить монотонно.

---

### 17. Присоска (к известному бренду)
> Привязка к приложению/бренду, который есть у каждого

```
"У вас на телефоне есть приложение Сбербанк?
Вы знали, что там можно настроить автокопилку?"

"Лукойл зарабатывает на чём? Сегодня расскажу..."
```

**Бренды для присоски:** Сбер, Тинькофф, Яндекс, Пятёрочка, Авито, Wildberries.

---

### 18. Загадка
> Вопрос с несколькими "неправильными" ответами

```
Вера: "От чего зависит женский оргазм?"
Коля: "От белья?"
Вера: "Нет."
Коля: "От запаха мужчины?"
Вера: "Нет."
Коля: "От выключенного света?"
Вера: "Нет! На самом деле — от того, насколько она чувствует себя желанной."
```

---

### 19. Провокация / Спорное мнение
> Противоречие общепринятому

```
"Насрать на визуальные хуки."

"Тинькофф круче, чем Сбербанк — и вот почему."
```

---

### 20. Волшебная таблетка
> "Одно действие → много результатов"

```
"Одно упражнение от ожирения, запоров и головной боли."

"Одна фраза, которая изменит ваши отношения навсегда."
```

---

### 21. Самый худший
> Негатив работает лучше позитива

```
"Самая худшая водка среди дорогих."

"Три района Москвы, где жить не советую."

"Три способа точно потерять все деньги."
```

---

### 22. Слова-триггеры (усилители)
> Слова, которые закрывают несколько компонентов ВИСП

| Слово | Что закрывает |
|-------|---------------|
| **"убийцы"** | В+И+С |
| **"как"** | В+И |
| **"это"** | И (создаёт интригу) |
| **"худший"** | работает лучше "лучший" |
| **"секрет"** | И+В |
| **"навсегда"** | В+С |

## Три вектора срочности

### 1. Актуальность
```
Слова: "сейчас", "новое", "в 2025", "пока не поздно"
Пример: "Срочно сделай это до конца года"
```

### 2. Уклонение от боли
```
Слова: "перестань", "хватит", "очнись", "верни"
Пример: "Перестань терять деньги на рекламе"
```

### 3. Ресурсы (САМЫЙ МОЩНЫЙ)
```
- Деньги: "дешево", "за копейки", "бесплатно"
- Время: "быстро", "за неделю", "моментально"
- Усилия: "легко", "просто", "без подготовки"
- Без специалиста: "дома", "своими руками"

Пример: "Как за две недели избавиться — стоит копейки"
```

---

## Три уровня причастности

| Уровень | Пример |
|---------|--------|
| 0 — Базовый | "Для вас", "Для новичков" |
| 1 — Образы | "покупаешь книги и не читаешь" |
| 2 — Специфика | "комедоны на носу" > "прыщи" > "проблемы с кожей" |

## Anti-patterns (Чего НЕ делать)

### Категорически запрещено:
- **Долгое представление:** "Меня зовут..., я основатель..., 7 лет занимаюсь..." — НИКОГДА
- **"Welcome to my channel"** — скучно
- **"In this video we're going to..."** — generic
- **Начинать с credentials** — никто не спрашивал

### Как представиться правильно:
1. **Плашкой** — текст поверх видео
2. **После hook** — сначала зацепить, потом представиться

```
[ХОРОШИЙ HOOK]
"...а меня зовут Николай Велижанин, я основатель студии
YouTube-продвижения. Вот, кстати, наши топ-клиенты."
```

## Чек-лист hook

### Обязательно:
- [ ] Первая строка валидирует title
- [ ] Есть curiosity gap (интрига)
- [ ] Stakes понятны (что на кону)
- [ ] Все 4 компонента ВИСП присутствуют
- [ ] Нет jargon/overexplaining
- [ ] Длина 50-100 слов (~30 секунд)

### Дополнительно:
- [ ] Самые сильные слова — в начало предложения
- [ ] Есть конкретика (числа, сроки)
- [ ] Нет представления в первые секунды
- [ ] Прошёл тест на 200 миллионов (попадает в широкую аудиторию)
