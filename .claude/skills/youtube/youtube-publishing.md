# YouTube Publishing (Публикация)

## Когда использовать
- Загрузка и публикация видео
- Настройка параметров видео
- Планирование публикаций
- Чеклист перед публикацией

---

## SEO-пакет для видео

### Структура (4 блока)
```
1. Название видео
2. Ссылки (для начала описания)
3. Описание с вшитыми тегами + таймкоды
4. Теги видео (отдельно, через запятую)
```

---

## Пошаговый процесс публикации

### Шаг 1: Загрузка
```
Творческая студия → "Создать" → "Добавить видео"
```

### Шаг 2: Основная информация

**Название:**
- Копировать из SEO-пакета

**Описание (порядок важен!):**
```
1. Ссылки (вверху)
2. Описание с хэштегами
3. Таймкоды (отдельной строкой)
```

**Превьюшка:**
- Загрузить готовую
- Или несколько для А/Б теста (требует 8 видео + 16 шортсов)

**Плейлисты:**
- Создать/выбрать для структуризации

### Шаг 3: Дополнительные свойства

| Параметр | Рекомендация |
|----------|--------------|
| Прямая реклама | Отметить если есть |
| Теги | Через запятую, новой строкой |
| Язык | Выбрать основной |
| Ремиксы | Разрешить ✅ |
| Лицензия | Стандартная YouTube |
| Категория | "Люди и блоги" (по умолчанию) |
| Комментарии | Включены, базовая модерация |

### Шаг 4: Конечные заставки (ОБЯЗАТЕЛЬНО!)

**Добавить:**
- "Самое подходящее видео и канал"
- "Самое новое видео"

**Настройка:**
- Длительность: максимальная
- Размер: растянуть по кадру
- Цель: увеличить вовлеченность

### Шаг 5: Подсказки

**Типы:**
- Видео/плейлист — отсылки на контент
- Канал — отсылки на другой канал
- Ссылки — требует полной модерации канала

**Текст:**
- Делать максимально продающим
- Пример: "Посмотреть другие видео по [тема]"

### Шаг 6: Проверка
```
Использовать "Доступ по ссылке" для проверки:
- ✅ Название
- ✅ Описание
- ✅ Таймкоды
- ✅ Конечные заставки
- ✅ Подсказки
- ✅ Плейлист
- ✅ Превьюшка
- ✅ Теги
```

### Шаг 7: Закрепленный комментарий
```
- Подготовить текст с призывом
- Опубликовать под видео
- Закрепить (требует 8 видео + 16 шортсов)
```

### Шаг 8: Публикация

**Варианты:**
| Тип | Описание |
|-----|----------|
| Открытый доступ | Сразу всем |
| Премьера | Трансляция с чатом |
| Запланированная | Дата + время + часовой пояс |

**⚠️ ВАЖНО:** Нажать "Сохранить"!

---

## Ограничения нового канала

### Требует 8 видео + 16 шортсов:
- А/Б тестирование превьюшек
- Закрепление комментариев

### Требует полной модерации:
- Ссылки в подсказках

### Работает сразу:
- ✅ Ссылки в комментариях (активны)

---

## Запланированная публикация

**Применение:**
- Отпуск/отсутствие
- Стабильный график
- Подготовка контента заранее

**Настройка:**
1. Загрузить и оформить видео
2. Выбрать "Запланированная публикация"
3. Указать дату, время, часовой пояс
4. Можно назначить премьеру
5. Сохранить

---

## Публикация Shorts

- Процесс идентичен обычному видео
- Нет превьюшки
- Те же настройки SEO
