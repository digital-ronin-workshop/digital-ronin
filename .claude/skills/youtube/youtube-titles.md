# YouTube Titles (Заголовки)

## Когда использовать
- Написание заголовков для видео
- Оптимизация названий под рекомендации
- SEO-оптимизация заголовков
- Проверка читабельности

---

## Главное правило

**Идея видео должна уместиться в первые 5-7 слов.**

Всё остальное обрезается:
- На главной странице
- В похожих видео (sidebar)
- На мобильных устройствах

---

## Структура заголовка

```
[5-7 слов: ИДЕЯ + ТРИГГЕР] + [остаток: SEO/ключевые слова]

Пример:
"Почему YouTube Shorts набирают 0 просмотров | Как набрать просмотры"
         └─────── видно ──────────┘               └─── обрезано (SEO) ───┘
```

---

## Формула связки Превью + Заголовок

```
Превью говорит: "НАЖМИ"
Заголовок говорит: "НА МЕНЯ"

Вместе = действие "нажми на меня"
```

**Ключевой объект на превью вызывает ассоциацию → Заголовок подтверждает и добавляет любопытство.**

---

## Заглавные буквы (CAPS)

| Использование | Когда |
|---------------|-------|
| Умеренно | Только для триггерных слов |
| Весь CAPS | Не рекомендуется (трудно читать) |
| Каждое Слово С Заглавной | Англоязычный стиль |

**Пример:**
```
Хорошо: "Это СЕКРЕТ успеха на YouTube"
Плохо: "ЭТО СЕКРЕТ УСПЕХА НА YOUTUBE"
```

## Примеры хороших заголовков

| Заголовок | Почему работает |
|-----------|-----------------|
| "Я посетил самые странные отели России" | Интрига, конкретика |
| "Что будет если не оплачивать Яндекс..." | Вопрос, страх |
| "Я продал каждую секунду в своём видео" | Уникальность, результат |
| "Ютуб в России разблокировали =)" | Актуальность, эмоция |

---

## Примеры плохих заголовков

| Заголовок | Проблема |
|-----------|----------|
| "Что с каналом?" | Никому не интересно |
| "Попросил ChatGPT придумать анекдоты про..." | Обрезано, нет сути |
| "Мультимиллионер говорит - деньги будут..." | Обрезано на важном |
| "#67 Советую уделять время вашей Fun Page семь с..." | Непонятно о чём |
| "Мне 18 лет. [Игры из моего детства]" | Кто ты? Никому не интересно |

## Проверка заголовков

### Инструменты:

| Сервис | Назначение |
|--------|------------|
| **VidIQ плагин** | Показывает как заголовок выглядит в рекомендациях |
| **thumbnailchecker.com/ru** | Проверка длины на главной |
| **testmythumbnails.com** | Тест в реальных условиях |

### Чек-лист проверки:
1. Открой страницу канала — заголовок читается?
2. Открой VidIQ на видео — в sidebar всё видно?
3. Идея понятна из первых 5-7 слов?

## SEO-часть заголовка

**После 7 слов можно добавить ключевые слова для поиска:**

```
"Ремонт холодильника основные неисправности | Не работает холодильник что делать"
         └─── рекомендации ───┘                    └─────── поиск (SEO) ──────┘
```

**Зритель в поиске заинтересован → CTR в поиске всегда выше.**

---

## CTR по источникам

| Источник | Типичный CTR | Почему |
|----------|--------------|--------|
| Поиск | 15-30% | Зритель сам ищет |
| Главная страница | 8-12% | Алгоритм подбирает |
| Похожие видео | 1-5% | Зритель смотрит другое |

**Общий CTR** = среднее всех источников

**Минимальный CTR** для старта = 4-5%

---

## Anti-patterns

- Заголовок обрезается на важном месте
- Весь текст КАПСОМ
- Непонятно о чём видео из первых слов
- Заголовок не связан с превью
- Слишком узкая тема ("Мой влог ч.4")
- Вопросы типа "Что с каналом?"

---

## Чек-лист заголовка

- [ ] Идея в первых 5-7 словах
- [ ] Не обрезается в похожих видео
- [ ] Связан с превью (нажми + на меня)
- [ ] Триггерные слова выделены (умеренно)
- [ ] SEO-часть после основной идеи
- [ ] Проверен через VidIQ/thumbnailchecker
