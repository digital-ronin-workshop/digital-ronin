# YouTube CTA (Call to Action)

## Когда использовать
- Добавление CTA в скрипт
- Оптимизация конверсии
- Работа с end screens
- Интеграция продукта

---

## Типы CTA по сложности

| Тип | Где ставить | Сложность | Конверсия |
|-----|-------------|-----------|-----------|
| **Лайк** | После первого payoff | Лёгкая | Высокая |
| **Комментарий** | После решения проблемы | Средняя | Средняя |
| **Подписка** | В конце видео | Сложная | Низкая |
| **Ссылка/продукт** | После payoff | Сложная | Зависит от доверия |

**Правило:** Чем сложнее действие, тем позже в видео.

---

## CTA на лайк

```
Место: после первого payoff или причастного момента
Формула: Причастность + просьба
```

**Примеры:**
- "Ставьте лайк, если знакомо"
- "Если узнали себя — палец вверх"
- "Лайк, если было полезно"

---

## CTA на комментарий

```
Место: после решения проблемы
Формула: Социальная ответственность + конкретный вопрос
```

**Примеры:**
- "Если вы уже прошли этот этап, напишите свою историю. Это поможет другим."
- "Напишите в комментариях, какой пункт был самым полезным"
- "Поделитесь своим опытом — это важно для нашего сообщества"

**Почему работает:** Люди хотят помочь другим, не просто "оставить коммент".

---

## CTA на подписку

```
Место: в конце видео
Формула: Окружение + ценности
```

**Примеры:**
- "Правильное окружение — где люди схожи по ценностям. Подписывайтесь."
- "Если хотите больше такого контента — подписка и колокольчик"
- "Присоединяйтесь к тем, кто хочет лучшего будущего"

**Важно:** Не говори "лайк и подписка" в начале — это не работает.

---

## CTA на продукт/ссылку

```
Место: после payoff, через НОВУЮ проблему
Формула: Подсветить проблему + простое решение
```

**Пример:**
```
"Это легко сказать — говорите о своих чувствах.
Но как понять, что это не навязанные обществом чувства?
Есть список вопросов, чтобы отделить себя от социума.
Он есть у нас в ТГ-канале — ссылка в описании."
```

**Правило:** Продавать продукт можно только в видео **40+ минут** (достаточно доверия).

---

## Hook-Curiosity-Action Framework

Для end screen CTA:

1. **Hook** — захвати внимание перед концом
2. **Curiosity** — создай новый gap/проблему
3. **Action** — направь на действие

**Пример:**
```
Hook: "Вот как можно редактировать 3 видео в неделю..."
Curiosity: "Но быстрый оборот = больше клиентов. И это новая проблема..."
Action: "Поэтому я сделал это видео. Кликайте сюда."
```

## PCM типы личности

Разные люди реагируют на разные CTA:

| Тип | Мотивация | Как обращаться |
|-----|-----------|----------------|
| **Деятель** | Действие, результат | "Сделай", "Попробуй", "Начни" |
| **Бунтарь** | Уникальность | "Не будь как все", "Выделись" |
| **Логик** | Факты | "Исследования показали", "Статистика" |
| **Упорный** | Ценности | "Важно", "Правильно", "Должен" |
| **Мечтатель** | Визуализация | "Представь", "Вообрази" |
| **Душевный** | Эмоции | "Почувствуй", "Вместе", "Близкие" |

---

## Метрики

- Типичный end screen click rate = **3%**
- С хорошим CTA = **10-15%**
- Лайки зависят от эмоций в видео, не от слов CTA

---

## Размещение CTA в скрипте

```
HOOK
↓
PAYOFF 1
↓
[CTA на лайк] ← после причастного момента
↓
PAYOFF 2
↓
[CTA на продукт] ← через новую проблему
↓
PAYOFF 3
↓
[CTA на подписку] ← в конце
↓
[End screen CTA]
```

---

## Anti-patterns

**НЕ ДЕЛАТЬ:**
- "Лайк и подписка" в начале видео
- Питчить продукт в коротких видео (< 40 мин)
- Vague benefits: "кликни для большего"
- Несколько CTA подряд
- CTA без связи с контентом

---

## Чек-лист CTA

- [ ] CTA соответствует месту в видео
- [ ] Есть связь с контентом
- [ ] Hook-Curiosity-Action для end screen
- [ ] Продукт только в длинных видео
- [ ] Конкретное действие (не "поддержите")
