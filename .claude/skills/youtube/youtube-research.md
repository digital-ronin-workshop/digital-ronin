# YouTube Research & Validation

## Когда использовать
- Исследование перед написанием скрипта
- Валидация идей и вопросов
- Анализ конкурентов
- Поиск референсов

---

## 4 компонента исследования

1. **Audience Understanding** — что хотят, чего боятся, что ценят
2. **Competitor Analysis** — какие видео работают, какие gaps есть
3. **Content Gathering** — факты, примеры, insights
4. **Validation** — проверка accuracy

---

## Упражнение "Астрофизика"

**Метод валидации вопросов через YouTube:**

1. Напиши вопросы, которые хочешь задать в видео
2. Вбей каждый в поиск YouTube
3. Посмотри просмотры на похожих видео
4. Если есть референсы 500K+ — вопрос хороший
5. Если референсов нет — вопрос душный

### Правило хорошего вопроса

```
Хорошие вопросы касаются лично зрителя:

"Что будет со МНОЙ?" > "Как это работает?"
"Могу ли Я это использовать?" > "Интересный факт"

Техника: спроси себя "А меня это как ебёт?"
```

| Критерий | 1 | 2 | 3 |
|----------|---|---|---|
| **Insight** | Банально | Интересно | Уникально |
| **Hooks** | Слабый | Нормальный | Сильный |

| **Tonality** | Не попал | Частично | В точку |
| **Storytelling** | Запутано | Ясно | Захватывает |

## Поиск триггерных слов

1. Введи тему в YouTube
2. Смотри названия видео с большими просмотрами
3. Выписывай повторяющиеся слова/фразы
4. Используй эти слова в своём сценарии

**Пример для "поджелудочная":**
- "симптомы" — 2M просмотров
- "навсегда" — 1.6M просмотров
- "без лекарств" — 1.2M просмотров
→ Включи эти слова в свой скрипт

## Насмотренность — Методология VPH

### VPH (Views Per Hour) — Главная метрика

**Формула:** VPH = Просмотры / Время на YouTube

```
❌ Плохой референс: 1M просмотров за 10 лет
✅ Хороший референс: 20K просмотров за 2 недели
```

### 3 критерия оценки референса

**1. VPH (просмотры/время)**
- Видео с меньшими просмотрами за короткое время ЛУЧШЕ миллионника за годы

**2. Соотношение просмотров к подписчикам**
```
Идеал: Канал 5-10K подписчиков → Видео 100K+ просмотров
Вывод: Смотрели из-за ТЕМЫ, не из-за блогера
```

**3. Причина успеха**
- Почему это видео залетело?
- Какие слова в названии/превью?
- Что в первых 30 секундах?

### Методы насмотренности

**1. Google поиск** → Частые вопросы аудитории
**2. YouTube поиск** → Автодополнение = SEO-темы
**3. Похожие видео** → Прыжки по тегам
**4. Популярные на канале** → Сортировка "Популярные"

### "Магические слова" по нишам

| Ниша | Триггерные слова |
|------|------------------|
| Инвестиции | "доллар", "рухнет", "дефолт" |
| Кулинария | "пышный" |
| Рыболовство | "одурел от прикормки" |
| Недвижимость Крым | "офигел", "охренел" (КАПСОМ) |
| Недвижимость Москва | "недорогих", карта на превью |

### Визуальные правила ниш

| Ниша | Стиль |
|------|-------|
| Премиум-недвижимость | Лакшери, дорогая съёмка |
| Юриспруденция | Чем хуже качество — тем лучше |
| Бюджетное строительство | iPhone, трясущаяся камера |

### Workflow насмотренности

```
1. Поисковой запрос (Google/YouTube)
2. Найти 3-5 видео с высоким VPH
3. Проверить: Просмотры >> Подписчиков
4. Задать: "Почему залетело?"
5. Проанализировать первые 30 секунд
6. Выписать паттерны (слова, визуал)
7. Прыгнуть в "Похожие видео"
8. Зайти на канал → "Популярные"
9. Повторить для 5-10 каналов
10. Собрать контент-план
```

### Переснимание — это норма
```
Один референс переснимается 10+ раз
У каждого своя ЦА по демографии
Не бойся вдохновляться
```

### Время на насмотренность
- Лайт: 30 минут
- Полная: 2 часа
- Углублённая: 2-5 часов

## Поиск референсов

### Что искать:
- Видео с 1M+ просмотров в твоей нише
- Успешные openers (первые 30 сек)
- Структуры, которые работают
- Слова-триггеры

### Как анализировать:
1. Смотри первые 30 секунд
2. Разбери по ВИСП
3. Выпиши работающие фразы
4. Адаптируй под свою тему

---

## Валидация темы

Перед написанием скрипта проверь:

1. **Есть ли спрос?** — ищут ли эту тему
2. **Есть ли конкуренты?** — кто уже снял
3. **Можно ли сделать лучше?** — найди gaps
4. **Есть ли уникальный angle?** — чем отличишься

## Anti-patterns исследования

- Пропускать research и сразу писать
- Не проверять вопросы на спрос
- Копировать конкурентов 1-в-1
- Игнорировать триггерные слова
- Не иметь насмотренности в нише
