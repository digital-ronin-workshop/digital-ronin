# YouTube Thumbnails (Превью)

## Когда использовать
- Создание превью для видео
- Анализ кликабельности
- Оптимизация CTR
- Работа с дизайнером

---

## Главное правило

**Импульсный объект — основа превью.**

Импульсный объект = изображение, которое:
- Актуально для вашей аудитории
- Понятно за 0.5 секунды
- Цепляет как "заусенец" при скролле

---

## Идеальное превью

```
┌─────────────────────────────────┐
│  ТРИГГЕРНАЯ     ↘              │
│    ФРАЗА          ●            │
│                  ↗             │
│              [объект]          │
│                         21:00  │
└─────────────────────────────────┘

3 объекта:
1. Импульсный объект (центр)
2. Триггерная фраза
3. Связующий элемент (стрелка/лицо)
```

---

## Тепловая карта превью

```
┌─────────────────────────────────┐
│ ██████████ │ ▓▓▓▓▓▓ │ ░░░░░░░ │
│  ЗЕЛЁНАЯ   │ ЖЁЛТАЯ │ КРАСНАЯ │
│  (важно!)  │ (важно)│(не важно)│
└─────────────────────────────────┘
```

**Глаз движется:** слева направо, по часовой стрелке или "восьмёркой"

**Правило:** Всё важное — в зелёной и жёлтой зонах.

---

## Правило трёх

| Элемент | Максимум |
|---------|----------|
| Объекты | 3 |
| Цвета | 3 |
| Слов в тексте | 3-5 |

**Больше = рассеивается внимание.**

---

## Использование лица

**Лицо на превью только в 2 случаях:**

### 1. Яркая эмоция
```
Удивление, страх, радость, шок
— контекстно связанная с идеей превью
```

### 2. Взаимодействие с объектом
```
Держит объект, указывает на него,
отталкивает, смотрит на него
```

**НЕ НУЖНО:** Просто лицо без эмоции в зелёной зоне.

### Статистика лиц (исследования)

- **90%** лучших видео используют лица
- **+921,000** дополнительных просмотров в среднем
- **2.3x** CTR множитель (самый высокий среди элементов)

### Рейтинг эмоций по эффективности

| Эмоция | Использовать для | CTR эффект |
|--------|-----------------|------------|
| Шок/удивление | Разоблачения, мифы | **Максимальный** |
| Энтузиазм | Мотивация, результаты | **Очень высокий** |
| Любопытство | Образование, советы | **Высокий** |
| Серьёзность | Сложные темы | **Средний** |

## CTR Психология

### Главный принцип

**КОНТРАСТНОСТЬ > КРАСОТА**

Высокий контраст "уродливых" цветов бьёт красивые низкоконтрастные на **+30% CTR**.

```
Чёрное + лимонно-жёлтое (уродливо) > Розовое + светло-зелёное (красиво)
```

### Цветовые комбинации по нишам

| Ниша | Основной | Акцент | Эффект | CTR прирост |
|------|----------|--------|--------|-------------|
| Мужское здоровье | Красный | Жёлтый | Энергия, срочность | +25-35% |
| Сон/восстановление | Синий | Белый | Спокойствие, доверие | +15-20% |
| Питание/диета | Зелёный | Оранжевый | Здоровье, энергия | +20-25% |
| Научное объяснение | Фиолетовый | Белый | Авторитет | +10-15% |
| Развлечение | Красный | Жёлтый | Максимум энергии | +30-40% |

### Противоположные пары (максимальная видимость)

- Чёрный текст на жёлтом фоне
- Белый текст на красном фоне
- Оранжевый на синем фоне
- Лимонно-жёлтый на чёрном фоне

## Проверенные формулы

### MrBeast Formula (максимум CTR)

1. **Центральный герой** — огромное лицо в центре
2. **Насыщенные цвета** — красный, жёлтый, оранжевый (максимально яркие)
3. **Минимум деталей** — лицо + 2-3 слова
4. **Интрига без раскрытия** — психология любопытства

```
"ПОТЕРЯЛ $1,000,000" + шокированное лицо + красный/жёлтый контраст
```

### PewDiePie Formula (узнаваемость)

- **Шрифт Impact** — толстый белый с чёрным контуром
- **Хронические цвета** — красный, жёлтый, чёрный на КАЖДОЙ обложке
- **Экстремальные выражения** — узнаваемо за 0.5 сек
- **Результат:** обложка = бренд

### MKBHD Formula (статус)

- **Минимум текста** — часто 0 слов
- **Резкий фокус** на продукт
- **Ультравысокое качество** — профессиональное разрешение
- **Blend modes** — текст адаптируется к фону

### Medical Expert Formula (доверие)

- **Доктор справа** (40%), объект слева
- **Серьёзная но доступная эмоция**
- **Красный акцент** на ключевом тексте
- **Тёмный фон** с молекулярными формулами

## A/B Тестирование

### YouTube Native Test & Compare

1. YouTube Studio → Контент → "Тест и сравнение"
2. Загрузить 2-3 альтернативные обложки
3. Подождать 48-72 часа
4. Отслеживать: CTR, AVD, Watch time

### Интерпретация результатов

| Результат | Действие |
|-----------|----------|
| Ясный победитель (10%+ разница) | Установить основным, масштабировать паттерн |
| Без результата (2-5%) | Переиспытать с 2K+ показами |
| Конфликт (CTR vs AVD) | Приоритизировать CTR |

### Что тестировать

1. **Разные эмоции** (одна сцена)
2. **Варианты текста** (одна визуальная часть)
3. **Расположение объектов** (лево/право)
4. **Цветовые схемы** (тёплая/холодная)

## Размещение текста

| Позиция | Когда использовать |
|---------|-------------------|
| По центру сверху | Основной вариант |
| Левый верхний угол | Если объект в центре |
| Левый нижний угол | Если сверху занято |
| По центру снизу | Осторожно с таймингом! |

**Нижний правый угол** — перекрывается плашкой длительности.

### Длинный текст — по диагонали:
```
ЗНАЙ
 ЭТО
  КВАДРАТ
```

---

## Цветовые правила

**МОЖНО:**
- 2-3 цветовые схемы
- Контрастный текст на фоне
- Белый/жёлтый текст на тёмном фоне
- Чёрный текст на светлом фоне

**НЕЛЬЗЯ:**
- Красный текст на чёрном (бьётся на пикселях)
- Текст сливается с фоном
- Пёстрый фон с множеством цветов
- Радужные градиенты

---

## Фон

**Фон должен быть сепарирован** — подчёркивать объекты, а не забивать их.

| Хорошо | Плохо |
|--------|-------|
| Однотонный | Лес с листвой |
| Градиент | Много объектов |
| Размытый | Пёстрый |
| Минималистичный | Детализированный |

---

## Читабельность на разных размерах

Превью показывается:
- Большое (TV, главная на PC)
- Среднее (главная на mobile)
- Маленькое (похожие видео, sidebar)

**Правило:** Если на 14% масштабе не читается — переделывай.

---

## Стрелочки и обводки

**Используй только когда:**
- Указывают на деталь, вызывающую интригу
- Подчёркивают импульсный объект
- Создают связь между элементами

**НЕ НУЖНО:**
- Обводить лицо просто так
- Стрелка в никуда
- Обводка без смысла

## Anti-patterns

| Ошибка | Проблема |
|--------|----------|
| Нет импульсного объекта | Нечего цеплять |
| Лицо без эмоции в зелёной зоне | Бесполезное место |
| Много объектов (>3) | Рассеивается внимание |
| Много цветов (>3) | Пестрота |
| Текст сливается с фоном | Не читается |
| Нечитабельный шрифт | Декоративные fonts |
| Пёстрый фон | Забивает объекты |
| Текст перекрыт таймингом | Нижний правый угол |

---

## Примеры плохих превью

- "Летний влог ч.4" — нет триггерных объектов
- Декоративный шрифт — нечитабельно при уменьшении
- Много объектов + узкая тема — "Барахольщики"
- Фон забивает текст — лес, деревья

## Photoshop Tips

### Вырезание объекта:
1. Print Screen → Ctrl+V
2. Прямолинейное лассо
3. Инверсия → Delete

### Подчёркивание текста:
1. Параметры наложения → Тень
2. Режим: Нормальный
3. Цвет: белый (свечение) или чёрный (тень)
4. Размах + непрозрачность

### Улучшение лица:
1. Коррекция → Яркость/Контрастность
2. Цветовой баланс (тепло)
3. Насыщенность для ярких элементов (шапка)
4. Штамп для неровностей кожи

Каждый проект определяет свой стиль превью в:

### Агент генерации

Полная документация: [agents/production/thumbnail-generator.md](../../../../agents/production/thumbnail-generator.md)

**Триггеры:**
- "thumbnail for [topic]"
- "обложка для [topic]"
- "face swap thumbnail"

### Ключевые правила Nano Banana

1. **Описывай сцену, не список ключевых слов**
2. **Детализация = качество** (800-2000 символов минимум)
3. **Identity Lock** — явно указывай сохранение лица
4. **Aspect ratio** — всегда 16:9 (1920x1080)
5. **Промпты на английском** (даже для RU контента)
