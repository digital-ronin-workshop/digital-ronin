# YouTube AI Prompts & Checklists

## Когда использовать
- Генерация сценариев через AI
- Проверка AI-контента
- Работа с ChatGPT/Claude для скриптов

---

## Шаблон 1: Тезисный сценарий

```
"Ты будешь писать тезисные сценарии для видео на ютуб-канал.
В сценарии должна быть четкая структура:

Вступление — короткое, яркое, цепляющее внимание.
Обозначь тему, проблему или вопрос.

Основная часть — разделена на логичные блоки,
каждый тезис раскрывает важный аспект темы.
Включи примеры, факты, сравнения, короткие истории.
Всё понятно и доступно, без воды и сложных терминов.

Заключение — мотивация зрителя к действию.

Стиль: живой, разговорный, вовлекающий."
```

---

## Шаблон 2: Полный сценарий с параметрами

```
"Напиши полный сценарий без раскадровки для видео на тему [ТЕМА]:

Спикер: [описание — кто он и чем занимается]
Продукт: [что продаёт] — интегрируй CTA от проблемы зрителя
Целевая аудитория: [краткое описание ЦА]
Боль зрителей: [какую проблему решает видео]

Пиши просто, понятно, вовлекая в содержание.
Начало — триггерное и интригующее.
Завершение — мотивирующее."
```

"Ты будешь писать сценарии для видео на ютуб-канал о [ТЕМА].

## Шаблон 4: Shorts

```
"Напиши сценарий для шортс-видео — короткий и цепляющий текст.
Тема: [ТЕМА]

Простой и понятный язык.
Без приветствия и прощания, сразу к сути.
Динамичный, с ярким началом.

Если тема — ответ на вопрос:
Сохраняй интригу, ответ напиши в самом конце."
```

## Чек-лист AI-сценария

После генерации **обязательно** проверь:

### Факты и данные
- [ ] Факты перепроверены (AI выдумывает)
- [ ] Цифры и расчёты проверены
- [ ] Имена и названия точные
- [ ] Исследования и ссылки существуют

### Качество текста
- [ ] Нет повторов одной мысли разными словами
- [ ] Логика выстроена по порядку
- [ ] Нет сложных слов (если нужен простой стиль)
- [ ] Учтены региональные особенности

### Структура
- [ ] Есть ВИСП в открытии
- [ ] Payoffs структурированы
- [ ] Open loops закрыты
- [ ] CTA на местах

---

## Особенности работы с AI

### AI забывает контекст
Если работа растянулась — напоминай ключевые моменты:
- Кто спикер
- Какая ЦА
- Какой стиль

### Длина промпта важна
Слишком длинные промпты ухудшают качество.
Пиши чётко и по делу.

### AI не знает твою нишу
Давай контекст:
- Примеры успешных видео
- Специфику аудитории
- Табу и ограничения

## Итеративный процесс

```
1. Сгенерируй первый драфт
2. Проверь по чек-листу
3. Попроси AI исправить конкретные проблемы
4. Адаптируй под свой голос
5. Прочитай вслух
```

---

## Промпт для улучшения

```
"Улучши этот сценарий:

1. Добавь ВИСП в открытие (Выгода, Интрига, Срочность, Причастность)
2. Проверь logic flow — каждое предложение ведёт к следующему
3. Добавь конкретные примеры и цифры
4. Убери filler words и повторы
5. Добавь open loops между секциями

[Вставь сценарий]"
```

---

## Anti-patterns

- Принимать AI-текст без проверки
- Не давать контекст о спикере/ЦА
- Слишком длинные промпты
- Забывать про ВИСП и структуру
- Не проверять факты
