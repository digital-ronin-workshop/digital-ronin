# YouTube Channel Setup (Настройка канала)

## Когда использовать
- Создание и настройка канала
- Оформление брендпака
- Настройка безопасности
- Оптимизация профиля

---

## Создание Google аккаунта

### Правила регистрации
```
❌ НЕ использовать VPN при регистрации
✅ Регистрировать из России напрямую
   → Не потребуется верификация телефона
```

### Безопасность (ОБЯЗАТЕЛЬНО)
- 2FA включить
- Google Authenticator добавить
- Резервная почта
- Резервный телефон
- Ключ доступа (отпечаток)

---

## Создание YouTube канала

### После регистрации Google
```
1. Включить VPN (только для YouTube)
2. Создать канал
3. Подтвердить номер телефона
   → Разблокирует видео >8 мин и превьюшки
```

### Ограничение
- Макс 2 канала на 1 номер в год

### Расширенные функции (Россия)
- Включаются автоматически после ~8 видео + 16 шортсов
- Или верификация телефона

## URL канала — КРИТИЧЕСКИ ВАЖНО

### ⚠️ СОХРАНИТЬ URL КАНАЛА!

**Почему:**
- URL НИКОГДА не меняется
- Нужен для восстановления при взломе
- Доказательство владения

**Где сохранить:**
- Заметки телефона
- Google Docs
- Надёжное место

## Название канала

### Формула
```
[Имя/Бренд] + [Тематика]

Примеры:
"Николай Вележанин. Все о YouTube продвижении"
"[Имя] про недвижимость"
```

### Исключение — деликатные темы
```
❌ "Банкротство физлиц" — люди не хотят в подписках
✅ "Финансовая грамотность" — шире и нейтральнее
```

---

## Теги канала

### Три типа (500 символов макс, использовать ~250)

**1. Брендированные (2-4 шт)**
```
Имя, фамилия, название бренда
Дублировать в КАЖДОМ видео
```

**2. Тематические**
```
Вектор канала: "бизнес", "YouTube", "предпринимательство"
```

**3. Поисковые**
```
"Как раскрутить YouTube"
"Как масштабировать бизнес"
```

---

## Описание канала

### Структура
```
1. Призыв + кликабельная ссылка
2. О чём канал (кратко)
3. Кто вы (регалии, достижения)
4. Что будет на канале
5. Призыв подписаться
```

### ⚠️ НЕ делать сплошным текстом!
- Разбивать на блоки
- Использовать переносы

---

## Ссылки на канале

### Название ссылок
```
✅ Призыв: "Подписывайся в Телеграме"
✅ Или название: "Telegram"
```

### Логотипы
- Оригинальные ссылки (t.me/...) показывают логотип
- Сокращённые (mobz) — без логотипа

## Брендпак канала

### Компоненты
1. Шапка (баннер)
2. Аватарка
3. Превьюшки
4. Водяной знак

---

## Шапка канала

### Размеры
```
Полный размер: 2560×1440 px (или 2048×1152)
Вес: <6 МБ

Зоны отображения:
- ТВ: 2560×440 px
- Компьютер: 2560×423 px
- Планшет: 1850×423 px
- СЕЙФ-ЗОНА (телефон): 1546×423 px ⭐
```

### Правило сейф-зоны
**Всё важное — только в сейф-зоне!**

### Что размещать
- Спикер (крупно, лицо видно)
- Логотип/имя
- Короткий слоган
- График выхода видео

### ❌ Не размещать
- Много текста
- Тонкие шрифты
- Мелкие детали
- Важное за пределами сейф-зоны

---

## Аватарка

### Размеры
```
Минимум: 800×800 px
Рекомендуется: 1080×1080 px
Вес: <4 МБ
Формат: Квадрат (отображается кругом)
```

### Варианты
- Логотип (для компаний) — крупно
- Лицо спикера — крупно, на нейтральном фоне

### Проверка
- Нарисовать круг в квадрате
- Проверить, что обрезается

---

## Водяной знак

### Размеры
```
150×150 px минимум
Формат: PNG (прозрачный фон)
```

### Что размещать
- "Подпишись"
- Или логотип

### Настройка
- На протяжении всего видео
- Или с 5-й секунды

### Лайфхак
- Добавить тень под квадратик
- Будет видно на любом фоне

---

## Настройки канала

### Страна
- Выбирать реальную (Россия)
- МИФ: "заблокируют" — неправда

### Аудитория
- "Нет, мой канал не для детей" ✅
- Настроить один раз

### Реклама на основе интересов
- ДО монетизации: не трогать
- ПОСЛЕ монетизации: НЕ отключать (больше дохода)

---

## Главная страница канала

### Структура вкладки "Главное"
```
1. Для вас (персонализация)
2. Видео
3. Короткие видео (Shorts)
4. Плейлисты
```

### Продвинутая стратегия
- Вынести продающие видео вначало
- Видео с высокой конверсией

---

## Цветовая схема по нишам

| Ниша | Стиль |
|------|-------|
| Тортики/Сладости | Розовые, пастельные |
| Байкеры | Коричневые, чёрные, грубые |
| Премиум | Изумрудный, золотой, минимализм |

**Принцип:** Стереотипные цвета = быстрое распознавание ниши

---

## Превьюшки — базовые правила

### Размер
```
1920×1080 px
Формат: JPEG или PNG
```

### Текст
- **Максимум 3-5 слов**
- Жирный шрифт
- Контрастные цвета
- Два цвета: основной + второстепенный

### Триггерные элементы
- Вопросительные знаки (?)
- Восклицательные знаки (!)
- Стрелочки
- Цифры
- Чёрные силуэты

### Спикер на превью
- Крупное лицо
- С эмоцией (удивление, радость, шок)
- Эмоция должна легко считываться

### Единый стиль
- Все превью в одном стиле
- Люди запоминают визуал раньше лица
