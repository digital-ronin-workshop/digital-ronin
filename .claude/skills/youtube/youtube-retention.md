# YouTube Retention & Transitions

## Когда использовать
- Работа с удержанием на всей длине
- Написание переходов между секциями
- Создание open loops
- Анализ retention графика

---

## Open Loops

**Open loop = незакрытый вопрос, заставляющий смотреть дальше**

### 4-step Framework:
1. **Start with curiosity** — создай вопрос
2. **Build tension** — не раскрывай сразу
3. **Strategic delay** — добавляй новые loops
4. **Always close** — обязательно закрой каждый

**ПРАВИЛО:** Каждый open loop ДОЛЖЕН быть закрыт.

---

## Open Loop Patterns

### 1. Contrast Loop
```
"[THING] is important, but 99% get this wrong."
```

### 2. Delayed Reveal
```
"I'll show you [PROMISE], but first [PREREQUISITE]."
```

### 3. Mistake Tease
```
"This one mistake cost me [BAD OUTCOME]. Stay tuned."
```

### 4. Layered Loop
```
Main: "[BIG PROMISE]"
Secondary: "But before that, let me tell you about [X]"
Supporting: "The reason this works is fascinating..."
```

### 5. Question Implant
```
"How did [PERSON] achieve [RESULT]?
Is [METHOD] effective?
Can I use it for [MY SITUATION]?"
→ Затем отвечай на каждый вопрос
```

## Transition Formula

```
1. Wrap up current point (zinger)
2. Open loop to next point
3. Setup next payoff
```

### Wrap-up Zingers
- "That's literally it."
- "Simple as that."
- "Does that make sense?"
- "And that's [TOPIC]."

### Transition Openers
- "So now that you know [X], let's talk about [Y]."
- "But [X] is only half the battle."
- "Speaking of [X], this leads me to [Y]."
- "And this brings us to the most important part..."

### Curiosity Bridges
- "But wait, there's something I haven't told you."
- "The trick that made the difference wasn't expected."
- "But [THING] alone won't work. You also need [NEXT]."

## Retention Triggers

### Pattern Interrupts
- Изменение темпа/энергии
- Визуальные изменения
- Смена формата (talking head → screen share)
- Unexpected reveal

### Curiosity Sustainers
- "But here's the thing..."
- "And it gets better..."
- "Wait, it gets worse..."
- "You're probably wondering..."

### Engagement Hooks
- Promise списка ("5 steps", "3 mistakes")
- Numbered payoffs с build-up
- "Stay to the end and I'll show you..."
- Mega payoff teaser в начале

## Техника "Вопрос из-за кадра"

```
Место: когда нужна причастность
Формат: Кто-то задаёт вопрос → спикер отвечает
```

**Преимущества:**
- Зритель = "подсматриватель" беседы
- Удерживает внимание
- Добавляет динамику
- Можно задавать неудобные вопросы

## Layering (A-plot + B-plot)

```
A-plot: Главная тема видео
B-plot: Второстепенная история/тема

A-plot создаёт структуру
B-plot добавляет глубину и интерес
```

---

## 3 ошибки с Open Loops

| Ошибка | Проблема | Решение |
|--------|----------|---------|
| Too vague | "You won't believe..." — generic | Быть specific |
| Resolving too soon | Нет tension | Delay на 3-4 минуты |
| Forgetting to close | Broken promise | Всегда закрывать |

---

## Техника "Снимай-удаляй"

Для анализа retention:

1. Сними видео
2. Посмотри retention график
3. В местах падения — удали/перепиши
4. Повтори

**Паттерны падения:**
- Длинное вступление → сократи
- Нет payoff после setup → добавь
- Монотонный темп → pattern interrupt
- Нет open loop перед секцией → добавь

## Open Loops Map

При написании скрипта веди карту:

```
Loop 1: [Setup в Hook] → [Closed at: Payoff 2]
Loop 2: [Setup в Payoff 1] → [Closed at: Payoff 3]
Loop 3: [Setup в Payoff 2] → [Closed at: CTA]
```

**Проверка:** Все loops закрыты? Нет "висящих" обещаний?

---

## Anti-patterns

**В Transitions:**
- "Moving on..." — weak
- "Next..." — no curiosity
- "Let's dive into..." — filler
- "Here's a harsh truth..." — overused

**В Open Loops:**
- Стекать loops без закрытия
- Generic teasers
- Обещать и не выполнять

---

## Чек-лист retention

- [ ] Open loops создают curiosity
- [ ] Все loops закрыты к концу
- [ ] Zinger перед каждым переходом
- [ ] Pattern interrupts каждые 2-3 минуты
- [ ] A-plot + B-plot для глубины
- [ ] Нет монотонных участков > 2 минут
