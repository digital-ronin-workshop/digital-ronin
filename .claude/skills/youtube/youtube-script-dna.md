## Hook Formulas

### 1. Credential + Promise
```
"In the past [TIME], I've [CREDENTIAL]. And now I'm going to [PROMISE]."

Пример:
"In the past 9 months, I've written YouTube scripts for 40+ educational channels
and helped them gain millions of views. And now I'm going to teach you everything
I know about writing scripts."
```

### 2. Vulnerability → Authority
```
"I don't know a whole lot about [X]. I'm not that good at [Y].
But if there's one thing I'm good at, it's [YOUR EXPERTISE]."

Пример:
"I don't know a whole lot about YouTube. I'm not that good at coming up with ideas.
My thumbnails are okay. But if there's one thing that I'm good at,
it's how to write a great YouTube script."
```

### 3. Input Bias Hook
```
"I spent [TIME] [DOING RESEARCH], so all you have to do is watch this video."

Пример:
"I spent the past 6 months watching all of Alex's content to figure out
what he's doing differently. So all you have to do is watch this video."
```

### 4. Title Validation + Stakes
```
"[REPEAT TITLE]. [ADD STAKES/BENEFIT]."

Пример:
"Listen up, because Iman Gaji made $40 million from one YouTube video.
And he used it to buy his private jet."
```

### 5. Problem Agitation
```
"I know what it's like [PROBLEM]. It feels like [EMOTIONAL DESCRIPTION].
But let me tell you something..."

Пример:
"I know what it's like being stuck at a 9-to-5 job that you hate.
It feels like you're stuck on a treadmill that you can't afford to step off.
But let me tell you something..."
```

## Open Loop Patterns

### 1. Contrast Loop
```
"[THING] is important, but 99% of creators get this wrong."

Пример:
"The first principle has to do with something that 99% of creators get wrong,
but is the most important if you want to convert viewers into sales."
```

### 2. Delayed Reveal
```
"I'll show you [PROMISE], but first you need to understand [PREREQUISITE]."

Пример:
"I'll show you how I doubled my income in 30 days,
but the way I did it might surprise you. More on that in a minute."
```

### 3. Mistake Tease
```
"This one mistake cost me [NEGATIVE OUTCOME]. Stay tuned to find out what it was."
```

### 4. Layered Loop
```
Main loop: "[BIG PROMISE]"
Secondary loop: "But before I show you that, let me tell you about [RELATED THING]"
Supporting loop: "The reason this works is fascinating..."
```

### 5. Question Implant
```
"How did [PERSON] achieve [RESULT]?
Is [METHOD] effective?
Is it easy to implement?
Can I use it for [MY SITUATION]?"

→ Затем ответь на каждый вопрос в hook
```

- "Speaking of [X], this leads me to [Y]."

### Curiosity Bridges
- "But wait, there's something I haven't told you yet."
- "And the trick that made all the difference wasn't what you expected."
- "But [THING] alone won't work. You also need [NEXT THING]."

## Payoff Structures

### 1. Problem → Solution → Proof
```
SETUP: "Most people [PROBLEM]"
DEVELOP: "Here's why that happens: [EXPLANATION]"
SOLUTION: "Instead, you should [SOLUTION]"
PROOF: "For example, [SPECIFIC CASE]"
ZINGER: "[ONE-LINE SUMMARY]"
```

### 2. Mistake → Why It's Wrong → How to Fix
```
SETUP: "Mistake #[N]: [MISTAKE]"
WHY WRONG: "This is problematic because [REASON]"
EXAMPLE: "For example: [BAD EXAMPLE]"
FIX: "Here's how to fix it: [SOLUTION]"
GOOD EXAMPLE: "So instead: [GOOD EXAMPLE]"
```

### 3. Framework Reveal
```
SETUP: "I've created [FRAMEWORK NAME] for [PURPOSE]"
OVERVIEW: "It has [N] parts: [LIST]"
DEEP DIVE: Each part gets its own mini-payoff
SUMMARY: "So remember: [FRAMEWORK NAME] = [COMPONENTS]"
```

### 4. Story → Lesson
```
SETUP: "Let me tell you about [PERSON/SITUATION]"
CONTEXT: "[BACKGROUND]"
CONFLICT: "[WHAT WENT WRONG/CHALLENGE]"
RESOLUTION: "[WHAT THEY DID]"
LESSON: "The takeaway here is [PRINCIPLE]"
```

## CTA Templates

### Video End Screen CTA
```
HOOK: "So that's [SUMMARY OF VIDEO]."
CURIOSITY: "But [NEW PROBLEM] means you'll need [NEXT THING]."
ACTION: "Which is why I created this video. Click here to watch it
and I'll see you in the next one."
```

### Course/Product CTA (40+ min videos only)
```
HOOK: "Now that you know [EVERYTHING FROM VIDEO]..."
GAP: "The only thing missing is [WHAT THEY STILL NEED]."
SOLUTION: "That's why I created [PRODUCT]."
BENEFIT: "It will help you [SPECIFIC OUTCOME]."
ACTION: "Click the link in the description to [ACTION]."
```

### Soft CTA (Mid-video)
```
"If you want [RESOURCE], click the link in description.
I'll send it right to your inbox. It will make [BENEFIT]."
```

## Logic Flow Principles

### Sentence → Sentence
```
Line 1 → должна вести к Line 2
Line 2 → должна вести к Line 3

Если можно удалить предложение без потери смысла → удали
```

### The "So What?" Chain
```
Claim: "[STATEMENT]"
So what? → "[WHY IT MATTERS]"
So what? → "[WHAT TO DO ABOUT IT]"
So what? → "[WHAT HAPPENS IF YOU DO/DON'T]"
```

### Setup → Payoff Balance
```
Short setup → Quick payoff (for obvious points)
Long setup → Big payoff (for complex/surprising reveals)

Rule: The bigger the buildup, the bigger the payoff must be
```

## Word Economy Rules

### Cut These:
- "I think that..." → just state it
- "In order to..." → "To..."
- "The reason why is because..." → "Because..."
- "At this point in time..." → "Now..."
- "Due to the fact that..." → "Because..."

### Keep Sentences Short
- Target: 5-7th grade reading level
- Ideal sentence length: 10-15 words
- Max: 25 words before breaking up

### One Idea Per Sentence
```
BAD: "When you're writing hooks and you want to create curiosity,
you should avoid jargon and also make sure to validate the click."

GOOD: "Strong hooks create curiosity. They avoid jargon.
And they validate the click."
```

## Anti-Patterns (Чего избегать)

### В Hooks
- "Welcome to my channel" — boring
- "In this video, we're going to talk about..." — too generic
- Starting with credentials paragraph — nobody asked
- "You won't believe what happens next" — overused

### В Payoffs
- Stacking 3+ payoffs without breathing room
- Making claims without "so what?"
- Jumping topics without transition
- Repeating the same point with different words

### В Transitions
- "Moving on..." — weak
- "Next..." — no curiosity
- "Let's dive into..." — filler
- "Here's a harsh truth..." — overused

### В CTAs
- "Like and subscribe" — if content is good, they will
- Pitching product in short videos — not enough trust built
- Vague benefits — "click here for more" — why?

## Примеры применения

### Тема: "How to Escape the Rat Race Without Quitting Your 9-to-5"

**Hook (Vulnerability + Stakes):**
```
I know what it's like being stuck at a 9-to-5 job that you hate.
It feels like you're stuck on a treadmill that you can't afford to step off.

But contrary to popular belief, you don't have to quit your job
and take on large amounts of risk.

I've personally started multiple businesses that I've grown to multi-7 figures.
And along the way, I've helped tens of thousands of people escape the rat race
the right way.

In this video, I'm going to walk you through the four principles
that once you internalize and implement into your daily life,
will help you finally achieve financial freedom.
```

**Payoff 1 Setup → Open Loop:**
```
The first principle has to do with something that 99% of people get wrong,
but is the most important if you want to actually escape the rat race.

[PAYOFF CONTENT]

And that's the first principle. But knowing this alone won't set you free.
The second principle is what separates people who stay stuck from those who actually escape.
Let me show you what I mean...
```

## ВИСП Framework (Открытия/Openers)

**ВИСП = 4 компонента, которые зритель должен получить с первых секунд:**

| Компонент | Значение | Вопрос для проверки |
|-----------|----------|---------------------|
| **В**ыгода | Конкретная польза после просмотра | Что получит зритель? |
| **И**нтрига | Неопределённость, желание узнать | Есть ли вопрос в голове? |
| **С**рочность | Ощущение "надо СЕЙЧАС" | Почему не отложить? |
| **П**ричастность | Это про меня, моя история | Для кого это? |

### Универсальный шаблон ВИСП
```
[СЛОВО СРОЧНОСТИ] + [ВЫГОДА] + [ДЛЯ ЦЕЛЕВОЙ ГРУППЫ] + [ИНТРИГА]

Пример:
"Секрет для тех, кому надоело давить комедоны, который спас сотни моих клиентов"
- Выгода: "спас клиентов"
- Интрига: слово "секрет"
- Срочность: "сотни клиентов" (другие уже получили)
- Причастность: "кому надоело" (прямое обращение к боли)
```

### Слова-убийцы (закрывают несколько компонентов сразу)
- "секрет" — интрига + выгода
- "трюк" — интрига + простота
- "лайфхак" — выгода + срочность
- "способ" — выгода
- "техника" — интрига + авторитет

## Три вектора срочности

### 1. Актуальность (по времени)
```
Слова: "сейчас", "новое", "в 2025", "в этом сезоне", "пока не поздно"

Пример: "Срочно сделай это до конца года"
```

### 2. Уклонение (от боли)
```
Повелительные наклонения: "сделай", "очнись", "проснись", "верни", "перестань"

Пример: "Перестань терять деньги на рекламе"
```

### 3. Ресурсы (много результата за мало затрат) — САМЫЙ МОЩНЫЙ
```
- Деньги: "дешево", "за копейки", "бесплатно"
- Время: "быстро", "за неделю", "моментально"
- Усилия: "легко", "просто", "без подготовки"
- Без специалиста: "дома", "своими руками", "без докторов"

Пример: "Как за две недели избавиться от купероза — доступно каждому,
стоит копейки, продается в аптеках"
```

---

## Три уровня причастности

### Уровень 0 — Базовый
```
"Для вас", "Для новичков", "Для всех"
```

### Уровень 1 — Образы и ситуации
```
Описывай ситуации, которые аудитория узнает:
- "покупаешь книги и не читаешь"
- "давишь комедоны каждый вечер"
- "сидишь на работе и мечтаешь уволиться"
```

### Уровень 2 — Специфичность
```
Чем уже проблема, тем сильнее причастность:
"проблемы с кожей" < "прыщи" < "комедоны на носу"
```

## Логлайн (Logline)

**Логлайн = суть видео в 1-2 предложениях. Пиши ДО сценария.**

```
Формула: [КТО] + [ЧТО ДЕЛАЕТ/УЗНАЕТ] + [ЗАЧЕМ/РЕЗУЛЬТАТ]

Примеры:
- "Девушка узнает 3 шага, чтобы перестать быть удобной и начать жить свою жизнь"
- "Эксперт показывает, как за копейки избавиться от купероза дома"
- "Зритель получает пошаговый план выхода из крысиных бегов без увольнения"
```

Если не можешь написать логлайн — не понимаешь, о чём видео.

## CTA по типам (Курс КМК)

### CTA на лайк
```
Место: после первого payoff или после причастного момента
Формула: Причастность + просьба

Пример: "Ставьте лайк, если знакомо"
Пример: "Если узнали себя — палец вверх"
```

### CTA на комментарий
```
Место: после решения проблемы
Формула: Социальная ответственность + конкретный вопрос

Пример: "Дорогие девочки, если вы уже прошли этот этап и научились говорить 'нет',
пожалуйста, напишите свою историю. Это поможет нашей аудитории увидеть,
что в этом нет ничего страшного."
```

### CTA на подписку
```
Место: в конце видео
Формула: Окружение + ценности

Пример: "Правильное окружение — там, где люди схожи с вами по ценностям.
Если вы смотрите это видео — вы уже в правильном месте.
Подписывайтесь и будьте среди тех, кто хочет лучшего будущего."
```

### CTA на продукт/ссылку
```
Место: после payoff, через другую проблему
Формула: Новая проблема + простое решение

Пример: "Это легко сказать — говорите о своих чувствах.
Но как понять, что это не навязанные обществом чувства?
Есть список вопросов, чтобы отделить себя от социума.
Они есть у нас в ТГ-канале — переходите по ссылке."
```

## PCM типы личности для CTA

| Тип | Мотивация | Как обращаться |
|-----|-----------|----------------|
| **Деятель** | Действие, результат | "Сделай", "Попробуй", "Начни сейчас" |
| **Бунтарь** | Уникальность, против системы | "Не будь как все", "Выделись" |
| **Логик** | Факты, доказательства | "Исследования показали", "Статистика" |
| **Упорный** | Ценности, убеждения | "Важно", "Правильно", "Должен" |
| **Мечтатель** | Визуализация, будущее | "Представь", "Вообрази", "Однажды" |
| **Душевный** | Эмоции, отношения | "Почувствуй", "Близкие", "Вместе" |

## Упражнение "Астрофизика" (проверка вопросов)

**Метод валидации вопросов через YouTube:**

1. Напиши вопросы, которые хочешь задать в видео
2. Вбей каждый вопрос в поиск YouTube
3. Посмотри просмотры на видео с похожими вопросами
4. Если есть референсы с 500K+ просмотров — вопрос хороший
5. Если референсов нет — вопрос душный

```
Хорошие вопросы касаются лично зрителя:
"Что будет со МНОЙ?" > "Как это работает?"
"Могу ли Я это использовать?" > "Интересный факт"

Техника: спроси себя "А меня это как ебёт?"
```

## Квадрат Декарта для усугубления

| | Произойдёт | Не произойдёт |
|---|------------|---------------|
| **Будет** | Что будет, если это произойдёт? | Чего НЕ будет, если это произойдёт? |
| **Не будет** | Что будет, если это НЕ произойдёт? | Чего НЕ будет, если это НЕ произойдёт? |

Используй разные квадраты для разнообразия усугублений в топ-листах.

## Техника "Вопрос из-за кадра"

```
Место: когда нужна причастность
Формат: Кто-то задаёт вопрос, спикер отвечает

Преимущества:
- Зритель становится "подсматривателем" беседы
- Удерживает внимание
- Добавляет динамику
- Можно задавать неудобные/интересные вопросы

Пример: "Василий, а по вашим ощущениям, как много девушек
в позиции удобной?"
→ Спикер перечисляет признаки с эмоцией
→ Зритель узнаёт себя
```

## AI Prompts для сценариев (Шаблоны)

### Шаблон 1: Тезисный сценарий
```
"Ты будешь писать тезисные сценарии для видео на ютуб-канал.
Структура:
- Вступление — короткое, яркое, цепляющее
- Основная часть — логичные блоки с примерами и фактами
- Заключение — мотивация к действию

Стиль: живой, разговорный, вовлекающий"
```

### Шаблон 2: Полный сценарий с параметрами
```
"Напиши полный сценарий для видео на тему [ТЕМА]:
- Спикер: [кто и чем занимается]
- Продукт: [что продаём]
- ЦА: [описание аудитории]
- Боль: [какую проблему решает видео]

Начало триггерное, завершение мотивирующее."
```

### Шаблон 3: Шортс
```
"Напиши сценарий для шортс — короткий и цепляющий.
Тема: [ТЕМА]
Без приветствия и прощания, сразу к сути.
Динамичный, с ярким началом.
Если есть ответ на вопрос — держи интригу до конца."
```

## Чек-лист AI-сценария

После генерации проверь:

- [ ] Факты перепроверены (AI может выдумать)
- [ ] Нет повторов одной мысли разными словами
- [ ] Логика выстроена по порядку
- [ ] Цифры и расчёты проверены
- [ ] Имена и названия точные
- [ ] Нет сложных слов (если нужен простой стиль)
- [ ] Учтены региональные особенности
- [ ] Исследования и ссылки проверены
