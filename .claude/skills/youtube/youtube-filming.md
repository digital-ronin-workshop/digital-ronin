# YouTube Filming (Съёмка)

## Когда использовать
- Настройка оборудования
- Постановка света
- Съёмка на телефон/камеру
- Работа со звуком

---

### Оборудование

**Камера — iPhone:**
- Минимум: iPhone 11/12 Pro Max
- Оптимально: iPhone 15/16 Pro Max
- Киноэффект (размытие фона)

**Звук — Петличка:**
| Модель | Цена | Комментарий |
|--------|------|-------------|
| HolyLand | до 15K руб | Хорошее соотношение |
| **DJI** | 23-25K руб | Рекомендуется, все переходники |

**Свет:**
- На улице: естественный
- В помещении: см. раздел "Свет"

## Настройки iPhone

### Основные
```
Разрешение: 4K (всегда!)
FPS: 25 или 30 (говорящая голова)
FPS: 50 или 60 (для замедления)
Формат: "Наиболее совместимый" ✅
HDR: ВЫКЛЮЧИТЬ ❌
```

### Почему эти настройки

| Параметр | Причина |
|----------|---------|
| 4K | Можно кропнуть на монтаже |
| 25-30 FPS | Достаточно для речи |
| Совместимый формат | H.265 сложнее монтировать |
| HDR выкл | На Windows/Android выглядит иначе |

## Свет

### ❌ Не делать
- Потолочный свет
- Открытое окно (свет меняется)

### ✅ Делать
- Один раз поставить хороший свет

### Бюджетный вариант
- Кольцевая лампа (2 шт)
- Световые панели

### Рекомендуемый сетап (~15-20K руб)
```
Godox SL60W (блок): 10-12K руб
Стойка: 2-3K руб
Софтбокс (круглый): 3-5K руб
```

### Постановка рисующего света
```
1. Спикер садится
2. Свет выше спикера на 15-20-30 см
3. Наклон: 45 градусов
4. Расстояние: ~2 метра
```

### Контровой свет
- Чуть выше основного
- Отбивает от фона
- Создаёт контур на волосах/плечах

### Проблема: Блеск на коже
```
Решения:
1. Матирующие салфетки
2. Отодвинуть свет дальше
3. Увеличить мощность
```

## Камеры

### Принцип выбора
```
"Плохая камера + хороший объектив >
офигенная камера + плохой объектив"
```

### Хороший объектив
- Фиксированное фокусное: 24mm, 35mm, 50mm, 85mm
- Минимальная диафрагма: f/1.4, f/1.8, f/2.0

### Рекомендуемые камеры (Sony)
| Уровень | Модели |
|---------|--------|
| Топ | FX3, FX30 |
| Средний | Alpha 7 III/IV |
| Бюджет | A6000 серия, ZV-E10, ZV-1 |

### Ограничения бюджетных камер
- Запись до 30 минут
- Перегрев (решение: кулеры на Type-C)
- Питание (решение: Type-C от розетки)

---

## Настройки камеры

### Порядок настройки
```
1. Диафрагма → максимально открыть (f/1.4-2.8)
2. Выдержка → кратно FPS (1/50 для 25fps)
3. ISO → регулировать последним
```

### Важно
- Сильно поднятый ISO = шум на тёмных тканях

---

## Постановка двух камер

### Схема
```
Камера 1: Общий план (по горизонту)
Камера 2: Крупный план (чуть сверху)
```

### Правила

**Рабочая сторона спикера:**
- Губа чуть выше при разговоре
- Подойти справа/слева и посмотреть

**Крупный план:**
- На уровне лба/глаз
- Наклон чуть вниз
- ❌ НЕ снизу (видны неровности)

**Композиция:**
- Общий план: центр
- Крупный план: сдвинуть вбок
- Пустое место — со стороны взгляда

### Правило третей
- Включить сетку в камере
- Ставить объект по линиям
