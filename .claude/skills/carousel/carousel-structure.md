# Carousel Structure

Формулы структуры: Hook → Body → CTA

---

## Общая структура карусели

```
┌─────────────────────────────────────────┐
│  СЛАЙД 1: HOOK                          │
│  Останавливает скролл                   │
│  Интригует свайпнуть                    │
└─────────────────────────────────────────┘
              ↓
┌─────────────────────────────────────────┐
│  СЛАЙДЫ 2-N: BODY                       │
│  Раскрытие темы                         │
│  1 идея = 1 слайд                       │
└─────────────────────────────────────────┘
              ↓
┌─────────────────────────────────────────┐
│  ПОСЛЕДНИЙ СЛАЙД: CTA                   │
│  Призыв к действию                      │
│  Конверсия в подписку/комментарий       │
└─────────────────────────────────────────┘
```

---

## HOOK (Слайд 1)

### Цель
- Остановить скролл
- Вызвать желание свайпнуть
- Передать ценность карусели

### 3 триггера Hook

| Триггер | Как работает | Пример |
|---------|--------------|--------|
| **Зависть** | Хочу таких результатов | "Как я получил 50K просмотров с одной карусели" |
| **Жадность** | Хочу эту выгоду | "7 AI-инструментов, которые заменят дизайнера" |
| **Любопытство** | Хочу узнать | "Все говорят о ChatGPT, но я нашёл кое-что получше..." |

### 8 формул Hook для карусели

#### 1. Контрарный
```
"[Общепринятое мнение] — это миф"
"Почему [популярный совет] больше не работает"
```
Пример: "Canva убивает ваш дизайн. Вот почему."

#### 2. Числа + Обещание
```
"[N] [что-то], которые [результат]"
"[N] способов [получить выгоду]"
```
Пример: "9 YouTube курсов, которые дадут больше, чем 4 года универа"

#### 3. Новость / Breaking
```
"NEW: [что случилось]"
"BREAKING: [важная информация]"
```
Пример: "BREAKING: Instagram CEO подтвердил, что Edits не буст"

#### 4. Секрет / Эксклюзив
```
"[Кто-то] скрывали это от вас"
"Секрет [экспертов/успешных], о котором не говорят"
```
Пример: "Creators kept these websites to themselves"

#### 5. Результат / Кейс
```
"Как я [достиг результата] за [срок]"
"[Результат] с помощью [метода]"
```
Пример: "Как я заработал первые $10K на AI-арте"

#### 6. Предсказание
```
"[N] прогнозов на [год]"
"Что изменится в [сфере] в [период]"
```
Пример: "7 predictions for Instagram in 2026"

#### 7. Вопрос-провокация
```
"Почему [большинство] делают [что-то] неправильно?"
"Ты тоже совершаешь эту ошибку?"
```
Пример: "Почему 99% AI-промптов не работают?"

#### 8. Пошаговый гайд
```
"Как [сделать что-то] за [N] шагов"
"Полный гайд по [теме]"
```
Пример: "Как делать реалистичные генерации с продуктом на PRO уровне?"

### Элементы Hook слайда

```
┌─────────────────────────────────────┐
│                                     │
│    [ВИЗУАЛ: фото/AI-изображение]    │
│                                     │
│  ┌─────────────────────────────┐    │
│  │  ЗАГОЛОВОК (макс 7 слов)   │    │
│  │  Крупный, читаемый          │    │
│  └─────────────────────────────┘    │
│                                     │
│  Подзаголовок (опционально)         │
│  "Swipe for details →"              │
│                                     │
└─────────────────────────────────────┘
```

### Hook badges (опционально)

| Badge | Когда |
|-------|-------|
| NEW | Свежая информация |
| BREAKING | Срочная новость |
| FREE | Бесплатный ресурс |
| SECRET | Эксклюзив |
| LUXURY | Премиум контент |

## BODY (Слайды 2-N)

### Правило "1 идея = 1 слайд"

**Плохо:**
```
Слайд 2: "Первые 3 инструмента: Midjourney, DALL-E, Stable Diffusion..."
```

**Хорошо:**
```
Слайд 2: Midjourney — [почему крут]
Слайд 3: DALL-E — [почему крут]
Слайд 4: Stable Diffusion — [почему крут]
```

### Структуры Body

#### Структура: Список
```
Слайд 2: Пункт 1 — [название + краткое описание]
Слайд 3: Пункт 2 — [название + краткое описание]
...
Слайд N: Пункт N — [название + краткое описание]
```
Когда: Топ-N, инструменты, советы

#### Структура: Пошаговая инструкция
```
Слайд 2: Шаг 1 — [что делать]
Слайд 3: Шаг 2 — [что делать]
...
Слайд N: Шаг N — [что делать]
```
Когда: Туториалы, гайды, процессы

#### Структура: Проблема → Решение
```
Слайд 2: Проблема — [боль аудитории]
Слайд 3: Почему это важно — [последствия]
Слайд 4-N: Решение — [как исправить]
```
Когда: Ошибки, антипаттерны

#### Структура: До → После
```
Слайд 2: Было — [старый способ]
Слайд 3: Стало — [новый способ]
Слайд 4-N: Как достичь — [шаги]
```
Когда: Трансформации, результаты

#### Структура: Промпты
```
Слайд 2: [AI-фото] + [промпт который его создал]
Слайд 3: [AI-фото] + [промпт]
...
Слайд N: [AI-фото] + [промпт]
```
Когда: Prompt carousels

### Ограничения текста на слайде

| Элемент | Лимит |
|---------|-------|
| Заголовок | 7 слов |
| Основной текст | 20 слов |
| Подпись/детали | 15 слов |
| **Всего на слайде** | **~40 слов макс** |

### Визуальная иерархия

```
┌─────────────────────────────────────┐
│                                     │
│         [ВИЗУАЛ 60-70%]             │
│                                     │
├─────────────────────────────────────┤
│  ЗАГОЛОВОК СЛАЙДА (крупно)          │
│  Детали/описание (мельче)           │
│                                     │
└─────────────────────────────────────┘
```

## CTA (Последний слайд)

### Цель
- Конвертировать просмотр в действие
- Получить подписку / комментарий / сохранение

### Типы CTA

#### 1. Follow CTA
```
"Follow for more [тема]"
"Подписывайся, чтобы не пропустить"
```
Визуал: Фото автора + плашка с ником

#### 2. Comment CTA
```
"Comment 'LINK' and I'll send you the full list"
"Напиши 'ДА' в комментарии"
```
Визуал: Простой фон + крупный текст

#### 3. Save CTA
```
"Save this for later"
"Сохрани, чтобы не потерять"
```
Визуал: Иконка сохранения + текст

#### 4. Engagement CTA
```
"Was this helpful? Follow me for more"
"Какой совет понравился больше всего?"
```
Визуал: Фото автора + вопрос

#### 5. Link CTA
```
"Full guide in bio"
"Link in comments"
```
Визуал: Стрелка к bio/комментариям

### Элементы CTA слайда

```
┌─────────────────────────────────────┐
│                                     │
│     [ФОТО АВТОРА / АВАТАР]          │
│           (круглое)                 │
│                                     │
│  ┌─────────────────────────────┐    │
│  │    @username                │    │
│  └─────────────────────────────┘    │
│                                     │
│      ПРИЗЫВ К ДЕЙСТВИЮ              │
│      "Follow for more"              │
│                                     │
│   [иконки: сердце, коммент, шэр]    │
│                                     │
└─────────────────────────────────────┘
```

## Оптимальное количество слайдов

| Тип контента | Слайдов | Почему |
|--------------|---------|--------|
| Короткий список (3-5 пунктов) | 5-6 | Hook + пункты + CTA |
| Средний список (6-8 пунктов) | 8-9 | Hook + пункты + CTA |
| Длинный гайд | 10 | Максимум Instagram |
| Промпт карусель | 8-10 | Много визуалов |
| Цитаты/мысли | 5-7 | Не перегружать |

**Формула:** `Слайдов = 1 (Hook) + N (пунктов) + 1 (CTA)`

## Связность между слайдами

### Визуальная связность
- Одна цветовая палитра
- Одинаковые шрифты
- Консистентный стиль фото
- Единое освещение

### Нарративная связность
- Логичный порядок (от простого к сложному / по важности)
- Переходные фразы: "Next...", "But wait...", "The best part..."
- Нумерация пунктов

### Transition hints
```
Слайд N: "...but that's not all →"
Слайд N: "The next one is my favorite →"
Слайд N: "Swipe for the game-changer →"
```
