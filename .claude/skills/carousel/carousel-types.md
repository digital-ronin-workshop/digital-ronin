# Carousel Types

5 типов каруселей с детальным описанием и примерами.

---

## Выбор типа карусели

| Тип | Когда использовать | Сложность |
|-----|-------------------|-----------|
| **Prompt** | AI-промпты, обучение генерации | Средняя |
| **Expert** | Советы, инструкции, экспертный контент | Низкая |
| **Minimalist** | Цитаты, мысли, чистый контент | Низкая |
| **Viral** | Хайп, новости, интрига | Средняя |
| **Grid** | Много пунктов, обзоры, сравнения | Высокая |

---

## 1. PROMPT CAROUSEL

### Описание
AI-фотографии с промптом, который их создал, прямо на слайде.
Идеально для контента про AI-генерацию.

### Примеры аккаунтов
- @talesramiroart
- @filatov.design

### Структура
```
Слайд 1: HOOK
├── AI-фото (результат генерации)
├── Заголовок: "LUXURY PROMPTS" / "Copy these prompts"
└── "Swipe to copy →"

Слайды 2-N: PROMPT SLIDES
├── AI-фото (80% слайда)
├── Промпт overlay внизу (20% слайда)
│   └── Полный текст промпта, белый на тёмной плашке
└── Номер слайда (опционально)

Слайд CTA:
├── Фото автора / аватар
├── "Comment 'PROMPTS' and I'll send you all"
└── @username
```

### Визуальный стиль
```
ФОТО:
- Высококачественные AI-генерации
- Консистентный стиль (все luxury / все cinematic / все portrait)
- Aspect ratio: 4:5

ПРОМПТ OVERLAY:
- Позиция: нижние 20% изображения
- Фон: полупрозрачный тёмный (#000000, opacity 70%)
- Текст: белый, sans-serif, 12-14pt
- Выравнивание: по левому краю
- Padding: 16px
```

### Особенности NanoBanana промпта
```
Опция A (текст в NanoBanana):
"Include text overlay at the bottom: '[ПРОМПТ ТЕКСТ]'
Semi-transparent dark banner, white sans-serif text.
Text should be readable, not exceed 20% of image height."

Опция B (текст в редакторе):
"Leave clean space at the bottom 20% of the image.
No visual elements in this zone for text overlay later."
```

### Лучшие темы
- Luxury lifestyle prompts
- Portrait photography prompts
- Product photography prompts
- Cinematic scene prompts
- Fantasy/Sci-Fi prompts

## 2. EXPERT CAROUSEL

### Описание
Фото эксперта (автора) + образовательный контент.
Идеально для советов, списков, инструкций.

### Примеры аккаунтов
- @brock11johnson
- @awakpenn

### Структура
```
Слайд 1: HOOK
├── Фото эксперта (поясной/крупный план)
├── Badge: "NEW" / "FREE" / номер
├── Заголовок: "7 tips that..." / "9 courses that..."
├── Подзаголовок (опционально)
└── "Swipe for details"

Слайды 2-N: CONTENT SLIDES
├── Визуал (скриншот/иллюстрация/фото)
├── Номер пункта (Video 01, Tip #3, etc.)
├── Заголовок пункта
├── Краткое описание (1-2 предложения)
└── Автор (если курс/ресурс): "By [Name]"

Слайд CTA:
├── Фото эксперта
├── Текст: "Want more [тема] advice?"
├── "Follow me for strategies that work!"
└── @username
```

### Визуальный стиль
```
HOOK СЛАЙД:
- Фото эксперта: профессиональное, дружелюбное
- Фон: контекстный (офис, студия, улица)
- Текст: крупный, контрастный
- Badge: яркий акцент (жёлтый, красный)

CONTENT СЛАЙДЫ:
- Единый шаблон для всех
- Нумерация: заметная, стилизованная
- Цветовая схема: 2-3 цвета максимум
```

### Особенности NanoBanana промпта
```
Слайд 1 (с экспертом):
"Professional photo of [описание человека], [pose], in [environment].
[Lighting]. [Expression: friendly, confident].
Shot on [camera]. Aspect ratio 4:5.
Leave space at bottom for large text overlay."

Content слайды:
Часто не требуют AI-генерации — используются:
- Скриншоты интерфейсов
- Стоковые фото
- Графика из Canva
```

### Лучшие темы
- Топ-N курсов/ресурсов
- Советы по [теме]
- Ошибки, которых избегать
- Инструменты для [задачи]

## 3. MINIMALIST CAROUSEL

### Описание
Чистый фон, крупная типографика, минимум визуальных элементов.
Идеально для цитат, мыслей, чистого контента.

### Примеры аккаунтов
- @awakpenn (некоторые посты)

### Структура
```
Слайд 1: HOOK
├── Чистый фон (монохром/градиент)
├── Заголовок: крупный, центрированный
└── "Swipe" или стрелка

Слайды 2-N: CONTENT SLIDES
├── Чистый фон (тот же стиль)
├── Текст: 1-3 предложения
└── Акценты: выделение ключевых слов цветом

Слайд CTA:
├── Фон: тот же стиль
├── Фото автора (маленькое, круглое)
├── @username
└── "Follow for more"
```

### Визуальный стиль
```
ФОН:
- Монохром: чёрный, белый, серый
- Градиент: subtle, 2 близких цвета
- Сетка/линии: subtle grid pattern

ТИПОГРАФИКА:
- Шрифт: bold sans-serif (Montserrat, Inter, SF Pro)
- Размер: крупный (40-60pt для заголовков)
- Выравнивание: по центру или по левому краю
- Межстрочный интервал: увеличенный

АКЦЕНТЫ:
- Выделение слов: цветом (1 акцентный цвет)
- Подчёркивание
- Highlighting (маркер)
```

### Особенности NanoBanana промпта
```
Minimalist карусели обычно НЕ требуют NanoBanana.
Создаются в:
- Canva
- Figma
- Adobe Express

Если нужен AI-фон:
"Abstract minimalist background, [color palette].
Subtle gradient from [color1] to [color2].
Clean, modern aesthetic. No objects or people.
Aspect ratio 4:5."
```

### Лучшие темы
- Цитаты
- Мотивационные мысли
- Простые советы
- Статистика/факты

## 4. VIRAL/BRIGHT CAROUSEL

### Описание
Тёмный фон, яркие неоновые акценты, celebrity faces, интрига.
Идеально для хайпа, новостей, вирального контента.

### Примеры аккаунтов
- @thedigitalkinggg

### Структура
```
Слайд 1: HOOK
├── Тёмный фон с эффектами (дым, свет, неон)
├── Узнаваемые лица (celebrity, известные люди)
├── Интригующий заголовок
├── Яркие акценты (неон, glow)
└── Badge: "SECRET" / "THEY HID THIS"

Слайды 2-N: CONTENT SLIDES
├── Тёмный фон (консистентный)
├── Контент с яркими акцентами
├── Каждый слайд — "открытие секрета"
└── Визуальные эффекты (glow, neon, particles)

Слайд CTA:
├── Фото автора с эффектами
├── Вопрос: "Was this helpful?"
├── "Follow me for more"
└── Иконки соцсетей
```

### Визуальный стиль
```
ФОН:
- Очень тёмный: #0a0a0a, #1a1a1a
- Эффекты: дым, bokeh, particles, gradients

АКЦЕНТЫ:
- Неон: cyan (#00FFFF), magenta (#FF00FF), lime (#00FF00)
- Glow effects на тексте
- Подсветка элементов

ТИПОГРАФИКА:
- Bold, uppercase заголовки
- Несколько цветов в одном заголовке
- Эффект свечения на ключевых словах

CELEBRITY/FACES:
- Узнаваемые люди в индустрии
- Драматичные выражения
- Face-to-camera
```

### Особенности NanoBanana промпта
```
"Dramatic portrait of [person description], facing camera.
Dark moody background with [smoke/particles/bokeh effects].
[Neon color] accent lighting from [direction].
Cinematic, high contrast. [Expression: intense/shocked/secretive].
Shot on Sony A7IV, 85mm lens, f/1.8.
Aspect ratio 4:5."

Для celebrity lookalike (осторожно с правами!):
"Person resembling [description of features, NOT name]..."
```

### Лучшие темы
- Секреты индустрии
- "Они скрывали от вас..."
- Breaking news
- Скандальные разоблачения
- Эксклюзивный контент

## 5. GRID CAROUSEL

### Описание
Несколько мини-карточек на одном слайде (4-6 штук).
Идеально для обзоров, сравнений, больших списков.

### Примеры
- IMG_4892, IMG_4893 из референсов

### Структура
```
Слайд 1: HOOK
├── Сетка превью (2x3 или 3x2)
├── Общий заголовок
└── "Swipe for details"

Слайды 2-N: GRID SLIDES
├── Сетка 2x3 или 2x2
├── Каждая карточка:
│   ├── Иконка/изображение
│   ├── Заголовок
│   └── Краткое описание (5-10 слов)
└── Единый стиль карточек

Слайд CTA:
├── Итоговая карточка или фото автора
├── Summary / key takeaway
└── "Follow for more"
```

### Визуальный стиль
```
СЕТКА:
- 2x2: 4 карточки (крупные)
- 2x3: 6 карточек (средние)
- 3x3: 9 карточек (мелкие, сложно читать)

КАРТОЧКИ:
- Единый размер
- Единый стиль (рамка, тень, радиус)
- Единая цветовая схема
- Gap между карточками: 8-16px

СОДЕРЖИМОЕ КАРТОЧКИ:
- Иконка/фото: 40-60% карточки
- Заголовок: bold, 2-4 слова
- Описание: regular, 1 строка
```

### Особенности NanoBanana промпта
```
Grid карусели обычно собираются в редакторе из:
- Отдельных AI-изображений
- Иконок
- Скриншотов

Для отдельных изображений в сетке:
"[Product/person/scene] on clean [color] background.
Centered composition, object fills 70% of frame.
Soft even lighting, no harsh shadows.
High quality, sharp details.
Square aspect ratio 1:1."

Потом собрать в Canva/Figma в сетку.
```

### Лучшие темы
- Обзор 6-9 инструментов
- Ежедневные привычки
- Сравнение продуктов
- Чеклисты

## Auto-detect логика

Если тип не указан, определить по контенту:

```python
if "промпт" in тема or "AI генерация" in тема:
    тип = "prompt"
elif "топ" in тема or "советы" in тема or "курсы" in тема:
    тип = "expert"
elif "цитата" in тема or "мысль" in тема:
    тип = "minimalist"
elif "секрет" in тема or "скрывали" in тема or "breaking" in тема:
    тип = "viral"
elif количество_пунктов > 6:
    тип = "grid"
else:
    тип = "expert"  # default
```
