# Tone Formulas — Universal

> Универсальные формулы тона для разных типов аккаунтов.

---

## Lifestyle/Personal Account

**Источник:** expert-threads

### ToV Formula
```
Честный 40% + Уязвимый 30% + Практичный 20% + Тёплый 10%
```

### Checklist
- [ ] Первое лицо (я, мне, моё)
- [ ] Уязвимость ("раньше стыдился", "боялся")
- [ ] Практика (конкретные действия, промпты)
- [ ] Теплота ("может и тебе зайдёт")
- [ ] НЕТ менторства сверху
- [ ] НЕТ успешного успеха

### Маркеры
- ✅ "Впервые понял..."
- ✅ "Раньше я..."
- ✅ "Вот что попробовал..."
- ❌ "Запомни правило..."
- ❌ "Ты должен..."
- ❌ "Каждый успешный..."

## Expert/Business Account

**Источник:** expert-carousel

### ToV Formula
```
Практичный 50% + Конкретный 30% + Доступный 20%
```

### Checklist
- [ ] Система/процесс (не технология)
- [ ] Конкретные результаты
- [ ] Что делает (не что является)
- [ ] Производственный язык
- [ ] НЕТ страшилок про AI

### Маркеры
- ✅ "Система производства..."
- ✅ "Снимается вместо тебя..."
- ✅ "Создаёт [формат] автоматически..."
- ❌ "AI заменит людей..."
- ❌ "Революция в контенте..."
- ❌ "Инновационная технология..."

## Health/Medical Niche

**Источник:** expert-video

### ToV Formula
```
Авторитетный 40% + Конкретный 40% + Заботливый 20%
```

### Checklist
- [ ] Связные предложения (не стаккато!)
- [ ] Без имён пациентов (мультиязык)
- [ ] Ответ после 10 сек (не в хуке)
- [ ] Конкретные числа (дозировка, время)

### Маркеры
- ✅ "Миллионы мужчин..."
- ✅ "Исследования показывают..."
- ✅ "Вот что происходит в организме..."
- ❌ "Пациент Сергей, 54 года..."
- ❌ Стаккато "Сырой. Не жареный."

## Адаптация формулы под проект

При создании нового проекта:

1. Определи тип аккаунта (lifestyle/expert/health)
2. Возьми базовую формулу
3. Адаптируй под специфику ниши
4. Запиши в `projects/{name}/brand/voice-style.md`

### Шаблон voice-style.md

```markdown
# Voice & Style — {project}

## ToV Formula
{Формула из универсальных + адаптация}

## Маркеры голоса
✅ Использовать:
- {слово/фраза}
- {слово/фраза}

❌ Избегать:
- {слово/фраза}
- {слово/фраза}

## Специфика ниши
{Особенности аудитории и темы}
```

## PAS Formula (Carousel/Landing)

**Универсально для продающего контента**

```
Problem → Agitate → Solution
```

### Problem (20%)
- Чёткая боль аудитории
- Конкретная ситуация
- "Ты сталкивался с..."

### Agitate (30%)
- Усиление проблемы
- Последствия бездействия
- "И это приводит к..."

### Solution (50%)
- Конкретное решение
- Action verbs
- Результат, не процесс

---

## Story Arc (Threads/Long-form)

**Для storytelling контента**

```
Hook → Context → Method → Insight → Resolution
```

### Hook (10%)
- Интрига
- Неожиданное утверждение
- Зацепить внимание

### Context (25%)
- Детальное описание ситуации
- Личный опыт
- Паттерн/проблема

### Method (20%)
- Что сделал
- Конкретные действия
- Процесс

### Insight (25%)
- Неожиданное откровение
- Кульминация
- Aha-moment

### Resolution (20%)
- Что делаю теперь
- Практическое применение
- Эксперимент/результат
