# Reels/Shorts/TikTok — Система вирусного контента и продаж

Полный skill: от идеи до денег. Не просто "как снять рилс", а как построить машину продаж через вертикальный контент.

## Философия

```
Reels — это НЕ контент. Reels — это верхушка автоматизированной воронки продаж.
Каждый ролик = точка входа в систему: Reels → бот → лид-магнит → КЭВ → продажа.

Первые 3 секунды = 80% успеха ролика
200 миллионов тест — ролик должен быть понятен КАЖДОМУ
Формула ВИСП — минимум 2-3 элемента в каждом опенинге
Качество > Количество: 26 видео = +60K подписчиков (кейс Тимочко)
Фокус на людях, а не алгоритмах — боли меняются редко, алгоритмы — постоянно
Регулярность > частоты — раз в 2-3 дня лучше, чем каждый день трэш
```

## Workflow: Создание Reels

```
1. Идея → Тест 200 миллионов
   - Понятно КАЖДОМУ?
   - Выгода универсальна?
   - Привязана к боли/страху/желанию ЦА?

2. Шаблон → Выбор по ЦЕЛИ ролика
   - Привлечение ЦА → Инструкция, Секрет, Страх
   - Охват → Лайфхаки, Новости, Тренды
   - Вовлечение → Вызов, Топ
   - Продажа → Инструкция/Секрет + CTA в воронку

3. Опенинг → Проверка по ВИСП
   - В (Выгода) + И (Интрига) + С (Срочность) + П (Причастность)
   - Минимум 2-3 буквы

4. Сценарий → Структура
   - Проблема → Актуализация → (Самопрезентация) → Решение → Вывод → CTA
   - Ответ НЕ в первых секундах!
   - Возврат внимания каждые 10-15 сек

5. CTA → Куда ведёт?
   - Подписка / Комментарий / Кодовое слово в воронку
   - CTA по ВИСП, не "переходи по ссылке"

6. Проверка → Чек-лист §14
   - 6 метрик: 2+ выше верхней = успех
```

## Формула ВИСП (Quick Reference)

| Буква | Значение | Проверочный вопрос |
|-------|----------|-------------------|
| **В** | Выгода | Это желание 5-летнего ребёнка? (техника "Дед Мороз") |
| **И** | Интрига | Ответ заменён на "это"? Нет ответа сразу? |
| **С** | Срочность | Есть повелительное наклонение? Почему СЕЙЧАС? |
| **П** | Причастность | Зритель узнаёт себя? Общий опыт/враг/ситуация? |

**Правило:** Минимум 2-3 буквы в каждом опенинге.

## Ключевые формулы

### 6 метрик успеха

| Метрика | Нижняя | Норма | Верхняя |
|---------|--------|-------|---------|
| Retention на всём ролике | 50% | 65% | 80%+ |
| Retention 4+ сек | 60% | 70% | 80%+ |
| Конверсия в подписку | 0.05% | 0.17% | 0.6%+ |
| Конверсия в лайки | — | — | 2%+ |

**2+ выше верхней = успех. 1 ниже нижней = провал.**

### 8 шаблонов → 4 цели

| Цель | Лучшие шаблоны |
|------|---------------|
| **Привлечение ЦА** | Инструкция, Секрет, Страх |
| **Максимальный охват** | Лайфхаки, Новости, Тренды |
| **Вовлечение** | Вызов, Топ |
| **Продажа через воронку** | Инструкция + CTA, Секрет + CTA |

### Воронка (экспресс)

```
REELS → Кодовое слово → Instagram бот → Telegram → Лид-магнит → КЭВ → Продажа
10K      3% (300)        80% (240)       50% (120)   25% (30)    35%→15%  = 1-2 продажи

Масштаб ×10 (100K просмотров) = 10-20 продаж
При среднем чеке 100K₽ = 1-2 млн₽
```

## Диагностика проблем

| Симптом | Причина | Что чинить |
|---------|---------|------------|
| Мало просмотров | Первые 3 секунды не цепляют | ВИСП-опенинг, визуальный хук |
| Просмотры есть, подписок нет | Тема слишком широкая / нет CTA | Сузить тему, добавить "здесь всё о..." |
| Подписки есть, продаж нет | Нет воронки или нет КЭВ | Встроить кодовое слово → бот → воронка |
| Нецелевая аудитория | Развлекательный контент без экспертности | Больше экспертного контент-аут |
| Низкий retention | Ответ сразу / много тем / низкая харизма | Техника "печёночный торт", сократить длину |
| Мало комментариев | Нет провокации | Вызов, намеренные ошибки, спорное мнение |

## Тест 200 миллионов

```
❌ "Почему люди не покупают на Avito" — только для продавцов
❌ "Цены на авто после НГ" — узкая аудитория
✅ "Три способа сэкономить на покупке машины" — универсально
✅ "Как заработать на ненужных вещах дома" — у каждого есть хлам
```

## Убийцы удержания

1. **Ответ сразу** — главный убийца
2. **Несколько тем** — 1 видео = 1 проблема
3. **Низкая харизма** — сокращай длину
4. **Неожиданные звуки** — эффекты в середине
5. **Однообразная картинка** — нужны субтитры, перебивки

## Принципы (выжимка)

- Никогда не говори "топ" — привет из 2000-х
- "Техника печёночного торта" — ответ в самом конце
- "Техника Деда Мороза" — выгода как желание 5-летнего
- Третье место в топе тоже должно быть сильным
- **Тимочко:** Регулярность > частоты. Раз в 3 дня > каждый день
- **КЭВ:** Продавай свидание, а не ночь
- **Воронка:** Reels — это не контент. Reels — это верхушка машины продаж.
- **Персонаж:** RPG-герой, каждое улучшение даёт +N к привлечению
