# Структуры Reels/Shorts

7 структур + тайминги + приёмы удержания. Для генерации сценариев, не для чтения.

---

## Структура 1: Пирамида (классика, но правильно)

**Суть:** Самое сильное обещание в начале → раскрытие → усиление → CTA

```
[0-3 сек]  ХУК — провокация или шокирующий факт
[3-7 сек]  КОНТЕКСТ — почему это важно, для кого
[7-20 сек] РАСКРЫТИЕ — основная ценность (1-3 пункта)
[20-25 сек] УСИЛЕНИЕ — самый сильный аргумент / поворот
[25-30 сек] CTA — подписка / комментарий / сохранение
```

**Когда:** Экспертный контент, инструкции, советы
**Пример хука:** «Один приём, который отличает рилсы на 500 просмотров от рилсов на 500К»

---

## Структура 2: Обратная Пирамида (тизер)

**Суть:** Обещание результата → путь к нему → раскрытие в конце. «Печёночный торт» — ответ на последнем слое.

```
[0-2 сек]  ХУК — «Есть один способ / одна вещь / одна ошибка...»
[2-5 сек]  ТИЗЕР — намёк на ответ, но НЕ ответ
[5-15 сек] НАГНЕТАНИЕ — контекст, боль, усиление проблемы
[15-25 сек] ЛОЖНЫЙ ОТВЕТ (опц.) — «Многие думают, что это X... но нет»
[25-30 сек] РАСКРЫТИЕ — настоящий ответ + CTA
```

**Когда:** Вирусный контент, удержание до конца
**Пример хука:** «Я 3 года снимал рилсы неправильно. Одна деталь всё изменила...»

---

## Структура 3: Список-обратный отсчёт

**Суть:** ТОП-N, от слабого к сильному. Номер 1 — самый мощный, зритель ждёт.

```
[0-2 сек]  ХУК — «ТОП-5 [чего], номер 1 — бомба»
[2-5 сек]  #5 — самый слабый / очевидный
[5-10 сек] #4 — чуть сильнее
[10-15 сек] #3 — уже интересно
[15-22 сек] #2 — сильный, неожиданный
[22-28 сек] #1 — самый мощный, ради него досмотрели
[28-30 сек] CTA — «Сохрани, пригодится»
```

**Когда:** Подборки, рекомендации, инструменты
**Приём удержания:** Между пунктами — микро-тизеры («а вот номер 2 вас удивит»)

---

## Структура 4: Мини-история (сторителлинг)

**Суть:** Конфликт → развитие → развязка → мораль. Сжатая драматургия.

```
[0-3 сек]  ЗАВЯЗКА — «Клиент написал мне: отмените всё» / ситуация
[3-8 сек]  КОНФЛИКТ — что пошло не так, препятствие
[8-18 сек] РАЗВИТИЕ — что делал, попытки, напряжение растёт
[18-25 сек] ПОВОРОТ — неожиданная развязка
[25-30 сек] МОРАЛЬ + CTA — вывод в одну фразу
```

**Когда:** Кейсы, личный опыт, «путь героя»
**Пример хука:** «В прошлую пятницу мне написал подписчик. После этого сообщения я удалил 50 рилсов»

---

## Структура 5: Миф vs Реальность

**Суть:** Сначала общепринятое заблуждение → потом правда с доказательствами.

```
[0-2 сек]  ХУК — «Вам говорят [миф]. Это не работает»
[2-8 сек]  МИФ — подробно, что все думают/делают
[8-12 сек] СЛОМ — «На самом деле...» / «Но вот что происходит»
[12-25 сек] РЕАЛЬНОСТЬ — правда + доказательства / цифры
[25-30 сек] ВЫВОД + CTA
```

**Когда:** Разоблачения, экспертный контент, хайп-темы
**Пример хука:** «"Постите каждый день и будет рост" — это самый вредный совет в 2025»

---

## Структура 6: До/После (трансформация)

**Суть:** Показать контраст между «было» и «стало». Визуально или через текст.

```
[0-2 сек]  ХУК — результат или контраст одной фразой
[2-8 сек]  «БЫЛО» — боль, проблема, цифры ДО
[8-12 сек] ПЕРЕХОД — «Одно изменение» / «Тогда я сделал вот что»
[12-22 сек] «СТАЛО» — результат, цифры ПОСЛЕ, доказательства
[22-27 сек] КАК — кратко, 1-3 шага
[27-30 сек] CTA
```

**Когда:** Кейсы, до/после, портфолио, результаты
**Пример хука:** «200 просмотров → 2 миллиона. Один рилс. Что я изменил?»

---

## Структура 7: Провокация-дискуссия

**Суть:** Жёсткое заявление → аргументы → приглашение к спору. Работает на комментарии.

```
[0-2 сек]  ПРОВОКАЦИЯ — «[Популярное] — бесполезно / мертво / не работает»
[2-7 сек]  ПАУЗА НА ШОКЕ — «Да, я это сказал. И вот почему»
[7-20 сек] АРГУМЕНТЫ — 2-3 довода с конкретикой
[20-25 сек] НЮАНС — «Но есть одно исключение...» (смягчение)
[25-30 сек] ВОПРОС + CTA — «Согласны? Пишите в комментариях»
```

**Когда:** Рост комментариев, вирусность, Threads
**Пример хука:** «Нетворкинг — это пустая трата времени. И я могу доказать»

## Тайминги по форматам

### 15 секунд (максимальное удержание)

```
[0-2 сек]  ХУК — одна фраза, максимум 8 слов
[2-5 сек]  СУТЬ — 1 мысль, 1 факт
[5-12 сек] РАСКРЫТИЕ — быстро, без воды
[12-15 сек] CTA / ПЕТЛЯ — для повторного просмотра
```
**Правило:** 1 мысль. Никаких списков. Темп = быстрый. Идеально для мемов, лайфхаков, одной идеи.

### 30 секунд (золотой стандарт)

```
[0-3 сек]  ХУК
[3-7 сек]  КОНТЕКСТ
[7-22 сек] КОНТЕНТ (1-3 пункта)
[22-27 сек] КУЛЬМИНАЦИЯ
[27-30 сек] CTA
```
**Правило:** 1-3 пункта максимум. Каждые 5-7 сек — смена ритма (кадр/тезис/визуал).

### 60 секунд (экспертный формат)

```
[0-3 сек]  ХУК
[3-10 сек] ПРОБЛЕМА / контекст
[10-20 сек] ПУНКТ 1 (закрыть мини-петлю)
[20-30 сек] ПУНКТ 2 + тизер на 3-й
[30-45 сек] ПУНКТ 3 (самый сильный)
[45-55 сек] ВЫВОД / синтез
[55-60 сек] CTA
```
**Правило:** Каждые 8-10 сек — новый элемент. Между пунктами — мини-хуки («но это не всё»).

### 90 секунд (сторителлинг)

```
[0-3 сек]  ХУК — конфликт / интрига
[3-15 сек] ЗАВЯЗКА — контекст истории
[15-30 сек] РАЗВИТИЕ — нарастание проблемы
[30-50 сек] КОНФЛИКТ — пик напряжения
[50-70 сек] ПОВОРОТ — неожиданная развязка
[70-85 сек] МОРАЛЬ / выводы
[85-90 сек] CTA
```
**Правило:** Только для историй и кейсов. Без истории 90 сек не удержать. Каждые 10-15 сек — мини-клиффхэнгер.

## Приёмы удержания зрителя

### Визуальные приёмы

| Приём | Что делает | Когда менять |
|-------|-----------|-------------|
| **Смена кадра/ракурса** | Сброс адаптации глаза | Каждые 3-5 сек |
| **Текст на экране** | Дублирует речь, цепляет глаз | На каждый ключевой тезис |
| **Zoom-in на лицо** | Усиливает эмоцию | На важных фразах |
| **B-roll/перебивки** | Разбивает «говорящую голову» | Каждые 5-8 сек |
| **Стрелки/выделения** | Направляет фокус | На цифрах и фактах |
| **Субтитры (captions)** | +40% удержания (смотрят без звука) | Всегда |

### Речевые приёмы

| Приём | Что делает | Пример |
|-------|-----------|--------|
| **Мини-хуки между блоками** | Не дают уйти | «Но подождите, это не самое интересное» |
| **Прямое обращение** | Вовлекает | «Вы это знали?» / «Знакомо?» |
| **Пауза + ускорение** | Ритмический контраст | Замедлиться на важном → ускориться |
| **Повтор ключевого слова** | Запоминание | «Главное — конкретика. Конкретика побеждает» |
| **Императив** | Создаёт срочность | «Запомните это» / «Сохраните» / «Никогда не делайте» |

### Структурные приёмы

| Приём | Что делает |
|-------|-----------|
| **Обещание числа пунктов** | Зритель считает и ждёт последний |
| **"А теперь самое важное"** | Перезагрузка внимания на 15-20 сек |
| **Ложный финал** | Кажется, что конец → но есть бонус |
| **Петля на повторный просмотр** | Последняя фраза = первая фраза (loop) |
| **Клиффхэнгер → часть 2** | Серийный формат, подписка ради продолжения |

## Первые 1-3 секунды: что цепляет

### ТОП-7 типов открывашек

| # | Тип | Шаблон | Пример |
|---|-----|--------|--------|
| 1 | **Шок-факт** | «[Число]% людей [делают ошибку]» | «97% рилсов не набирают и 500 просмотров» |
| 2 | **Запрет** | «Никогда не [делайте X]» | «Никогда не начинайте рилс с "Привет"» |
| 3 | **Вызов** | «Спорим, вы [делаете X]» | «Спорим, вы тоже делаете эту ошибку в сторис» |
| 4 | **Результат** | «Я [результат] за [время]» | «Я набрал 50К подписчиков за 2 месяца» |
| 5 | **Секрет** | «[Кто-то] скрывает [что]» | «Блогеры-миллионники скрывают этот приём» |
| 6 | **Боль** | «Если у вас [проблема] — смотрите» | «Если ваши рилсы не залетают — смотрите до конца» |
| 7 | **Конфликт** | «[X] — не работает. Вот что работает» | «Контент-план — не работает. Вот что работает» |

### Чеклист первых секунд

- [ ] Первые 3 слова — самые сильные?
- [ ] Есть движение/действие в кадре с 0 секунды?
- [ ] Ответ НЕ дан (петля открыта)?
- [ ] Понятно, для кого это (идентификация)?
- [ ] Текст на экране дублирует хук?
- [ ] Энергия выше средней (голос, темп)?

## Быстрый выбор структуры

| Задача | Лучшая структура | Хронометраж |
|--------|-----------------|-------------|
| Совет/лайфхак | Пирамида (#1) | 15-30 сек |
| Вирусный охват | Обратная пирамида (#2) | 30-60 сек |
| Подборка/топ | Список-обратный отсчёт (#3) | 30-60 сек |
| Кейс/история | Мини-история (#4) | 60-90 сек |
| Экспертный разбор | Миф vs Реальность (#5) | 30-60 сек |
| Портфолио/результат | До/После (#6) | 15-30 сек |
| Рост комментариев | Провокация (#7) | 15-30 сек |
