## Выбор тем

### topic-matrix.md:
- Категории со статусом СТОП — **НЕ ТРОГАТЬ**
- Приоритет: НОВЫЕ категории, затем OK, затем На грани
- Максимум **1 тема из категории в неделю**
- Проверить что угол помечен `[ ]` (свободен)

### Тест 200 миллионов:
```
✅ Понятно КАЖДОМУ (не только специалистам)?
✅ Выгода универсальна (здоровье, деньги, отношения)?
✅ Привязана к боли/страху/желанию вашей ЦА?
```

**Пример:**
```
❌ "Индекс N при оценке синдрома X" — только для специалистов
✅ "Одна привычка разрушает ваше здоровье после сорока" — каждый человек 40+
```

## Структура сценария (20-25 секунд)

| Блок | Время | Задача |
|------|-------|--------|
| Hook | 3-5 сек | Утверждение (НЕ вопрос). Зацепить, не раскрывая ответ |
| Intrigue | 5-7 сек | Усилить интерес. Ответ всё ещё скрыт |
| Reveal | 5-7 сек | Ответ + механизм (конкретика, цифры) |
| Action | 3-5 сек | Одно конкретное действие |

---

## ВИСП 4/4

| Буква | Что | Проверка |
|-------|-----|----------|
| **В** (Выгода) | Конкретная польза | Понятна 5-летнему? |
| **И** (Интрига) | Ответ скрыт | Заменён на "это"/"один"? |
| **С** (Срочность) | Почему сейчас | Есть повелительное? |
| **П** (Причастность) | Узнаёт себя | Попадание в ситуацию ЦА? |

## Правила текста

### Запрещено:
- ❌ Вопросы в хуке ("Знаете ли вы...?")
- ❌ Ответ до 10-й секунды
- ❌ Стаккато ("Боль. Усталость. Слабость.")
- ❌ Имена клиентов/пациентов
- ❌ Региональные бренды (используйте общие названия)
- ❌ Стыдящий/осуждающий язык
- ❌ Маркетинговый мусор ("like a young man again")
- ❌ Клише ("You won't believe...", "What doctors won't tell you")
- ❌ Гипофора ("Что хуже? Отвечаю: ...")
- ❌ Фраза "Это не магия" (клише)

### Обязательно:
- ✅ Утверждения, не вопросы
- ✅ Связки между предложениями (потому что, а, который, и, ведь)
- ✅ Научная/экспертная база (исследование, цифры, механизм)
- ✅ Конкретное действие в финале (что, сколько, как часто)
- ✅ Уважительный тон (без стыда, без насмешки)

## Адаптация для AI-аватаров (HeyGen)

### Числа → словами:
```
❌ 50 лет, 3 раза, 15%
✅ пятьдесят лет, три раза, пятнадцать процентов
```

### Латиница → кириллица:
```
❌ витамин D → аватар скажет "Ди"
✅ витамин Дэ → правильное произношение

❌ витамин B → "Би"
✅ витамин Бэ → правильно
```

### Анти-стаккато:
```
❌ "Третье место. Телефон перед сном."
✅ "На третьем месте телефон перед сном, потому что..."
```

### Развёрнутые фразы:
```
❌ "принимал витамин Дэ год"
✅ "принимал витамин Дэ в течение года"
```

### Аббревиатуры:
```
❌ ПСА
✅ "пэ эс а" или просто полное название
```

## Валидация (метод персон)

Создайте 3-5 персон из вашей ЦА. Минимум 4/5 должны одобрить:

| Персона | Что проверяет | Red flag |
|---------|--------------|----------|
| **Персона 1** (прагматик) | Конкретика действий | Размытые советы |
| **Персона 2** (скептик) | Новизна информации | Банальность |
| **Персона 3** (эмоционал) | Эмоциональная теплота | Холодный тон |
| **Персона 4** (чувствительный) | Отсутствие давления | Стыдящий язык |
| **Персона 5** (аналитик) | Научные доказательства | "Просто поверьте" |

> Заполните конкретные имена, возраст и контекст из вашего `brand/audience.md`

## Финальный чек-лист

- [ ] Хук — утверждение, не вопрос
- [ ] Ответ НЕ раскрыт до 10-й секунды
- [ ] ВИСП 4/4
- [ ] Нет запрещённых слов (voice-style.md)
- [ ] Нет стаккато
- [ ] Числа словами (если AI-аватар)
- [ ] Латиница → кириллица (если AI-аватар)

- [ ] Нет региональных брендов
- [ ] Конкретное действие в финале
- [ ] 4/5 персон одобрили
- [ ] Хронометраж 20-25 секунд
- [ ] Тон соответствует формуле
- [ ] Проходит тест 200 миллионов

## Workflow перевода

После одобрения (статус → "Переводим"):

1. Перевести на целевые языки
2. Добавить блок "Для публикации" с хештегами
3. Статус → "Взять в монтаж"

### Адаптация по языкам:

| Язык | Менталитет | Адаптация |
|------|-----------|-----------|
| RU | Прямой | Оригинал |
| EN | Прагматичный | Чёткие действия |
| ES | Эмоциональный | Больше экспрессии, "tú" |

**НЕ менять:** Структуру, тайминг, факты, ВИСП.
