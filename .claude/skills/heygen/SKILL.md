- `heygen`, `хейджен`
- `ai avatar`, `аватар`
- `photo avatar`, `оживить фото`

## Когда использовать

### AI Voiceover (озвучка)
Когда пишешь скрипт для HeyGen-аватара:
- Загрузи `heygen-rules.md`
- Применяй правила форматирования текста
- Проверяй по чеклисту перед финализацией

### Photo Avatar (оживление фото)
Когда нужно оживить статичное фото:
- Загрузи `heygen-motion.md`
- Выбери подходящий промпт по типу контента
- Учитывай склейку если нужно больше 10 сек

---

## Quick Reference

### Запрещённые символы (озвучка)
```
" " ... — – * & % / ( ) [ ]
```

### Правила текста
- Все числа словами: `500` → `five hundred`
- Сокращения развернуть: `BPH` → `B P H`
- Максимум 20 слов в предложении
- Без гипофоры (риторический вопрос + ответ)

### Motion движки
- **Kling** — рекомендуется для говорящей головы
- Хорошо слушается промпты
- Предсказуемый результат для склейки
