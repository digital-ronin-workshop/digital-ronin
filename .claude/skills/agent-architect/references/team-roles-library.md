# Team Roles Library

Pre-built role templates for rapid agent composition.

---

## How to Use

1. Identify domain from user request
2. Find matching roles below
3. Combine roles into agent responsibilities
4. Customize for specific context

---

## Content Production

### Content Strategist

**Responsibilities:**
1. Develop content themes and pillars
2. Plan content calendar
3. Identify trending topics and opportunities
4. Align content with business goals
5. Define content mix across formats

**Inputs:** Business goals, audience data, market trends
**Outputs:** Content strategy, editorial calendar, topic lists
**Decisions:** What topics to cover, content priorities
**Escalate:** Major strategy pivots, budget allocation

---

### Writer / Copywriter

**Responsibilities:**
1. Write content in brand voice
2. Adapt messaging for different formats
3. Create hooks and CTAs
4. Iterate based on feedback
5. Maintain style consistency

**Inputs:** Brief, topic, reference materials
**Outputs:** Draft content, multiple variants
**Decisions:** Word choice, structure, tone adjustments
**Escalate:** Voice changes, controversial topics

---

### Editor

**Responsibilities:**
1. Review content for quality and clarity
2. Check grammar, spelling, style
3. Ensure brand voice consistency
4. Suggest structural improvements
5. Final approval before publish

**Inputs:** Draft content
**Outputs:** Edited content, feedback notes
**Decisions:** Minor fixes, structural tweaks
**Escalate:** Major rewrites, factual concerns

---

### Researcher

**Responsibilities:**
1. Gather data and insights on topics
2. Find supporting examples and stats
3. Analyze competitors and trends
4. Verify facts and claims
5. Compile research briefs

**Inputs:** Topic, research questions
**Outputs:** Research brief, sources list, key findings
**Decisions:** Which sources to trust, relevance of data
**Escalate:** Conflicting information, sensitive topics

---

### SEO Specialist

**Responsibilities:**
1. Keyword research and selection
2. Optimize content for search
3. Structure content for discoverability
4. Monitor search performance
5. Recommend content updates

**Inputs:** Content, target audience, goals
**Outputs:** Keyword list, optimization recommendations
**Decisions:** Keyword priorities, meta descriptions
**Escalate:** Major SEO strategy changes

---

### Social Media Manager

**Responsibilities:**
1. Adapt content for social platforms
2. Schedule and publish posts
3. Engage with audience comments
4. Track social metrics
5. Identify viral opportunities

**Inputs:** Content, brand guidelines, calendar
**Outputs:** Social posts, engagement reports
**Decisions:** Post timing, response tone
**Escalate:** Crisis situations, controversial comments

## Role Composition Patterns

### Content Team (minimum)

```
Writer + Editor + Researcher
```

### Content Team (full)

```
Strategist + Writer + Editor + Researcher + SEO + Social Manager
```

### Video Team (minimum)

```
Scriptwriter + Video Editor
```

### Video Team (full)

```
Strategist + Scriptwriter + Editor + Thumbnail Designer + SEO
```

### Support Team

```
Support Agent + QA Reviewer + Documentation Writer
```

### Analytics Team

```
Business Analyst + Data Analyst + Competitive Analyst
```
