# Selling Meanings

> Один сильный смысл > 100 средних сообщений

## Что это

**Продающие смыслы** — совокупность идей о продукте, проблеме и результате, которые формируются в голове клиента и приводят к покупке.

Агент умеет:
- **Извлекать** ядро смысла из продукта/ниши/ЦА
- **Формулировать** смысл в 1 предложение
- **Усиливать** через психологические триггеры
- **Адаптировать** под форматы (пост, видео, лендинг)
- **Оценивать** конверсионный потенциал

## Интеграция с проектом

Skill загружает:
1. **Этот skill** → универсальные формулы, психология
2. **project/brand/** → голос, ЦА, боли ниши
3. **project/brand/meanings.md** → core meanings проекта (если есть)

Генерирует:
- Ядро смысла = **универсальная формула** + **нишевый контекст**

## Связанные skills

| Skill | Когда использовать |
|-------|-------------------|
| **launch** | Запуск продукта по PLF (последовательности, PLC, ментальные триггеры) |
| **research-customer** | Исследование клиента, 6 шагов кастдева, извлечение болей и желаний |
| **offer-core** | Формулировка ядра оффера, тест "конференция" |
| **unpacking** | Распаковка смыслов, карта 8 столбиков, обещания |

### Полная цепочка (Сердитов)
```
research-customer → offer-core → unpacking → selling-meanings → launch
    (сырьё)         (ядро)      (тезисы)      (усиление)     (запуск)
```

Selling-meanings + launch = полная система продаж:
- **selling-meanings** → ЧТО говорить (смыслы, офферы, формулы)
- **launch** → КОГДА и КАК говорить (последовательности, тайминг)

### Глубокий синтез: Тимочко × Hormozi

`timochko-selling-patterns.md` содержит **синтезированную систему**, объединяющую:
- **Тимочко** (КЭВ, 32 способа, вебинары, CustDev, продуктовая линейка)
- **Hormozi** (Value Equation, Grand Slam Offers, Virtuous Price Cycle)
- **Западные фреймворки** (JTBD, PLF, Lean Startup, Mom Test)

Ключевые конструкторы:
- **КЭВ** → Подбор ключевого этапа воронки для любой ниши
- **Вебинар** → Пошаговая сборка продающего вебинара (6 блоков)
- **CustDev** → Готовый скрипт глубинного интервью (18 вопросов)
- **32 способа** → Матрица 2×2 (бесплатно/платно × входящий/исходящий)
- **Продуктовая линейка** → Tripwire → Основной → Премиум
