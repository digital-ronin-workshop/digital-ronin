# Формулы копирайтинга

> Формула — это скелет. Смысл — это душа. Без смысла формула мертва.

## Метод скользкой горки (по Халилову/Шугерману)

**Главный принцип всего копирайтинга:** Задача первого предложения — заставить прочитать второе. И так далее по «эстафете внимания».

```
Скользкая горка = структура текста, в которой:
1. С первых слов захватываем внимание
2. Каждое следующее предложение передаёт "эстафету любопытства"
3. Читатель не может остановиться — как санки на горке
4. В конце — целевое действие
```

### Элементы скользкой горки

| Элемент | Как работает | Пример |
|---------|--------------|--------|
| **Сцепление** | Первая фраза цепляет и не отпускает | "Главное в современном мире — отличать знаки судьбы от таргетированной рекламы" |
| **Интрига** | Открытый вопрос или загадка | "Как считаете, кто знает о вас всё? Нет, спецслужбы тут ни при чём..." |
| **Эстафета** | Каждое предложение → следующее | Заканчивать абзац так, чтобы хотелось читать дальше |
| **Твист** | Неожиданный поворот | "А вот здесь я делаю элегантный твист..." |
| **Конкретика** | Цифры, факты, имена | "За 21 год практики участвовал в 732 процессах" |

### Точки входа (где запускать горку)

| Формат | Точка входа | Время захвата |
|--------|-------------|---------------|
| Текстовый пост | Первое предложение | 3 секунды |
| Stories | Первая история серии | 2-3 секунды |
| Reels/TikTok | Первые 2 секунды | 1-2 секунды |
| YouTube | Первые 5 секунд | 3-5 секунд |
| Прямой эфир | Первые 30 секунд | 15-30 секунд |

### Способы "сцепления" с читателем

1. **Интрига** — открыть петлю, которую хочется закрыть
2. **Сильное обещание** — конкретный результат
3. **Юмор** — улыбка = внимание
4. **Узнаваемая боль** — "это про меня"
5. **Удивительный факт** — разрыв шаблона

### Пружины скользкой горки

```
Что не даёт остановиться:
├── Качество текста (упругий, без воды)
├── Сторителлинг (клиффхэнгеры, твисты)
├── Структура (логика, эстафета от части к части)
├── Голос автора (уверенность, экспертность)
├── Эмоции (не оставлять равнодушным)
├── Польза (ценность в каждом абзаце)
└── Адаптация под платформу
```

## Базовые правила сильного текста (по Халилову)

### 18 правил

| # | Правило | Антипаттерн |
|---|---------|-------------|
| 1 | **Конкретная цель** у каждого текста | Текст "для галочки" |
| 2 | **Пиши со скальпелем** — убирай лишнее | Раздувать водой |
| 3 | **Пиши пьяным, редактируй трезвым** | Редактировать на ходу |
| 4 | **Будь за белых или за красных** | Полутона, "с одной стороны..." |
| 5 | **Не дави на клиента** | >3 коммерческих слов на текст |
| 6 | **Главный герой — читатель** | Писать о себе/продукте |
| 7 | **Максимальная конкретика** | Абстракции без цифр |
| 8 | **Никаких канцеляризмов** | Официозный язык |
| 9 | **Выключи писателя, включи художника** | Текст без образов |
| 10 | **Следи за комментариями** | Игнорировать негатив |
| 11 | **Упругость текста** — нет ничего лишнего | Можно сократить без потери смысла |
| 12 | **Визуальность** — текст создаёт картинку | Сухое перечисление фактов |
| 13 | **Текст для пересказа** — можно рассказать другу | Сложно объяснить суть |
| 14 | **Экологичность** — не обманывать, не манипулировать | Фейки, манипуляции |
| 15 | **УТП** в каждом тексте | Нет уникального угла |
| 16 | **CJM** — контент под этап клиента | Продавать холодным |
| 17 | **Один смысл = один текст** | Много идей в одном посте |
| 18 | **Системность** — план, а не хаос | Писать что попало |

### Формула упругого текста

```
После написания:
1. Прочитать вслух
2. Отрезать всё, что можно отрезать без потери смысла
3. Сократить минимум в 1.5 раза
4. Проверить: за что цепляется взгляд?
5. Если нечего "унести" — добавить конкретику
```

### Что относится к конкретике

- **Цифры:** лет на рынке, клиентов, результатов
- **Факты:** история, технологии, уникальность
- **Достижения:** награды, рейтинги, признание
- **Имена:** известные клиенты, эксперты, партнёры

**Пример трансформации:**
```
❌ "Это высококлассный и востребованный специалист с большим опытом"

✅ "За 21 год практики участвовал в 732 процессах.
   Работает с 3 людьми из списка Forbes.
   Премия 'Адвокат года-2019'.
   Отец трёх дочерей."
```

## Customer Journey Map (CJM) — контент по этапам

| Этап | Цель контента | Форматы |
|------|---------------|---------|
| **1. Поиск** | Заинтересовать темой | Обзоры, гайды, кейсы, исследования |
| **2. Выбор** | Донести УТП | Тест продукта, сравнения, отзывы |
| **3. Покупка** | Преодолеть прокрастинацию | Дедлайны, рассрочки, гарантии |
| **4. Удержание** | Напомнить о себе | Обновления, лайфхаки, спецпредложения |
| **5. Лояльность** | Сделать адвокатом | Партнёрка, бонусы, персонализация |

**Правило:** Нельзя продавать на этапе поиска. Нельзя обучать на этапе покупки.

## Value Equation (по Hormozi)

### Формула ценности

```
              DREAM OUTCOME × LIKELIHOOD OF ACHIEVEMENT
VALUE = ────────────────────────────────────────────────────────
              TIME DELAY × EFFORT & SACRIFICE
```

**Как читать:** Чем больше числитель и меньше знаменатель — тем выше воспринимаемая ценность.

### 4 рычага ценности

| Рычаг | Направление | Как усилить |
|-------|-------------|-------------|
| **Dream Outcome** | ↑ Увеличить | Масштабнее результат, конкретнее цифры |
| **Likelihood** | ↑ Увеличить | Гарантии, кейсы, методология, социальное доказательство |
| **Time Delay** | ↓ Уменьшить | Быстрее результат: "за 14 дней" vs "за 6 месяцев" |
| **Effort** | ↓ Уменьшить | Готовые шаблоны, автоматизация, "без X, Y, Z" |

### Примеры трансформации

**Слабый оффер:**
```
Результат: научиться продавать (абстрактно)
Вероятность: неясна
Время: несколько месяцев
Усилия: много практики
→ Низкая ценность
```

**Сильный оффер:**
```
Результат: +50% к конверсии, +200К/мес к доходу
Вероятность: 94% студентов достигают (гарантия возврата)
Время: 21 день
Усилия: готовые скрипты, 30 мин/день
→ Высокая ценность
```

### Чек-лист Value Equation

- [ ] Dream Outcome конкретен и измерим?
- [ ] Есть доказательства вероятности успеха?
- [ ] Срок результата минимален?
- [ ] Усилия клиента снижены?

## Классические формулы

### AIDA (Attention → Interest → Desire → Action)

**Когда использовать:** Лендинги, email-рассылки, длинные посты

```
A — ATTENTION (Внимание)
    Зацепить заголовком, вызвать "что?"

I — INTEREST (Интерес)
    Показать преимущества, вызвать "интересно"

D — DESIRE (Желание)
    Усилить мотивацию кейсами, вызвать "хочу"

A — ACTION (Действие)
    Дать чёткий CTA, вызвать "беру"
```

**Пример (курс по инвестициям):**
```
A: "Почему 90% инвесторов теряют деньги в первый год?"

I: "Потому что они не знают 3 правила, которые используют профи.
   За 12 лет я вывел систему, которая работает в любой ситуации."

D: "Мой студент Алексей начал с 50К и за 2 года вышел на
   пассивный доход 150К/мес. Без рисков, без стресса, без везения."

A: "Скачайте бесплатный гайд '3 правила безубыточных инвестиций' →"
```

---

### PAS (Problem → Agitate → Solution)

**Когда использовать:** Короткие посты, таргет, баннеры

```
P — PROBLEM (Проблема)
    Назвать боль, которую клиент узнаёт

A — AGITATE (Усиление)
    Показать последствия, усилить дискомфорт

S — SOLUTION (Решение)
    Предложить выход
```

**Пример (сервис автоматизации):**
```
P: "Ваши менеджеры тратят 5 часов в день на рутину."

A: "Это 100 часов в месяц. 25 000₽ зарплаты на задачи,
   которые может делать бот. И так каждый месяц."

S: "Наш сервис автоматизирует 80% рутины за 3 дня.
   Бесплатный аудит — оставьте заявку."
```

---

### BAB (Before → After → Bridge)

**Когда использовать:** Трансформационные продукты, кейсы

```
B — BEFORE (До)
    Описать текущую ситуацию клиента

A — AFTER (После)
    Показать желаемый результат

B — BRIDGE (Мост)
    Объяснить, как туда попасть
```

**Пример (программа похудения):**
```
B: "До лета 2 месяца. Килограммы не уходят, диеты не работают,
   мотивация на нуле."

A: "Представьте: через 8 недель вы на пляже — стройная,
   уверенная, в том самом купальнике."

B: "Программа '8 недель до формы' — питание + тренировки +
   поддержка каждый день. Гарантия результата или деньги назад."
```

---

### 4P (Picture → Promise → Prove → Push)

**Когда использовать:** Когда важны доказательства

```
P — PICTURE (Картина)
    Визуализировать проблему или результат

P — PROMISE (Обещание)
    Дать конкретное обещание

P — PROVE (Доказательство)
    Подтвердить кейсами, цифрами

P — PUSH (Толчок)
    Подтолкнуть к действию
```

**Пример (курс по продажам):**
```
P: "Представьте: клиент говорит 'дорого' — и уходит.
   Вы знаете, что могли закрыть сделку, но не знали как."

P: "Через 2 недели вы будете закрывать 7 из 10 возражений
   на месте. Без давления, без манипуляций."

P: "247 продавцов прошли этот курс. Средний рост конверсии —
   с 15% до 45%. Вот их отзывы..."

P: "Старт следующего потока — понедельник. Осталось 5 мест
   по ранней цене. Записаться →"
```

---

### DIBABA (Desire → Identification → Bump → Action → Buy → Atmosphere)

**Когда использовать:** Сложные продукты, длинные воронки

```
D — DESIRE (Желание)
    Определить, что волнует аудиторию

I — IDENTIFICATION (Сопоставление)
    Показать, как продукт удовлетворяет желание

B — BUMP (Подталкивание)
    Преподнести товар выгодно

A — ACTION (Реакция)
    Дать время на обдумывание

B — BUY (Покупка)
    Призвать к покупке

A — ATMOSPHERE (Атмосфера)
    Создать благоприятную среду после покупки
```

## Усилители продаж (по Корзуну)

### 5 групп усилителей

#### 1. Триггеры немедленной покупки
**Цель:** Создать срочность

| Приём | Пример |
|-------|--------|
| Дедлайн | "Цена растёт через 48 часов" |
| Лимит мест | "Осталось 3 места в группе" |
| Бонус за скорость | "Первым 10 — личная консультация" |
| Сезонность | "Только в январе — такие условия" |

#### 2. Вовлечение и удержание
**Цель:** Не дать уйти

| Приём | Пример |
|-------|--------|
| Открытый вопрос | "Как думаете, почему это работает?" |
| Интрига | "Но главное я расскажу в конце..." |
| История с продолжением | "Что было дальше — завтра" |
| Провокация | "90% не согласятся, но..." |

#### 3. Доверие к эксперту
**Цель:** Установить авторитет

| Приём | Пример |
|-------|--------|
| Опыт | "12 лет в маркетинге" |
| Результаты | "Продал на 425 млн" |
| Регалии | "Сертифицированный тренер ICF" |
| Известные клиенты | "Работал с Яндекс, Сбер, Тинькофф" |
| Медиа | "Публиковался в Forbes, РБК" |

#### 4. Восприятие ценности
**Цель:** Показать, что это выгодно

| Приём | Пример |
|-------|--------|
| Якорь | "Обычно такое стоит 150К. Здесь — 39К" |
| Сравнение с альтернативой | "Консультант берёт 50К за час" |
| Разбивка на составляющие | "Курс + менторство + сообщество = по отдельности 200К" |
| ROI | "Окупается за первый месяц" |
| Цена за день | "Всего 100₽ в день за год обучения" |

#### 5. Выделение среди конкурентов
**Цель:** Показать уникальность

| Приём | Пример |
|-------|--------|
| Антипозиция | "Это не очередной курс по SMM" |
| Уникальный метод | "Авторская методика ВИСП" |
| Гарантия | "Единственные, кто гарантирует результат" |
| Нишевание | "Только для IT-специалистов" |
| Формат | "Единственный курс с личным сопровождением" |

## Структуры постов

### Hook + Body + CTA

```
HOOK (первая строка)
└─ Зацепить, остановить скролл

BODY (основной текст)
├─ Раскрыть идею
├─ Дать ценность
└─ Вызвать эмоцию

CTA (призыв)
└─ Сказать, что делать дальше
```

### Story Structure

```
1. СИТУАЦИЯ
   "Год назад я был в точке..."

2. ПРОБЛЕМА
   "Но столкнулся с..."

3. ПОИСК
   "Перепробовал всё: A, B, C..."

4. РЕШЕНИЕ
   "Пока не нашёл..."

5. РЕЗУЛЬТАТ
   "Теперь..."

6. УРОК
   "Главный вывод:..."

7. CTA
   "Если вы в похожей ситуации..."
```

### Controversy Structure

```
1. ПРОВОКАЦИЯ
   "Непопулярное мнение:..."

2. ОБОСНОВАНИЕ
   "Потому что..."

3. ДОКАЗАТЕЛЬСТВО
   "Вот что происходит на практике..."

4. ВЫВОД
   "Поэтому я рекомендую..."

5. ВОПРОС
   "А вы как думаете?"
```

| Задача | Лучшая формула | Почему |
|--------|----------------|--------|
| Привлечь холодную аудиторию | PAS | Быстро цепляет боль |

## Чек-лист продающего текста

### Заголовок/Hook
- [ ] Цепляет за 3 секунды?
- [ ] Содержит боль или выгоду?
- [ ] Вызывает желание читать дальше?

### Основной текст
- [ ] Есть конкретика (цифры, имена)?
- [ ] Использован хотя бы 1 усилитель из каждой группы?
- [ ] Закрыты основные возражения?
- [ ] Есть социальное доказательство?

- [ ] Понятно, что делать?
- [ ] Есть причина действовать сейчас?
- [ ] Низкий барьер входа?
